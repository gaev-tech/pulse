# Руководство пользователя Pulse

## Что такое Pulse

Pulse — трекер задач для команд и личного использования. Позволяет организовывать работу через гибкие фильтры, управлять командами и отслеживать активность.

## Ключевые сущности

### Задачи

Задача — основная единица работы. Каждая задача имеет название, описание, статус и может быть привязана к команде или оставаться личной.

### Фильтры

Фильтр — сохранённый набор условий для отображения задач. Фильтры бывают личными (видны только вам) и командными (доступны участникам команды). Кликнув по фильтру, вы переходите на экран результатов с отфильтрованным списком задач.

### Команды

Команда объединяет нескольких пользователей и их общие задачи. Участники команды имеют разные уровни доступа, определяемые через разрешения.

## Авторизация через magic link

Pulse не использует пароли. При входе на указанный email отправляется одноразовая ссылка. По клику на ссылку выполняется автоматический вход. Ссылка действительна 15 минут.

## Основные принципы работы

- **Навигация** осуществляется через боковую панель слева.
- **Личные фильтры** отображаются в разделе «Личные фильтры» сайдбара.
- **Командные фильтры** отображаются под соответствующей командой.
- **Ошибки** всегда сопровождаются уведомлением в правом нижнем углу экрана.
- **Опасные операции** (удаление и т.п.) требуют подтверждения во всплывающем окне.
