# TC-446: Mobile — модальное окно на весь экран

**Предусловие:** Пользователь авторизован. Viewport установлен в 390×844px. Открыт любой экран приложения.

**Шаги:**
1. Нажать кнопку создания задачи (открывается Task Create Modal).

**Ожидаемый результат:**
- Task Create Modal занимает весь экран (width: 100%, height: 100%).
- Кнопка закрытия (×) фиксирована в верхней части модального окна.
