# TC-447: Mobile — горизонтальный скролл таблицы Filter Results

**Предусловие:** Пользователь авторизован. Viewport установлен в 390×844px. Открыт Filter Results Screen с несколькими задачами и включёнными необязательными колонками (ответственный, статус, дата создания).

**Шаги:**
1. Провести пальцем по таблице задач вправо.

**Ожидаемый результат:**
- Таблица прокручивается горизонтально.
- Колонки «Ключ» и «Название» остаются видимыми (зафиксированы слева).
- Необязательные колонки доступны при горизонтальном скролле.
