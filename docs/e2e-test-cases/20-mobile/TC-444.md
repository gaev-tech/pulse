# TC-444: Mobile — открытие сайдбара через кнопку-гамбургер

**Предусловие:** Пользователь авторизован. Viewport установлен в 390×844px. Сайдбар закрыт.

**Шаги:**
1. Нажать кнопку-гамбургер в топбаре.

**Ожидаемый результат:**
- Navigation Sidebar появляется как overlay поверх контента.
- Поверх контента (за сайдбаром) отображается полупрозрачный backdrop.
