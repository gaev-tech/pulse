# TC-126: Полный отзыв доступа к задаче

**Предусловие:** Пользователь-владелец задачи «ACME-42» ранее выдал пользователю «Мария Сидорова» права `view`, `edit.title`, `edit.description`.

**Шаги:**
1. Войти как владелец задачи.
2. Открыть Task Share Modal задачи «ACME-42».
3. Найти пользователя «Мария Сидорова» в списке пользователей с доступом.
4. Выполнить полный отзыв доступа (удалить permission).

**Ожидаемый результат:**
- У пользователя «Мария Сидорова» отозваны все права по задаче «ACME-42», включая `view`.
- Пользователь «Мария Сидорова» больше не может просматривать задачу «ACME-42».
