# TC-135: Недоступность редактирования родителя без права edit.parent

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи «ACME-42». У пользователя есть право `view`, но нет права `edit.parent`.

**Шаги:**
1. В правой части Task Screen найти поле «Родительская задача».
2. Попытаться кликнуть на поле родительской задачи для изменения.

**Ожидаемый результат:**
- Поле родительской задачи недоступно для редактирования.
- Родительская задача отображается в режиме только для чтения.
- Нет кнопки очистки и возможности выбора другой родительской задачи.
