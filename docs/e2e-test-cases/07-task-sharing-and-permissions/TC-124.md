# TC-124: Шаринг с пустым набором прав -- выдаётся только view

**Предусловие:** Пользователь авторизован, открыт Task Share Modal задачи «ACME-42». У пользователя есть право `share`. Найден пользователь «Мария Сидорова».

**Шаги:**
1. Выбрать пользователя «Мария Сидорова» из результатов поиска.
2. Не отмечать ни один чекбокс прав (кроме автоматического `view`).
3. Подтвердить выдачу прав.

**Ожидаемый результат:**
- Пользователю «Мария Сидорова» выдано только право `view` по задаче «ACME-42».
- Пользователь «Мария Сидорова» может просматривать задачу, но не может её редактировать.
- В ленте событий задачи появилось событие `task.shared`.
