# TC-136: Недоступность редактирования вложений без права edit.attachments

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи «ACME-42» с вложением «report.pdf». У пользователя есть право `view`, но нет права `edit.attachments`.

**Шаги:**
1. В правой части Task Screen найти раздел «Вложения» с файлом «report.pdf».
2. Попытаться добавить новое вложение или удалить существующее.

**Ожидаемый результат:**
- Раздел вложений отображается в режиме только для чтения.
- Нет кнопки добавления нового вложения.
- Нет кнопки удаления существующего вложения «report.pdf».
