# TC-129: Недоступность изменения статуса без права edit.status

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи «ACME-42» со статусом `opened`. У пользователя есть право `view`, но нет права `edit.status`.

**Шаги:**
1. В правой части Task Screen найти поле «Статус» со значением `opened`.
2. Попытаться кликнуть на поле статуса для изменения.

**Ожидаемый результат:**
- Поле статуса недоступно для изменения.
- Статус отображается в режиме только для чтения.
- Нет возможности переключить статус задачи.
