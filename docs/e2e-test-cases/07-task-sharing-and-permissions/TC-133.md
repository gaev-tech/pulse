# TC-133: Недоступность редактирования связей без права edit.relations

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи «ACME-42». У пользователя есть право `view`, но нет права `edit.relations`.

**Шаги:**
1. В правой части Task Screen найти раздел «Связанные задачи».
2. Попытаться открыть Related Tasks Popup для добавления или удаления связей.

**Ожидаемый результат:**
- Раздел связанных задач отображается в режиме только для чтения.
- Нет кнопок добавления или удаления связанных задач.
