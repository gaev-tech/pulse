# TC-128: Недоступность редактирования поля без права edit.description

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи «ACME-42». У пользователя есть право `view`, но нет права `edit.description`.

**Шаги:**
1. В левой части Task Screen попытаться кликнуть на поле описания задачи для редактирования.

**Ожидаемый результат:**
- Поле описания задачи недоступно для редактирования.
- Описание отображается в режиме только для чтения (отрендеренный markdown).
- Нет возможности перейти в режим редактирования описания.
