# TC-123: Пользователь может выдать только те права, которые есть у него

**Предусловие:** Пользователь авторизован, открыт Task Share Modal задачи «ACME-42». У пользователя есть права `share`, `edit.title`, `edit.description`. У пользователя нет прав `edit.status`, `edit.assignee`, `edit.labels` и остальных.

**Шаги:**
1. Выбрать пользователя «Мария Сидорова» из результатов поиска.
2. Просмотреть список доступных чекбоксов прав.

**Ожидаемый результат:**
- Чекбоксы `edit.title` и `edit.description` доступны для выбора.
- Чекбоксы `edit.status`, `edit.assignee`, `edit.labels`, `edit.links`, `edit.relations`, `edit.blocking`, `edit.parent`, `edit.attachments`, `share` недоступны (задизейблены или скрыты), так как у текущего пользователя нет этих прав.
