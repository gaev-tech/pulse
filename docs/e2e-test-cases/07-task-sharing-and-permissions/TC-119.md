# TC-119: Открытие Task Share Modal

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи «ACME-42». У пользователя есть право `share`.

**Шаги:**
1. В верхней части Task Screen нажать кнопку шаринга задачи.

**Ожидаемый результат:**
- Открывается Task Share Modal.
- Modal содержит поле поиска пользователя и список прав для выдачи.
