# TC-131: Недоступность редактирования меток без права edit.labels

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи «ACME-42» с меткой «Срочно». У пользователя есть право `view`, но нет права `edit.labels`.

**Шаги:**
1. В правой части Task Screen найти поле «Метки» с меткой «Срочно».
2. Попытаться кликнуть на поле меток для редактирования.

**Ожидаемый результат:**
- Поле меток недоступно для редактирования.
- Метки отображаются в режиме только для чтения.
- Нет возможности добавить или удалить метки.
