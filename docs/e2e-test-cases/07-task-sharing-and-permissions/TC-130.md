# TC-130: Недоступность изменения ответственного без права edit.assignee

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи «ACME-42». У пользователя есть право `view`, но нет права `edit.assignee`.

**Шаги:**
1. В правой части Task Screen найти поле «Ответственный».
2. Попытаться кликнуть на поле ответственного для изменения.

**Ожидаемый результат:**
- Поле ответственного недоступно для изменения.
- Ответственный отображается в режиме только для чтения.
- Нет кнопки очистки и возможности выбора другого ответственного.
