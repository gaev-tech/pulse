# TC-137: Недоступность кнопки шаринга без права share

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи «ACME-42». У пользователя есть право `view` и некоторые права на редактирование, но нет права `share`.

**Шаги:**
1. В верхней части Task Screen найти область элементов управления задачей.
2. Попытаться найти кнопку шаринга задачи.

**Ожидаемый результат:**
- Кнопка шаринга задачи отсутствует или недоступна для нажатия.
- Пользователь не может открыть Task Share Modal.
- Нет возможности выдать доступ к задаче другим пользователям.
