# TC-127: Недоступность редактирования поля без соответствующего права (edit.title)

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи «ACME-42». У пользователя есть право `view`, но нет права `edit.title`.

**Шаги:**
1. В левой части Task Screen попытаться кликнуть на название задачи для редактирования.

**Ожидаемый результат:**
- Поле названия задачи недоступно для редактирования.
- Курсор не меняется на текстовый при наведении на название.
- Поле отображается в режиме только для чтения.
