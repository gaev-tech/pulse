# TC-134: Недоступность редактирования блокирования без права edit.blocking

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи «ACME-42». У пользователя есть право `view`, но нет права `edit.blocking`.

**Шаги:**
1. В правой части Task Screen найти раздел «Блокирующие задачи».
2. Попытаться открыть Blocking Tasks Popup для добавления или удаления блокирующих задач.

**Ожидаемый результат:**
- Раздел блокирующих задач отображается в режиме только для чтения.
- Нет кнопок добавления или удаления блокирующих задач.
