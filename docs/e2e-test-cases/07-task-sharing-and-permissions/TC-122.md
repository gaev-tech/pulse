# TC-122: Право view автоматически выдаётся и отображается задизейбленным

**Предусловие:** Пользователь авторизован, открыт Task Share Modal задачи «ACME-42». У пользователя есть право `share`. Найден пользователь «Мария Сидорова».

**Шаги:**
1. Выбрать пользователя «Мария Сидорова» из результатов поиска.
2. Просмотреть список чекбоксов прав.

**Ожидаемый результат:**
- Чекбокс права `view` отображается отмеченным и задизейбленным (нельзя снять галочку).
- Право `view` выдаётся автоматически и не может быть отозвано иначе как через полный отзыв доступа к задаче.
