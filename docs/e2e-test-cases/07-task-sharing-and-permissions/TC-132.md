# TC-132: Недоступность редактирования ссылок без права edit.links

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи «ACME-42». У пользователя есть право `view`, но нет права `edit.links`.

**Шаги:**
1. В правой части Task Screen найти раздел «Ссылки».
2. Попытаться открыть Related Links Popup для добавления или редактирования ссылок.

**Ожидаемый результат:**
- Раздел ссылок отображается в режиме только для чтения.
- Нет кнопок добавления, редактирования или удаления внешних ссылок.
