# TC-428: Импорт из GitHub — успешный запуск

**Предусловие:** Пользователь авторизован. Import Tasks Modal открыта на табе GitHub.

**Шаги:**
1. Заполнить поля: owner, repo, access token.
2. Нажать "Import".

**Ожидаемый результат:**
- Таб переходит в состояние прогресса.
- Отображается индикатор с текущим прогрессом.
- По завершении отображается результат импорта.
- Появляется Import Completed Toast.
