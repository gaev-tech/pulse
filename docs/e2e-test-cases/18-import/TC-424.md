# TC-424: Принятие результата импорта (Accept)

**Предусловие:** Пользователь авторизован. Импорт завершён, отображается результат с кнопками "Accept" и "Restart".

**Шаги:**
1. Нажать кнопку "Accept".

**Ожидаемый результат:**
- Таб сбрасывается в состояние формы ввода.
- Все поля формы пусты.
- Возможно начать новый импорт.
