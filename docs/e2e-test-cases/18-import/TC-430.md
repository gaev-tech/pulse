# TC-430: Валидация формы Jira — незаполненные поля

**Предусловие:** Пользователь авторизован. Import Tasks Modal открыта на табе Jira.

**Шаги:**
1. Оставить поля формы пустыми.
2. Нажать "Import".

**Ожидаемый результат:**
- Импорт не запускается.
- Отображаются сообщения об ошибках валидации для обязательных полей: instance URL, project key, email, API token.
