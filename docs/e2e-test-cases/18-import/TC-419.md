# TC-419: Переключение табов в Import Tasks Modal

**Предусловие:** Import Tasks Modal открыта.

**Шаги:**
1. Убедиться, что активен таб CSV.
2. Нажать на таб Jira.
3. Нажать на таб GitHub.
4. Нажать на таб CSV.

**Ожидаемый результат:**
- При переключении на Jira отображается форма с полями: instance URL, project key, email, API token.
- При переключении на GitHub отображается форма с полями: owner, repo, access token.
- При переключении обратно на CSV отображается форма с полями загрузки файла и id_column.
- Каждый таб показывает свою независимую форму.
