# TC-433: Импорт задач с parent-child связями из CSV

**Предусловие:** Пользователь авторизован. Подготовлен CSV с колонками id, title, parent (ссылка на id другой строки).

**Шаги:**
1. Открыть Import Tasks Modal, таб CSV.
2. Загрузить CSV, указать id_column: "id".
3. Нажать "Import".
4. Дождаться завершения.
5. Открыть одну из дочерних задач.

**Ожидаемый результат:**
- Импорт завершён.
- У дочерних задач корректно указана родительская задача.
- У родительских задач отображаются дочерние задачи.
