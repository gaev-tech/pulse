# TC-420: Успешный импорт из CSV — личные задачи

**Предусловие:** Пользователь авторизован. Import Tasks Modal открыта на табе CSV. Подготовлен CSV-файл с колонками: id, title, description, status, assignee_email, labels.

**Шаги:**
1. Загрузить CSV-файл.
2. Указать id_column: "id".
3. Нажать кнопку "Import".

**Ожидаемый результат:**
- Форма заменяется на индикатор прогресса с отображением "Обработано X из Y".
- Кнопка "Import" недоступна во время импорта.
- По завершении отображается результат: "Imported: X, Updated: Y".
- Появляются кнопки "Accept" и "Restart".
- В нижней части экрана появляется Import Completed Toast.
- Задачи созданы в личных задачах пользователя.
