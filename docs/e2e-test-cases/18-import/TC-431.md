# TC-431: Ошибка доступа к Jira API — неверный токен

**Предусловие:** Пользователь авторизован. Import Tasks Modal открыта на табе Jira.

**Шаги:**
1. Заполнить поля: корректный instance URL и project key, но невалидный api_token.
2. Нажать "Import".

**Ожидаемый результат:**
- Отображается Error Toast с сообщением об ошибке доступа к Jira.
- Таб остаётся в состоянии формы ввода.
