# TC-427: Импорт из Jira — успешный запуск

**Предусловие:** Пользователь авторизован. Import Tasks Modal открыта на табе Jira.

**Шаги:**
1. Заполнить поля: instance URL, project key, email, API token.
2. Нажать "Import".

**Ожидаемый результат:**
- Таб переходит в состояние прогресса.
- Отображается индикатор с текущим прогрессом.
- По завершении отображается результат импорта.
- Появляется Import Completed Toast.
