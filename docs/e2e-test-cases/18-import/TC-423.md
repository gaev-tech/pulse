# TC-423: Просмотр ошибок после импорта с частичными сбоями

**Предусловие:** Пользователь авторизован. Импорт завершён, часть задач не импортирована (например, отсутствует title).

**Шаги:**
1. Открыть Import Tasks Modal.
2. Перейти на таб, где выполнялся импорт.

**Ожидаемый результат:**
- Отображается результат: "Imported: X, Updated: Y".
- Присутствует раскрываемый список ошибок с указанием origin_id и описания ошибки для каждой непроимпортированной задачи.
- Доступны кнопки "Accept" и "Restart".
