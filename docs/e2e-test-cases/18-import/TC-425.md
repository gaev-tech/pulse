# TC-425: Перезапуск импорта (Restart)

**Предусловие:** Пользователь авторизован. Импорт завершён, отображается результат с кнопками "Accept" и "Restart".

**Шаги:**
1. Нажать кнопку "Restart".

**Ожидаемый результат:**
- Запускается новый импорт с теми же параметрами.
- Таб переходит в состояние прогресса.
- Ранее импортированные задачи обновляются (upsert по origin_id).
