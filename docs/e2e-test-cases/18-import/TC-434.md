# TC-434: Импорт задач в команду с правом team.import

**Предусловие:** Пользователь авторизован, является участником команды с правом team.import. Подготовлен CSV-файл.

**Шаги:**
1. В Navigation Sidebar нажать кнопку импорта задач команды.
2. В открывшейся Import Tasks Modal загрузить CSV, указать id_column.
3. Нажать "Import".
4. Дождаться завершения.

**Ожидаемый результат:**
- Импорт завершён.
- Задачи созданы как командные (с prefix команды в ключе).
- Задачи видны в фильтрах команды.
