# TC-422: Import Completed Toast при закрытой модалке

**Предусловие:** Пользователь авторизован. Импорт запущен и модалка закрыта.

**Шаги:**
1. Дождаться завершения импорта (модалка закрыта).

**Ожидаемый результат:**
- Появляется Import Completed Toast с кратким результатом: количество импортированных и обновлённых задач.
- Toast автоматически исчезает через несколько секунд.
