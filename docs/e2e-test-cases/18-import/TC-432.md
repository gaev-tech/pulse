# TC-432: Ошибка доступа к GitHub API — неверный токен

**Предусловие:** Пользователь авторизован. Import Tasks Modal открыта на табе GitHub.

**Шаги:**
1. Заполнить поля: корректный owner и repo, но невалидный access_token.
2. Нажать "Import".

**Ожидаемый результат:**
- Отображается Error Toast с сообщением об ошибке доступа к GitHub.
- Таб остаётся в состоянии формы ввода.
