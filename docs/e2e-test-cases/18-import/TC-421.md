# TC-421: Закрытие модалки во время импорта и повторное открытие

**Предусловие:** Пользователь авторизован. Импорт запущен, прогресс отображается в модалке.

**Шаги:**
1. Нажать кнопку закрытия Import Tasks Modal во время импорта.
2. Выполнить любое другое действие в системе.
3. Снова открыть Import Tasks Modal.

**Ожидаемый результат:**
- Модалка закрывается, импорт продолжается в фоне.
- При повторном открытии модалки активный таб показывает текущий прогресс импорта.
- Прогресс обновляется в реальном времени.
