# TC-429: Валидация формы CSV — отсутствует файл

**Предусловие:** Пользователь авторизован. Import Tasks Modal открыта на табе CSV.

**Шаги:**
1. Заполнить только поле id_column: "id".
2. Не загружать файл.
3. Нажать "Import".

**Ожидаемый результат:**
- Импорт не запускается.
- Отображается сообщение об ошибке валидации: файл обязателен.
