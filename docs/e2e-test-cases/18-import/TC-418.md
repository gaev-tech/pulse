# TC-418: Кнопка импорта команды скрыта без права team.import

**Предусловие:** Пользователь авторизован, является участником команды без права team.import.

**Шаги:**
1. В Navigation Sidebar найти строку команды.
2. Осмотреть доступные кнопки в строке команды.

**Ожидаемый результат:**
- Кнопка импорта задач команды не отображается в строке команды.
