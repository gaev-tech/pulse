# TC-426: Повторный импорт из CSV — обновление существующих задач

**Предусловие:** Пользователь авторизован. Ранее выполнен импорт из CSV с id_column "id". Подготовлен обновлённый CSV с теми же id, но изменёнными title и status.

**Шаги:**
1. Открыть Import Tasks Modal, таб CSV.
2. Загрузить обновлённый CSV-файл.
3. Указать id_column: "id".
4. Нажать "Import".
5. Дождаться завершения.

**Ожидаемый результат:**
- В результате: Updated > 0, Imported = 0 (все задачи уже существуют).
- Задачи в системе обновлены: title и status изменены согласно новому CSV.
- Дублирования задач не произошло.
