# TC-416: Открытие Import Tasks Modal для личных задач

**Предусловие:** Пользователь авторизован и находится на любом экране с Navigation Sidebar.

**Шаги:**
1. В Navigation Sidebar найти кнопку импорта личных задач (рядом с разделом "Личные фильтры").
2. Нажать кнопку импорта.

**Ожидаемый результат:**
- Открывается Import Tasks Modal.
- Отображаются три таба: CSV, Jira, GitHub.
- По умолчанию активен таб CSV.
- Отображается форма ввода для CSV-импорта.
