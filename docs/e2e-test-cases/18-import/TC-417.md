# TC-417: Открытие Import Tasks Modal для командных задач

**Предусловие:** Пользователь авторизован, является участником команды с правом team.import.

**Шаги:**
1. В Navigation Sidebar найти строку команды.
2. Нажать кнопку импорта задач команды (иконка в строке команды).

**Ожидаемый результат:**
- Открывается Import Tasks Modal.
- Отображаются три таба: CSV, Jira, GitHub.
- Модалка открывается в контексте команды (импорт будет в командные задачи).
