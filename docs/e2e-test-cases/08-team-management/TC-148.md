# TC-148: Добавление нового участника в команду

**Предусловие:** Пользователь авторизован и является владельцем команды "Alpha Team". Открыта Team Edit Modal. В системе зарегистрирован пользователь "newmember@example.com", который не является участником команды.

**Шаги:**
1. В поле добавления участника ввести email "newmember@example.com".
2. Выбрать пользователя из результатов поиска.
3. Выбрать права для нового участника: `task.create`.
4. Подтвердить добавление участника.

**Ожидаемый результат:**
- Пользователь "newmember@example.com" добавлен в команду "Alpha Team".
- У нового участника автоматически есть право `view` и выбранное право `task.create`.
- Новый участник отображается в списке участников команды.
