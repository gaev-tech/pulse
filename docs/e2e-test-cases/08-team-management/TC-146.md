# TC-146: Редактирование префикса команды

**Предусловие:** Пользователь авторизован и является владельцем команды "Alpha Team" с префиксом "ALPHA". Открыта Team Edit Modal. В системе нет команды с префиксом "ALPH".

**Шаги:**
1. Очистить поле "Prefix".
2. Ввести новый префикс "ALPH".
3. Сохранить изменения.

**Ожидаемый результат:**
- Префикс команды успешно изменён на "ALPH".
- Команда доступна по новому URL `/teams/ALPH`.
- Ключи существующих задач команды обновлены (например, `ALPH-1` вместо `ALPHA-1`).
