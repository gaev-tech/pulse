# TC-143: Добавление участников при создании команды

**Предусловие:** Пользователь авторизован. Открыта Team Create Modal. В системе зарегистрирован пользователь с email "member@example.com".

**Шаги:**
1. В поле "Название" ввести "Gamma Team".
2. В поле "Prefix" ввести "GAMMA".
3. В секции добавления участников ввести email "member@example.com".
4. Выбрать пользователя из результатов поиска.
5. Убедиться, что право `view` отображается задизейблированным и включённым.
6. Выбрать дополнительное право `task.create`.
7. Нажать кнопку создания команды.

**Ожидаемый результат:**
- Команда "Gamma Team" успешно создана.
- Пользователь "member@example.com" добавлен как участник команды с правами `view` и `task.create`.
