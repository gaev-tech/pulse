# TC-150: Настройка прав участника команды

**Предусловие:** Пользователь авторизован и является владельцем команды "Alpha Team". Открыта Team Edit Modal. В команде есть участник "member@example.com" с правом `view`.

**Шаги:**
1. Найти участника "member@example.com" в списке участников.
2. Включить права `task.create`, `team.edit_labels`, `team.manage_filters`.
3. Сохранить изменения прав.

**Ожидаемый результат:**
- Участнику "member@example.com" успешно назначены права: `view`, `task.create`, `team.edit_labels`, `team.manage_filters`.
- Изменённые права отображаются в списке участников.
