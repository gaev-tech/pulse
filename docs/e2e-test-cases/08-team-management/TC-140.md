# TC-140: Валидация уникальности имени команды

**Предусловие:** Пользователь авторизован. Открыта Team Create Modal. В системе уже существует команда с именем "Alpha Team".

**Шаги:**
1. В поле "Название" ввести "Alpha Team".
2. В поле "Prefix" ввести уникальный префикс "BETA".
3. Нажать кнопку создания команды.

**Ожидаемый результат:**
- Команда не создаётся.
- Отображается ошибка валидации о том, что команда с таким именем уже существует.
