# TC-151: Изменение прав существующего участника

**Предусловие:** Пользователь авторизован и является владельцем команды "Alpha Team". Открыта Team Edit Modal. Участник "member@example.com" имеет права `view`, `task.create`, `team.edit_labels`.

**Шаги:**
1. Найти участника "member@example.com" в списке участников.
2. Снять право `team.edit_labels`.
3. Добавить право `team.manage_labels`.
4. Сохранить изменения прав.

**Ожидаемый результат:**
- У участника "member@example.com" обновлён набор прав: `view`, `task.create`, `team.manage_labels`.
- Право `team.edit_labels` успешно отозвано.
- Право `view` остаётся задизейблированным и включённым.
