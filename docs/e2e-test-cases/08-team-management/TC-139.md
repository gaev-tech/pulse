# TC-139: Создание команды с уникальным именем и префиксом

**Предусловие:** Пользователь авторизован. Открыта Team Create Modal. В системе нет команды с именем "Alpha Team" и префиксом "ALPHA".

**Шаги:**
1. В поле "Название" ввести "Alpha Team".
2. В поле "Prefix" ввести "ALPHA".
3. Нажать кнопку создания команды.

**Ожидаемый результат:**
- Команда "Alpha Team" успешно создана.
- Модальное окно закрывается.
- В разделе "Команды" Navigation Sidebar появляется новая команда "Alpha Team".
- Команда доступна по URL `/teams/ALPHA`.
