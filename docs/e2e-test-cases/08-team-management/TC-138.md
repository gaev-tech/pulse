# TC-138: Открытие Team Create Modal

**Предусловие:** Пользователь авторизован и находится на любом экране с Navigation Sidebar.

**Шаги:**
1. В Navigation Sidebar найти раздел "Команды".
2. Нажать кнопку создания команды рядом с заголовком раздела "Команды".

**Ожидаемый результат:**
- Открывается модальное окно Team Create Modal.
- Модалка содержит поля: название команды, prefix команды.
- Присутствует возможность добавить участников.
- В списке прав участника право `view` отображается задизейблированным.
