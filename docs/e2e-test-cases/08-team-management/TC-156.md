# TC-156: Удаление команды — Confirmation Popup

**Предусловие:** Пользователь авторизован и является владельцем команды "Alpha Team". Открыта Team Edit Modal.

**Шаги:**
1. Нажать кнопку удаления команды в Team Edit Modal.

**Ожидаемый результат:**
- Появляется Confirmation Popup с текстом подтверждения удаления команды.
- Popup содержит красную кнопку "Да, подтверждаю" и кнопку "Отмена".
- Команда ещё не удалена.
