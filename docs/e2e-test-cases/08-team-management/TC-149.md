# TC-149: Право view автоматически выдаётся и задизейблено при добавлении участника

**Предусловие:** Пользователь авторизован и является владельцем команды "Alpha Team". Открыта Team Edit Modal.

**Шаги:**
1. Начать добавление нового участника в команду.
2. Выбрать пользователя из результатов поиска.
3. Просмотреть список доступных прав для выдачи.

**Ожидаемый результат:**
- Право `view` отображается включённым (чекбокс активен).
- Право `view` задизейблено — его нельзя снять.
- Остальные права доступны для выбора и могут быть включены или выключены.
