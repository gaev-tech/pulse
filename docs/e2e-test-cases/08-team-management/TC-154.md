# TC-154: Отмена удаления участника из команды

**Предусловие:** Пользователь авторизован и является владельцем команды "Alpha Team". Открыта Team Edit Modal. Нажата кнопка удаления участника "member@example.com". Отображается Confirmation Popup.

**Шаги:**
1. В Confirmation Popup нажать кнопку "Отмена".

**Ожидаемый результат:**
- Confirmation Popup закрывается.
- Участник "member@example.com" остаётся в команде "Alpha Team".
- Права участника не изменились.
