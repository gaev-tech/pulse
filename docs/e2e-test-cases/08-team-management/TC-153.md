# TC-153: Удаление участника из команды — Confirmation Popup

**Предусловие:** Пользователь авторизован и является владельцем команды "Alpha Team". Открыта Team Edit Modal. В команде есть участник "member@example.com".

**Шаги:**
1. Найти участника "member@example.com" в списке участников.
2. Нажать кнопку удаления участника.

**Ожидаемый результат:**
- Появляется Confirmation Popup с текстом подтверждения удаления участника.
- Popup содержит красную кнопку "Да, подтверждаю" и кнопку "Отмена".
- Участник ещё не удалён из команды.
