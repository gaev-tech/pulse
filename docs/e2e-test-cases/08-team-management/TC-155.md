# TC-155: Каскадное удаление прав при удалении участника из команды

**Предусловие:** Пользователь авторизован и является владельцем команды "Alpha Team". Участник "member@example.com" имеет право `share` по командной задаче ALPHA-1 и расшарил эту задачу пользователю "third@example.com" с правом `view`. Открыта Team Edit Modal.

**Шаги:**
1. Найти участника "member@example.com" в списке участников.
2. Нажать кнопку удаления участника.
3. В Confirmation Popup нажать красную кнопку "Да, подтверждаю".

**Ожидаемый результат:**
- Участник "member@example.com" удалён из команды "Alpha Team".
- Все командные права "member@example.com" отозваны.
- Все права, выданные "member@example.com" другим пользователям по командным задачам, каскадно удалены.
- Пользователь "third@example.com" больше не имеет доступа к задаче ALPHA-1.
