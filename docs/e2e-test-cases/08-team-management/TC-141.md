# TC-141: Валидация уникальности префикса команды

**Предусловие:** Пользователь авторизован. Открыта Team Create Modal. В системе уже существует команда с префиксом "ALPHA".

**Шаги:**
1. В поле "Название" ввести уникальное имя "Beta Team".
2. В поле "Prefix" ввести "ALPHA".
3. Нажать кнопку создания команды.

**Ожидаемый результат:**
- Команда не создаётся.
- Отображается ошибка валидации о том, что команда с таким префиксом уже существует.
