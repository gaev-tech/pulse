# TC-147: Просмотр списка участников команды

**Предусловие:** Пользователь авторизован и является владельцем команды "Alpha Team". В команде есть два участника: "user1@example.com" и "user2@example.com". Открыта Team Edit Modal.

**Шаги:**
1. Просмотреть список участников команды в Team Edit Modal.

**Ожидаемый результат:**
- Отображается список участников команды, включая "user1@example.com" и "user2@example.com".
- Для каждого участника отображаются назначенные ему права.
- Для каждого участника доступны действия: изменение прав, удаление из команды.
