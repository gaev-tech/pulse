# TC-142: Валидация обязательных полей при создании команды

**Предусловие:** Пользователь авторизован. Открыта Team Create Modal.

**Шаги:**
1. Оставить поле "Название" пустым.
2. Оставить поле "Prefix" пустым.
3. Нажать кнопку создания команды.

**Ожидаемый результат:**
- Команда не создаётся.
- Отображаются ошибки валидации для обязательных полей "Название" и "Prefix".
