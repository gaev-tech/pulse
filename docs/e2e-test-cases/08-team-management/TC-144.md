# TC-144: Открытие Team Edit Modal

**Предусловие:** Пользователь авторизован и является владельцем команды "Alpha Team" с префиксом "ALPHA".

**Шаги:**
1. В Navigation Sidebar найти команду "Alpha Team" в разделе "Команды".
2. Перейти на Team Activity Screen команды "Alpha Team".
3. Открыть Team Edit Modal (нажать кнопку редактирования команды).

**Ожидаемый результат:**
- Открывается модальное окно Team Edit Modal.
- Поле "Название" содержит текущее значение "Alpha Team".
- Поле "Prefix" содержит текущее значение "ALPHA".
- Отображается список участников команды с их правами.
- Присутствует кнопка удаления команды.
