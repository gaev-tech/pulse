# TC-159: Использование prefix в URL команды (/teams/PREFIX)

**Предусловие:** Пользователь авторизован. В системе существует команда "Alpha Team" с префиксом "ALPHA". Пользователь является участником этой команды.

**Шаги:**
1. В адресной строке браузера ввести URL `/teams/ALPHA`.
2. Перейти по введённому URL.

**Ожидаемый результат:**
- Открывается Team Activity Screen команды "Alpha Team".
- URL в адресной строке содержит `/teams/ALPHA`.
- Отображается лента событий команды "Alpha Team".
