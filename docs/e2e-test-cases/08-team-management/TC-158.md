# TC-158: Подтверждение удаления команды

**Предусловие:** Пользователь авторизован и является владельцем команды "Alpha Team" с префиксом "ALPHA". Открыта Team Edit Modal. Нажата кнопка удаления команды. Отображается Confirmation Popup.

**Шаги:**
1. В Confirmation Popup нажать красную кнопку "Да, подтверждаю".

**Ожидаемый результат:**
- Команда "Alpha Team" удалена из системы.
- Confirmation Popup и Team Edit Modal закрываются.
- Команда "Alpha Team" исчезает из раздела "Команды" в Navigation Sidebar.
- URL `/teams/ALPHA` более недоступен.
