# TC-157: Отмена удаления команды

**Предусловие:** Пользователь авторизован и является владельцем команды "Alpha Team". Открыта Team Edit Modal. Нажата кнопка удаления команды. Отображается Confirmation Popup.

**Шаги:**
1. В Confirmation Popup нажать кнопку "Отмена".

**Ожидаемый результат:**
- Confirmation Popup закрывается.
- Команда "Alpha Team" не удалена и продолжает отображаться в Navigation Sidebar.
- Team Edit Modal остаётся открытой.
