# TC-152: Пользователь может выдать только те командные права, которые есть у него

**Предусловие:** Пользователь авторизован и является участником команды "Alpha Team" с правами `view`, `task.create`, `team.manage_filters`. Открыта Team Edit Modal.

**Шаги:**
1. Начать добавление нового участника в команду.
2. Выбрать пользователя из результатов поиска.
3. Просмотреть список доступных прав для выдачи.

**Ожидаемый результат:**
- Доступны для выдачи только права, которые есть у текущего пользователя: `task.create`, `team.manage_filters`.
- Права, которых нет у текущего пользователя (`team.manage_owners`, `team.edit_labels`, `team.manage_labels`, `team.manage_automations`), недоступны для выдачи.
- Право `view` отображается задизейблированным и включённым.
