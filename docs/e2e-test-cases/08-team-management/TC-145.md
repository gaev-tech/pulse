# TC-145: Редактирование имени команды

**Предусловие:** Пользователь авторизован и является владельцем команды "Alpha Team". Открыта Team Edit Modal.

**Шаги:**
1. Очистить поле "Название".
2. Ввести новое название "Alpha Team Renamed".
3. Сохранить изменения.

**Ожидаемый результат:**
- Название команды успешно изменено на "Alpha Team Renamed".
- В Navigation Sidebar отображается обновлённое название "Alpha Team Renamed".
