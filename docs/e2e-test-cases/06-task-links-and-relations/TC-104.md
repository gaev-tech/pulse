# TC-104: Валидация обязательного поля URL при добавлении ссылки

**Предусловие:** Пользователь авторизован, открыт Related Links Popup задачи «ACME-42». У пользователя есть право `edit.links`.

**Шаги:**
1. Оставить поле «URL» пустым.
2. В поле «Название» ввести «Тестовая ссылка».
3. Нажать кнопку добавления ссылки.

**Ожидаемый результат:**
- Ссылка не добавлена.
- Отображается сообщение об ошибке валидации: поле URL обязательно для заполнения.
