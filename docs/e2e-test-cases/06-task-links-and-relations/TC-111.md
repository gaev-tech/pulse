# TC-111: Удаление связанной задачи

**Предусловие:** Пользователь авторизован, открыт Related Tasks Popup задачи «ACME-42». В списке связанных задач есть «ACME-10: Настройка CI/CD». У пользователя есть право `edit.relations`.

**Шаги:**
1. Кликнуть на кнопку удаления связи с задачей «ACME-10: Настройка CI/CD».

**Ожидаемый результат:**
- Задача «ACME-10» удалена из списка связанных задач «ACME-42».
- Задача «ACME-42» удалена из списка связанных задач «ACME-10» (двусторонность).
- В ленте событий задачи появилось событие `task.relations_changed`.
