# TC-106: Удаление внешней ссылки -- Confirmation Popup

**Предусловие:** Пользователь авторизован, открыт Related Links Popup задачи «ACME-42». В списке ссылок есть ссылка «Макет в Figma». У пользователя есть право `edit.links`.

**Шаги:**
1. Кликнуть на кнопку удаления ссылки «Макет в Figma».
2. В Confirmation Popup нажать красную кнопку «Да, подтверждаю».

**Ожидаемый результат:**
- Ссылка «Макет в Figma» удалена из списка.
- В ленте событий задачи появилось событие `task.links_changed`.
