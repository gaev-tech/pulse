# TC-105: Редактирование внешней ссылки

**Предусловие:** Пользователь авторизован, открыт Related Links Popup задачи «ACME-42». В списке ссылок есть ссылка «Макет в Figma» с URL `https://figma.com/design/123`. У пользователя есть право `edit.links`.

**Шаги:**
1. Кликнуть на кнопку редактирования ссылки «Макет в Figma».
2. Изменить название на «Обновлённый макет».
3. Изменить URL на `https://figma.com/design/456`.
4. Сохранить изменения.

**Ожидаемый результат:**
- Ссылка в списке обновлена: название «Обновлённый макет», URL `https://figma.com/design/456`.
- В ленте событий задачи появилось событие `task.links_changed`.
