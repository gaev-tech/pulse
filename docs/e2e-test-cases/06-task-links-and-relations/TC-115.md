# TC-115: Обнаружение цикла блокирования -- восклицательный знак, задача недоступна для выбора

**Предусловие:** Пользователь авторизован, открыт Blocking Tasks Popup задачи «ACME-42». Задача «ACME-5» блокирует «ACME-42», а «ACME-42» блокирует «ACME-30». Добавление «ACME-30» как блокирующей для «ACME-5» создало бы цикл.

**Шаги:**
1. Открыть Blocking Tasks Popup задачи «ACME-5».
2. В поле поиска ввести «ACME-30».
3. Просмотреть результаты поиска.

**Ожидаемый результат:**
- Задача «ACME-30» отображается в результатах поиска с восклицательным знаком.
- Задача «ACME-30» недоступна для выбора (нельзя кликнуть для добавления).
- Цикл блокирования обнаружен и предотвращён на уровне UI.
