# TC-116: Blocking Toast при попытке создать цикл блокирования

**Предусловие:** Пользователь авторизован. Задача «ACME-42» блокирует задачу «ACME-30». Пользователь пытается добавить «ACME-42» как блокирующую для «ACME-30» через API или обход UI.

**Шаги:**
1. Открыть Blocking Tasks Popup задачи «ACME-30».
2. Попытаться добавить задачу «ACME-42» как блокирующую (задача, которая уже блокируется «ACME-30»).

**Ожидаемый результат:**
- Отображается Blocking Toast с сообщением о том, что задача не может быть заблокирована, так как находится в цепочке блокирующих задач.
- Блокирующая задача не добавлена.
- Цикл блокирования не создан.
