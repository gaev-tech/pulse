# TC-114: Добавление блокирующей задачи

**Предусловие:** Пользователь авторизован, открыт Blocking Tasks Popup задачи «ACME-42». У пользователя есть право `edit.blocking`. Существует задача «ACME-5: Подготовка инфраструктуры», не создающая цикл блокирования.

**Шаги:**
1. В поле поиска ввести «ACME-5».
2. В результатах поиска кликнуть на задачу «ACME-5: Подготовка инфраструктуры».

**Ожидаемый результат:**
- Задача «ACME-5: Подготовка инфраструктуры» добавлена в список блокирующих задач «ACME-42».
- В ленте событий задачи «ACME-42» появилось событие `task.blocking_changed`.
