# TC-113: Поиск блокирующей задачи

**Предусловие:** Пользователь авторизован, открыт Blocking Tasks Popup задачи «ACME-42». Существуют задачи «ACME-5: Подготовка инфраструктуры» и «ACME-8: Создание базы данных».

**Шаги:**
1. В поле поиска ввести «инфраструктур».
2. Просмотреть результаты поиска.

**Ожидаемый результат:**
- В результатах поиска отображается задача «ACME-5: Подготовка инфраструктуры» (поиск по названию, contains).
- Результаты сортируются: сначала точное совпадение ключа или названия, затем по дате последнего открытия.
