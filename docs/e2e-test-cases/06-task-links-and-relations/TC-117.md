# TC-117: Удаление блокирующей задачи

**Предусловие:** Пользователь авторизован, открыт Blocking Tasks Popup задачи «ACME-42». В списке блокирующих задач есть «ACME-5: Подготовка инфраструктуры». У пользователя есть право `edit.blocking`.

**Шаги:**
1. Кликнуть на кнопку удаления блокирующей задачи «ACME-5: Подготовка инфраструктуры».

**Ожидаемый результат:**
- Задача «ACME-5» удалена из списка блокирующих задач «ACME-42».
- В ленте событий задачи появилось событие `task.blocking_changed`.
