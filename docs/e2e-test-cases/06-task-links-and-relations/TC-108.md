# TC-108: Поиск связанной задачи по ключу, названию, описанию

**Предусловие:** Пользователь авторизован, открыт Related Tasks Popup задачи «ACME-42». Существуют задачи «ACME-10: Настройка CI/CD» и «ACME-15: Деплой на стейджинг».

**Шаги:**
1. В поле поиска ввести «CI/CD».
2. Просмотреть результаты поиска.
3. Очистить поле поиска.
4. В поле поиска ввести «ACME-10».
5. Просмотреть результаты поиска.

**Ожидаемый результат:**
- При поиске по «CI/CD» в результатах отображается задача «ACME-10: Настройка CI/CD» (поиск по названию, contains).
- При поиске по «ACME-10» в результатах отображается задача «ACME-10: Настройка CI/CD» (поиск по ключу).
- Результаты сортируются: сначала точное совпадение ключа или названия, затем по дате последнего открытия.
