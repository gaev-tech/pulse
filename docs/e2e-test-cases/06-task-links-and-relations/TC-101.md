# TC-101: Открытие Related Links Popup

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи «ACME-42». У пользователя есть право `edit.links`.

**Шаги:**
1. В правой части Task Screen найти раздел «Ссылки».
2. Кликнуть на раздел «Ссылки» для открытия Related Links Popup.

**Ожидаемый результат:**
- Открывается Related Links Popup.
- Popup содержит список внешних ссылок задачи (пустой, если ссылок нет) и элементы для добавления новой ссылки.
