# TC-107: Открытие Related Tasks Popup

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи «ACME-42». У пользователя есть право `edit.relations`.

**Шаги:**
1. В правой части Task Screen найти раздел «Связанные задачи».
2. Кликнуть на раздел «Связанные задачи» для открытия Related Tasks Popup.

**Ожидаемый результат:**
- Открывается Related Tasks Popup.
- Popup содержит список связанных задач (пустой, если связей нет) и поле поиска для добавления новой связи.
