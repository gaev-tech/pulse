# TC-103: Добавление внешней ссылки только с URL (без названия)

**Предусловие:** Пользователь авторизован, открыт Related Links Popup задачи «ACME-42». У пользователя есть право `edit.links`.

**Шаги:**
1. В поле «URL» ввести `https://docs.google.com/document/d/abc`.
2. Поле «Название» оставить пустым.
3. Нажать кнопку добавления ссылки.

**Ожидаемый результат:**
- В списке ссылок появляется новая запись с URL `https://docs.google.com/document/d/abc`.
- Ссылка отображается с URL в качестве текста, так как название не указано.
- В ленте событий задачи появилось событие `task.links_changed`.
