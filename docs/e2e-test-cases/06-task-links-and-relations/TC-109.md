# TC-109: Добавление связанной задачи

**Предусловие:** Пользователь авторизован, открыт Related Tasks Popup задачи «ACME-42». У пользователя есть право `edit.relations`. Существует задача «ACME-10: Настройка CI/CD».

**Шаги:**
1. В поле поиска ввести «ACME-10».
2. В результатах поиска кликнуть на задачу «ACME-10: Настройка CI/CD».

**Ожидаемый результат:**
- Задача «ACME-10: Настройка CI/CD» добавлена в список связанных задач.
- В ленте событий задачи «ACME-42» появилось событие `task.relations_changed`.
