# TC-110: Двусторонность связи -- отображение у обеих задач

**Предусловие:** Пользователь авторизован. Задача «ACME-42» связана с задачей «ACME-10» (связь добавлена из «ACME-42»).

**Шаги:**
1. Открыть Task Screen задачи «ACME-42».
2. Проверить раздел «Связанные задачи» — убедиться, что отображается «ACME-10».
3. Открыть Task Screen задачи «ACME-10».
4. Проверить раздел «Связанные задачи» — убедиться, что отображается «ACME-42».

**Ожидаемый результат:**
- В задаче «ACME-42» в разделе «Связанные задачи» отображается «ACME-10».
- В задаче «ACME-10» в разделе «Связанные задачи» отображается «ACME-42».
- Связь двусторонняя: отображается у обеих задач.
