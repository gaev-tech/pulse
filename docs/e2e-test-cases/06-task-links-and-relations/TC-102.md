# TC-102: Добавление внешней ссылки с URL и названием

**Предусловие:** Пользователь авторизован, открыт Related Links Popup задачи «ACME-42». У пользователя есть право `edit.links`.

**Шаги:**
1. В поле «URL» ввести `https://figma.com/design/123`.
2. В поле «Название» ввести «Макет в Figma».
3. Нажать кнопку добавления ссылки.

**Ожидаемый результат:**
- В списке ссылок появляется новая запись с названием «Макет в Figma» и URL `https://figma.com/design/123`.
- В ленте событий задачи появилось событие `task.links_changed`.
