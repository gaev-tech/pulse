# TC-112: Открытие Blocking Tasks Popup

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи «ACME-42». У пользователя есть право `edit.blocking`.

**Шаги:**
1. В правой части Task Screen найти раздел «Блокирующие задачи».
2. Кликнуть на раздел «Блокирующие задачи» для открытия Blocking Tasks Popup.

**Ожидаемый результат:**
- Открывается Blocking Tasks Popup.
- Popup содержит список блокирующих задач (пустой, если блокеров нет) и поле поиска для добавления блокирующей задачи.
