# TC-273: Переключение режима фильтра (простой/RSQL) в Table Settings

**Предусловие:** Пользователь авторизован, находится на Filter Results Screen. Фильтр работает в простом режиме, в критериях указан статус «opened» и ответственный.

**Шаги:**
1. Нажать кнопку настроек таблицы для открытия Table Settings Popup.
2. Переключить режим фильтрации с «Простой» на «RSQL».
3. Закрыть Table Settings Popup.

**Ожидаемый результат:**
- Строка критериев на Filter Results Screen заменяется на textarea для ввода RSQL-выражения.
- Ранее введённые критерии простого режима (статус, ответственный) сбрасываются.
- Textarea пуста и готова к вводу RSQL-выражения.
