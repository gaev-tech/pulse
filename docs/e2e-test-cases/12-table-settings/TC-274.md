# TC-274: Сохранение настроек таблицы на бэкенде

**Предусловие:** Пользователь авторизован, находится на Filter Results Screen.

**Шаги:**
1. Нажать кнопку настроек таблицы для открытия Table Settings Popup.
2. Включить колонку «Дата создания».
3. Перетащить колонку «Дата создания» на первую позицию среди необязательных колонок.
4. Закрыть Table Settings Popup.
5. Перейти на другой экран (например, Personal Activity Screen) через Navigation Sidebar.
6. Вернуться на Filter Results Screen того же фильтра, нажав на него в Navigation Sidebar.

**Ожидаемый результат:**
- Колонка «Дата создания» отображается в таблице на первой позиции среди необязательных колонок.
- Настройки сохранились на бэкенде и корректно восстановились при повторном открытии фильтра.
