# TC-270: Включение необязательной колонки

**Предусловие:** Пользователь авторизован, находится на Filter Results Screen. Колонка «Ответственный» отключена в настройках таблицы.

**Шаги:**
1. Нажать кнопку настроек таблицы для открытия Table Settings Popup.
2. В списке необязательных колонок найти «Ответственный» и включить её (установить переключатель в активное положение).
3. Закрыть Table Settings Popup.

**Ожидаемый результат:**
- В таблице результатов фильтра появляется колонка «Ответственный».
- Колонка отображается в соответствии с текущим порядком необязательных колонок.
- Данные в колонке корректно заполнены для каждой задачи.
