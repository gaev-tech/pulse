# TC-272: Перетаскивание колонки для изменения порядка (drag-and-drop)

**Предусловие:** Пользователь авторизован, находится на Filter Results Screen. В настройках таблицы включены колонки «Метки», «Ответственный», «Статус» (в таком порядке).

**Шаги:**
1. Нажать кнопку настроек таблицы для открытия Table Settings Popup.
2. Перетащить колонку «Статус» выше колонки «Метки» с помощью drag-and-drop.
3. Закрыть Table Settings Popup.

**Ожидаемый результат:**
- Порядок необязательных колонок в таблице изменился: «Статус», «Метки», «Ответственный».
- Обязательные колонки (чекбокс, стрелка раскрытия, ключ, название) остаются на своих позициях слева.
- Данные в колонках корректно соответствуют заголовкам.
