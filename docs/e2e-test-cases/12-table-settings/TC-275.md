# TC-275: Настройки таблицы применяются для всех пользователей с доступом к фильтру

**Предусловие:** Существует командный фильтр команды «ACME». Пользователь A и Пользователь B являются участниками команды «ACME» и имеют доступ к фильтру.

**Шаги:**
1. Пользователь A открывает Filter Results Screen командного фильтра.
2. Пользователь A открывает Table Settings Popup и включает колонки «Ответственный» и «Дата изменения», выключает колонку «Метки».
3. Пользователь A закрывает Table Settings Popup.
4. Пользователь B открывает Filter Results Screen того же командного фильтра.

**Ожидаемый результат:**
- Пользователь B видит таблицу с колонками «Ответственный» и «Дата изменения» включёнными.
- Колонка «Метки» у Пользователя B отключена.
- Порядок колонок у Пользователя B совпадает с порядком, установленным Пользователем A.
