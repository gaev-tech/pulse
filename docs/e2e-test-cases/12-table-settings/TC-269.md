# TC-269: Открытие Table Settings Popup

**Предусловие:** Пользователь авторизован, существует фильтр с результатами. Пользователь находится на Filter Results Screen.

**Шаги:**
1. На верхней панели Filter Results Screen нажать кнопку настроек таблицы (Table Settings Popup).

**Ожидаемый результат:**
- Открывается Table Settings Popup.
- В попапе отображается список необязательных колонок с возможностью включения/отключения.
- Отображается переключатель режима фильтра (простой/RSQL).
- Порядок колонок соответствует текущим сохранённым настройкам фильтра.
