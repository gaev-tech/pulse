# TC-271: Отключение необязательной колонки

**Предусловие:** Пользователь авторизован, находится на Filter Results Screen. Колонка «Статус» включена и отображается в таблице.

**Шаги:**
1. Нажать кнопку настроек таблицы для открытия Table Settings Popup.
2. В списке необязательных колонок найти «Статус» и отключить её (установить переключатель в неактивное положение).
3. Закрыть Table Settings Popup.

**Ожидаемый результат:**
- Колонка «Статус» исчезает из таблицы результатов фильтра.
- Остальные колонки сохраняют свой порядок.
- Обязательные колонки (чекбокс, стрелка раскрытия, ключ, название) остаются на месте.
