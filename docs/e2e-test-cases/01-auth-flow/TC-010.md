# TC-010: Сохранение сессии после обновления страницы

**Предусловие:** Пользователь авторизован в системе. Открыт Personal Activity Screen.

**Шаги:**
1. Нажать F5 или кнопку обновления страницы в браузере.

**Ожидаемый результат:**
- Страница перезагружается.
- Пользователь остаётся авторизованным.
- Отображается тот же экран (Personal Activity Screen).
- В Navigation Sidebar отображаются аватарка и имя текущего пользователя.
