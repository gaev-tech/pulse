# TC-005: Повторная отправка magic link

**Предусловие:** Браузер открыт на Auth Screen. Пользователь уже запросил magic link, но не перешёл по нему.

**Шаги:**
1. Ввести тот же email в поле ввода на Auth Screen.
2. Нажать кнопку отправки повторно.
3. Открыть почтовый ящик указанного email.
4. Найти новое письмо с magic link от системы Pulse.
5. Перейти по ссылке из нового письма.

**Ожидаемый результат:**
- Новый magic link успешно отправлен на email.
- Переход по новому magic link авторизует пользователя.
- Открывается Personal Activity Screen.
