# TC-011: Истечение access token и обновление через refresh token

**Предусловие:** Пользователь авторизован в системе. Access token истёк (прошло более 15 минут). Refresh token действителен (прошло менее 30 дней).

**Шаги:**
1. Дождаться истечения access token (15 минут).
2. Выполнить любое действие, требующее запроса к API (например, перейти на Personal Activity Screen).

**Ожидаемый результат:**
- Система автоматически обновляет access token с помощью refresh token.
- Новый access token сохраняется в localStorage.
- Пользователь остаётся авторизованным.
- Запрос к API выполняется успешно — данные отображаются без ошибок.
