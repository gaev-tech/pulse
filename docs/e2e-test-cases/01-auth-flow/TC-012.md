# TC-012: Истечение refresh token — редирект на Auth Screen

**Предусловие:** Пользователь авторизован в системе. Refresh token истёк (прошло более 30 дней).

**Шаги:**
1. Дождаться истечения refresh token (30 дней).
2. Выполнить любое действие, требующее запроса к API (например, перейти на Personal Activity Screen).

**Ожидаемый результат:**
- Система не может обновить access token — refresh token недействителен.
- JWT-пара удаляется из localStorage.
- Пользователь перенаправляется на Auth Screen.
