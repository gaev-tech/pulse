# TC-004: Валидация пустого поля email

**Предусловие:** Браузер открыт на Auth Screen.

**Шаги:**
1. Оставить поле email пустым на Auth Screen.
2. Нажать кнопку отправки.

**Ожидаемый результат:**
- Отображается сообщение об ошибке — поле email обязательно для заполнения.
- Magic link не отправляется.
- Пользователь остаётся на Auth Screen.
