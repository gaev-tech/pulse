# TC-009: Редирект неавторизованного пользователя на Auth Screen

**Предусловие:** Пользователь не авторизован (JWT-пара отсутствует в localStorage).

**Шаги:**
1. Открыть в браузере URL любого защищённого экрана приложения (например, Personal Activity Screen).

**Ожидаемый результат:**
- Пользователь автоматически перенаправлен на Auth Screen.
- Защищённый экран не отображается.
