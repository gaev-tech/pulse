# TC-003: Валидация невалидного email при входе

**Предусловие:** Браузер открыт на Auth Screen.

**Шаги:**
1. Ввести невалидный email (например, "test@", "abc", "user@.com") в поле ввода на Auth Screen.
2. Нажать кнопку отправки.

**Ожидаемый результат:**
- Отображается сообщение об ошибке валидации email.
- Magic link не отправляется.
- Пользователь остаётся на Auth Screen.
