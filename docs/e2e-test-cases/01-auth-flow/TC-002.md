# TC-002: Регистрация нового пользователя через magic link

**Предусловие:** Пользователь не зарегистрирован в системе. Браузер открыт на Auth Screen.

**Шаги:**
1. Ввести новый email в поле ввода на Auth Screen.
2. Нажать кнопку отправки.
3. Открыть почтовый ящик указанного email.
4. Найти письмо с magic link от системы Pulse.
5. Перейти по ссылке из письма.

**Ожидаемый результат:**
- В системе создан новый пользователь с указанным email.
- Пользователь авторизован в системе.
- Открывается Personal Activity Screen.
- В Navigation Sidebar отображаются аватарка и имя пользователя.
- В localStorage сохраняется JWT-пара (access token + refresh token).
