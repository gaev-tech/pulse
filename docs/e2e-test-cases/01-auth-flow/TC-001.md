# TC-001: Вход в систему по email через magic link

**Предусловие:** Пользователь зарегистрирован в системе. Браузер открыт на Auth Screen. Пользователь не авторизован.

**Шаги:**
1. Ввести зарегистрированный email в поле ввода на Auth Screen.
2. Нажать кнопку отправки.
3. Открыть почтовый ящик указанного email.
4. Найти письмо с magic link от системы Pulse.
5. Перейти по ссылке из письма.

**Ожидаемый результат:**
- После перехода по magic link пользователь авторизован в системе.
- Открывается Personal Activity Screen.
- В Navigation Sidebar отображаются аватарка и имя пользователя.
- В localStorage сохраняется JWT-пара (access token + refresh token).
