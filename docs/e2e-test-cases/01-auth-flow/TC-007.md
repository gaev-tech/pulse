# TC-007: Одновременная авторизация нескольких аккаунтов

**Предусловие:** Пользователь авторизован под первым аккаунтом (user1@example.com). Существует второй зарегистрированный аккаунт (user2@example.com).

**Шаги:**
1. Нажать на аватарку и имя пользователя в нижней части Navigation Sidebar — открывается Profile Popup.
2. Нажать кнопку "Добавить пользователя" в конце списка авторизованных пользователей.
3. На Auth Screen ввести email второго аккаунта (user2@example.com).
4. Нажать кнопку отправки.
5. Открыть почтовый ящик user2@example.com и перейти по magic link.

**Ожидаемый результат:**
- Оба аккаунта авторизованы на клиенте.
- В localStorage хранятся JWT-пары обоих пользователей.
- После авторизации второго аккаунта открывается Personal Activity Screen второго пользователя.
- В Profile Popup отображаются оба пользователя в списке авторизованных.
