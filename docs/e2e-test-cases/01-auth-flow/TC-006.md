# TC-006: Переход по истёкшей magic link

**Предусловие:** Пользователь запросил magic link. Срок действия ссылки истёк.

**Шаги:**
1. Открыть почтовый ящик с полученным ранее письмом.
2. Перейти по magic link, срок действия которой истёк.

**Ожидаемый результат:**
- Пользователь не авторизован.
- Отображается сообщение об ошибке — ссылка истекла.
- Пользователь перенаправляется на Auth Screen с возможностью запросить новую ссылку.
