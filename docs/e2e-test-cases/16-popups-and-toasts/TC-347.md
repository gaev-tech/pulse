# TC-347: Confirmation Popup при удалении фильтра

**Предусловие:** Пользователь авторизован. Существует личный фильтр «Открытые задачи». Пользователь находится на Filter Results Screen этого фильтра.

**Шаги:**
1. Нажать кнопку «...» на верхней панели Filter Results Screen.
2. В открывшемся Filter Popup нажать кнопку удаления фильтра.
3. Убедиться, что открылся Confirmation Popup.
4. Нажать красную кнопку «Да, подтверждаю».

**Ожидаемый результат:**
- После шага 2: открывается Confirmation Popup с текстом подтверждения удаления фильтра.
- Popup содержит красную кнопку «Да, подтверждаю» и кнопку «Отмена».
- После шага 4: фильтр «Открытые задачи» удалён из системы. Пользователь перенаправлен с Filter Results Screen.
