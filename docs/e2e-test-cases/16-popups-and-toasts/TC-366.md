# TC-366: Error Toast — отображение тоста при ошибке запроса к бэкенду

**Предусловие:** Пользователь авторизован. Бэкенд временно недоступен или возвращает ошибку на запрос обновления задачи.

**Шаги:**
1. Открыть существующую задачу в Task Screen.
2. Изменить статус задачи на другой (например, с «Open» на «In Progress»).
3. Наблюдать за реакцией интерфейса при неуспешном ответе бэкенда.

**Ожидаемый результат:**
- В верхней части экрана появляется Error Toast с сообщением об ошибке.
- Тост содержит текст, описывающий произошедшую ошибку.
- Статус задачи не изменяется — остаётся прежним.
