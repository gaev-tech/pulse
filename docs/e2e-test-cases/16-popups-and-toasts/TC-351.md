# TC-351: Confirmation Popup при удалении вложения

**Предусловие:** Пользователь авторизован. Существует задача с ключом `ACME-20`, к которой прикреплено вложение «report.pdf». Пользователь имеет право `edit.attachments` на задачу. Задача открыта в Task Screen или Task Modal.

**Шаги:**
1. Найти вложение «report.pdf» в разделе вложений задачи.
2. Нажать кнопку удаления вложения.
3. Убедиться, что открылся Confirmation Popup.
4. Нажать красную кнопку «Да, подтверждаю».

**Ожидаемый результат:**
- После шага 2: открывается Confirmation Popup с текстом подтверждения удаления вложения.
- Popup содержит красную кнопку «Да, подтверждаю» и кнопку «Отмена».
- После шага 4: вложение «report.pdf» удалено из задачи. Confirmation Popup закрывается, вложение исчезает из списка.
