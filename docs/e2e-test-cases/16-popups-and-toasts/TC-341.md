**Предусловие:** Пользователь авторизован. Существует личная задача с ключом `#10`. Пользователь является владельцем задачи. Открыт Confirmation Popup удаления задачи.

1. Нажать красную кнопку «Да, подтверждаю» в Confirmation Popup.

**Ожидаемый результат:**
- Confirmation Popup закрывается.
- Операция удаления выполняется — задача `#10` удалена.
- Кнопка «Да, подтверждаю» визуально выделена красным цветом, чтобы привлечь внимание к опасности операции.
