# TC-340: Confirmation Popup: текст подтверждения операции

**Предусловие:** Пользователь авторизован. Существует личная задача с ключом `#10`. Пользователь является владельцем задачи.

**Шаги:**
1. Открыть задачу `#10` в Task Screen или Task Modal.
2. Нажать кнопку удаления задачи.

**Ожидаемый результат:**
- Открывается Confirmation Popup.
- Popup содержит текст «Вы подтверждаете <операция>?», где <операция> описывает удаление задачи.
- Popup содержит красную кнопку «Да, подтверждаю» и кнопку «Отмена».
