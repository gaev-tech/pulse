# TC-345: Confirmation Popup при удалении задачи

**Предусловие:** Пользователь авторизован. Существует личная задача с ключом `#15` и названием «Тестовая задача». Пользователь является владельцем задачи. Задача открыта в Task Screen или Task Modal.

**Шаги:**
1. Нажать кнопку удаления задачи.
2. Убедиться, что открылся Confirmation Popup.
3. Нажать красную кнопку «Да, подтверждаю».

**Ожидаемый результат:**
- После шага 1: открывается Confirmation Popup с текстом подтверждения удаления задачи.
- Popup содержит красную кнопку «Да, подтверждаю» и кнопку «Отмена».
- После шага 3: задача `#15` удалена из системы. Confirmation Popup закрывается.
