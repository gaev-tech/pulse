# TC-348: Confirmation Popup при удалении метки

**Предусловие:** Пользователь авторизован. Существует личная метка «Срочно» красного цвета. Открыт интерфейс управления личными метками (кнопка в Navigation Sidebar).

**Шаги:**
1. Найти метку «Срочно» в списке меток.
2. Нажать кнопку удаления метки «Срочно».
3. Убедиться, что открылся Confirmation Popup.
4. Нажать красную кнопку «Да, подтверждаю».

**Ожидаемый результат:**
- После шага 2: открывается Confirmation Popup с текстом подтверждения удаления метки.
- Popup содержит красную кнопку «Да, подтверждаю» и кнопку «Отмена».
- После шага 4: метка «Срочно» удалена из системы. Confirmation Popup закрывается.
