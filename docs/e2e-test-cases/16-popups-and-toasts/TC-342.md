# TC-342: Confirmation Popup: кнопка «Отмена» — закрытие без действия

**Предусловие:** Пользователь авторизован. Существует личная задача с ключом `#10`. Пользователь является владельцем задачи. Открыт Confirmation Popup удаления задачи.

**Шаги:**
1. Нажать кнопку «Отмена» в Confirmation Popup.

**Ожидаемый результат:**
- Confirmation Popup закрывается.
- Операция удаления не выполняется — задача `#10` по-прежнему существует.
- Пользователь возвращается к предыдущему состоянию экрана без изменений.
