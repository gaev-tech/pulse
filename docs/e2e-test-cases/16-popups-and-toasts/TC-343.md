# TC-343: Task Link Toast: автоматическое исчезновение

**Предусловие:** Пользователь авторизован. Существует задача с ключом `ACME-7`. Пользователь имеет право `view` на задачу. Задача открыта в Task Screen или Task Modal.

**Шаги:**
1. Нажать кнопку «Скопировать ссылку на задачу» в интерфейсе задачи.
2. Дождаться появления Task Link Toast.
3. Не совершать никаких действий и наблюдать за toast-уведомлением.

**Ожидаемый результат:**
- После нажатия кнопки появляется Task Link Toast с сообщением о том, что ссылка на задачу скопирована.
- Toast автоматически исчезает через несколько секунд без действий пользователя.
- Ссылка на задачу скопирована в буфер обмена.
