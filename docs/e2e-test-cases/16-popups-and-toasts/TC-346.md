# TC-346: Confirmation Popup при удалении команды

**Предусловие:** Пользователь авторизован. Существует команда «Backend» с префиксом `BACK`. Пользователь является владельцем команды. Открыт Team Edit Modal для команды «Backend».

**Шаги:**
1. Нажать кнопку удаления команды в Team Edit Modal.
2. Убедиться, что открылся Confirmation Popup.
3. Нажать красную кнопку «Да, подтверждаю».

**Ожидаемый результат:**
- После шага 1: открывается Confirmation Popup с текстом подтверждения удаления команды.
- Popup содержит красную кнопку «Да, подтверждаю» и кнопку «Отмена».
- После шага 3: команда «Backend» удалена из системы. Confirmation Popup и Team Edit Modal закрываются.
