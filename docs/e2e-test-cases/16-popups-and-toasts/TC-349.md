# TC-349: Confirmation Popup при удалении автоматизации

**Предусловие:** Пользователь авторизован. Существует личная автоматизация «Уведомление о новых задачах». Открыт Automation Modal (через кнопку в Navigation Sidebar).

**Шаги:**
1. Найти автоматизацию «Уведомление о новых задачах» в списке.
2. Нажать кнопку удаления автоматизации в строке.
3. Убедиться, что открылся Confirmation Popup.
4. Нажать красную кнопку «Да, подтверждаю».

**Ожидаемый результат:**
- После шага 2: открывается Confirmation Popup с текстом подтверждения удаления автоматизации.
- Popup содержит красную кнопку «Да, подтверждаю» и кнопку «Отмена».
- После шага 4: автоматизация «Уведомление о новых задачах» удалена. Confirmation Popup закрывается, автоматизация исчезает из списка в Automation Modal.
