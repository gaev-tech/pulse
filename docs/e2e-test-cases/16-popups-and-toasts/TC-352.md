# TC-352: Confirmation Popup при удалении внешней ссылки

**Предусловие:** Пользователь авторизован. Существует задача с ключом `ACME-25`, к которой прикреплена внешняя ссылка с URL «https://jira.example.com/issue/123» и названием «Связанный тикет в Jira». Пользователь имеет право `edit.links` на задачу. Задача открыта в Task Screen или Task Modal.

**Шаги:**
1. Открыть Related Links Popup для задачи.
2. Найти ссылку «Связанный тикет в Jira» в списке внешних ссылок.
3. Нажать кнопку удаления ссылки.
4. Убедиться, что открылся Confirmation Popup.
5. Нажать красную кнопку «Да, подтверждаю».

**Ожидаемый результат:**
- После шага 3: открывается Confirmation Popup с текстом подтверждения удаления внешней ссылки.
- Popup содержит красную кнопку «Да, подтверждаю» и кнопку «Отмена».
- После шага 5: внешняя ссылка удалена из задачи. Confirmation Popup закрывается, ссылка исчезает из списка в Related Links Popup.
