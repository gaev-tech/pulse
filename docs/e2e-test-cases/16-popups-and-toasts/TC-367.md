# TC-367: Error Toast — автоматическое исчезновение тоста ошибки

**Предусловие:** Пользователь авторизован. Бэкенд временно недоступен или возвращает ошибку на запрос.

**Шаги:**
1. Выполнить любое действие, вызывающее запрос к бэкенду (например, создать задачу), при условии, что бэкенд вернёт ошибку.
2. Дождаться появления Error Toast.
3. Не взаимодействовать с тостом — наблюдать за ним.

**Ожидаемый результат:**
- Error Toast появляется на экране с сообщением об ошибке.
- Через несколько секунд тост автоматически исчезает без действий пользователя.
