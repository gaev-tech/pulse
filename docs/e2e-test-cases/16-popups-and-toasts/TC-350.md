# TC-350: Confirmation Popup при удалении участника команды

**Предусловие:** Пользователь авторизован. Существует команда «Frontend» с префиксом `FRONT`. Пользователь является владельцем команды. В команде есть участник user@example.com. Открыт Team Edit Modal для команды «Frontend».

**Шаги:**
1. Найти участника user@example.com в списке участников команды.
2. Нажать кнопку удаления участника.
3. Убедиться, что открылся Confirmation Popup.
4. Нажать красную кнопку «Да, подтверждаю».

**Ожидаемый результат:**
- После шага 2: открывается Confirmation Popup с текстом подтверждения удаления участника из команды.
- Popup содержит красную кнопку «Да, подтверждаю» и кнопку «Отмена».
- После шага 4: участник user@example.com удалён из команды. Все его права отозваны, включая каскадное удаление прав, выданных им другим пользователям по командным задачам. Confirmation Popup закрывается.
