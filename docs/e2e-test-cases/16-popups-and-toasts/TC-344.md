# TC-344: Blocking Toast: автоматическое исчезновение

**Предусловие:** Пользователь авторизован. Существуют задачи A и B. Задача A блокирует задачу B. Пользователь имеет право `edit.blocking` на задачу A. Задача A открыта в Task Screen или Task Modal.

**Шаги:**
1. Открыть Blocking Tasks Popup для задачи A.
2. Попытаться добавить задачу B как блокирующую для задачи A (что создаст цикл: A блокирует B, B блокирует A).
3. Дождаться появления Blocking Toast.
4. Не совершать никаких действий и наблюдать за toast-уведомлением.

**Ожидаемый результат:**
- Появляется Blocking Toast с сообщением о том, что задача не может быть заблокирована из-за цикла в цепочке блокирования.
- Задача B не добавлена как блокирующая для задачи A.
- Toast автоматически исчезает через несколько секунд без действий пользователя.
