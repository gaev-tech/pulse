# TC-032: Схлопывание и разворачивание раздела "Команды"

**Предусловие:** Пользователь авторизован в системе. Раздел "Команды" в Navigation Sidebar развёрнут.

**Шаги:**
1. Нажать на заголовок раздела "Команды" в Navigation Sidebar для схлопывания.
2. Убедиться, что список команд скрыт.
3. Нажать на заголовок раздела "Команды" повторно для разворачивания.

**Ожидаемый результат:**
- При первом нажатии раздел схлопывается — список команд и кнопка создания команды скрываются.
- При повторном нажатии раздел разворачивается — список команд и кнопка создания команды отображаются.
