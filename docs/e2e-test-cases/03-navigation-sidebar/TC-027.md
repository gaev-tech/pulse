# TC-027: Сохранение состояния свёрнутости раздела "Личные фильтры" на бэкенде

**Предусловие:** Пользователь авторизован в системе. Раздел "Личные фильтры" развёрнут.

**Шаги:**
1. Схлопнуть раздел "Личные фильтры" в Navigation Sidebar, нажав на его заголовок.
2. Обновить страницу (F5).

**Ожидаемый результат:**
- После обновления страницы раздел "Личные фильтры" остаётся схлопнутым.
- Состояние свёрнутости сохранено на бэкенде и восстановлено при загрузке.
