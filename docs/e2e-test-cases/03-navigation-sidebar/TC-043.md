# TC-043: Кнопка открытия модалки личных автоматизаций

**Предусловие:** Пользователь авторизован в системе. Открыт любой экран приложения.

**Шаги:**
1. Нажать кнопку открытия модалки личных автоматизаций в Navigation Sidebar (расположена под разделом "Команды").

**Ожидаемый результат:**
- Открывается Automation Modal с личными автоматизациями пользователя.
- В модалке отображается список автоматизаций — каждая строка содержит: название, триггер, переключатель включения/выключения, кнопку удаления.
