# TC-025: Переход на Personal Activity Screen по кнопке

**Предусловие:** Пользователь авторизован в системе. Открыт любой экран, отличный от Personal Activity Screen.

**Шаги:**
1. Нажать кнопку перехода на Personal Activity Screen в Navigation Sidebar (расположена под иконкой и названием системы).

**Ожидаемый результат:**
- Открывается Personal Activity Screen.
- Отображается личная лента событий текущего пользователя.
