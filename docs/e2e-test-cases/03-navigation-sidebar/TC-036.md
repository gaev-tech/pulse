# TC-036: Схлопывание и разворачивание отдельной команды

**Предусловие:** Пользователь авторизован в системе. Пользователь является участником команды "ACME". Раздел "Команды" развёрнут. Команда "ACME" развёрнута — под ней отображаются командные фильтры.

**Шаги:**
1. Нажать на стрелку схлопывания команды "ACME" в Navigation Sidebar.
2. Убедиться, что список командных фильтров под командой "ACME" скрыт.
3. Нажать на стрелку разворачивания команды "ACME" повторно.

**Ожидаемый результат:**
- При первом нажатии команда схлопывается — список командных фильтров под ней скрывается.
- При повторном нажатии команда разворачивается — список командных фильтров снова отображается.
