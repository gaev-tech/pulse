# TC-033: Сохранение состояния свёрнутости раздела "Команды" на бэкенде

**Предусловие:** Пользователь авторизован в системе. Раздел "Команды" развёрнут.

**Шаги:**
1. Схлопнуть раздел "Команды" в Navigation Sidebar, нажав на его заголовок.
2. Обновить страницу (F5).

**Ожидаемый результат:**
- После обновления страницы раздел "Команды" остаётся схлопнутым.
- Состояние свёрнутости сохранено на бэкенде и восстановлено при загрузке.
