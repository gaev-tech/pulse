# TC-037: Сохранение состояния свёрнутости команды на бэкенде

**Предусловие:** Пользователь авторизован в системе. Команда "ACME" в Navigation Sidebar развёрнута.

**Шаги:**
1. Схлопнуть команду "ACME" в Navigation Sidebar.
2. Обновить страницу (F5).

**Ожидаемый результат:**
- После обновления страницы команда "ACME" остаётся схлопнутой.
- Состояние свёрнутости сохранено на бэкенде и восстановлено при загрузке.
