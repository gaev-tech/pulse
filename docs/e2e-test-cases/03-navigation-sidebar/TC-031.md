# TC-031: Кнопка перехода на Filter Activity Screen из строки фильтра

**Предусловие:** Пользователь авторизован в системе. У пользователя есть личный фильтр "Мои открытые задачи". Раздел "Личные фильтры" развёрнут.

**Шаги:**
1. Навести курсор на строку личного фильтра "Мои открытые задачи" в Navigation Sidebar.
2. Нажать кнопку перехода на Filter Activity Screen в строке фильтра.

**Ожидаемый результат:**
- Открывается Filter Activity Screen для выбранного фильтра.
- Отображается лента событий: изменения фильтра, попадание и исчезновение задач из результатов, события по задачам в период их вхождения в результаты.
