# TC-042: Кнопка Filter Activity Screen в строке командного фильтра

**Предусловие:** Пользователь авторизован в системе. Пользователь является участником команды "ACME". У команды есть фильтр "Все открытые". Команда "ACME" развёрнута в Navigation Sidebar.

**Шаги:**
1. Навести курсор на строку командного фильтра "Все открытые" под командой "ACME" в Navigation Sidebar.
2. Нажать кнопку перехода на Filter Activity Screen в строке фильтра.

**Ожидаемый результат:**
- Открывается Filter Activity Screen для командного фильтра "Все открытые".
- Отображается лента событий: изменения фильтра, попадание и исчезновение задач из результатов, события по задачам в период их вхождения в результаты.
