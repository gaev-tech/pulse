# TC-030: Клик по личному фильтру — переход на Filter Results Screen

**Предусловие:** Пользователь авторизован в системе. У пользователя есть личный фильтр "Мои открытые задачи". Раздел "Личные фильтры" развёрнут.

**Шаги:**
1. Нажать на название личного фильтра "Мои открытые задачи" в разделе "Личные фильтры" Navigation Sidebar.

**Ожидаемый результат:**
- Открывается Filter Results Screen для выбранного фильтра.
- В верхней панели отображается название фильтра "Мои открытые задачи".
- Отображается таблица задач, соответствующих критериям фильтра.
