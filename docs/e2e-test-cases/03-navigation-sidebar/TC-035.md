# TC-035: Отображение списка команд

**Предусловие:** Пользователь авторизован в системе. Пользователь является участником команд "ACME" и "BETA". Раздел "Команды" развёрнут.

**Шаги:**
1. Просмотреть раздел "Команды" в Navigation Sidebar.

**Ожидаемый результат:**
- В разделе "Команды" отображаются команды "ACME" и "BETA".
- Каждая команда отображается в виде строки с названием.
- Рядом с каждой командой отображается кнопка добавления командного фильтра.
