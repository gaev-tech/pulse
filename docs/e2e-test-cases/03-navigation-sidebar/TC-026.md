# TC-026: Схлопывание и разворачивание раздела "Личные фильтры"

**Предусловие:** Пользователь авторизован в системе. Раздел "Личные фильтры" в Navigation Sidebar развёрнут.

**Шаги:**
1. Нажать на заголовок раздела "Личные фильтры" в Navigation Sidebar для схлопывания.
2. Убедиться, что список личных фильтров скрыт.
3. Нажать на заголовок раздела "Личные фильтры" повторно для разворачивания.

**Ожидаемый результат:**
- При первом нажатии раздел схлопывается — список личных фильтров и кнопка создания фильтра скрываются.
- При повторном нажатии раздел разворачивается — список личных фильтров и кнопка создания фильтра отображаются.
