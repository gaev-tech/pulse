# TC-041: Клик по командному фильтру — переход на Filter Results Screen

**Предусловие:** Пользователь авторизован в системе. Пользователь является участником команды "ACME". У команды есть фильтр "Все открытые". Команда "ACME" развёрнута в Navigation Sidebar.

**Шаги:**
1. Нажать на название командного фильтра "Все открытые" под командой "ACME" в Navigation Sidebar.

**Ожидаемый результат:**
- Открывается Filter Results Screen для выбранного командного фильтра.
- В верхней панели отображается название фильтра "Все открытые".
- Отображается таблица задач команды "ACME", соответствующих критериям фильтра.
