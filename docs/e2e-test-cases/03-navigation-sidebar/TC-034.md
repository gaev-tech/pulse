# TC-034: Создание команды по кнопке "+" в разделе "Команды"

**Предусловие:** Пользователь авторизован в системе. Раздел "Команды" в Navigation Sidebar развёрнут.

**Шаги:**
1. Нажать кнопку "+" (создание команды) в разделе "Команды" Navigation Sidebar.

**Ожидаемый результат:**
- Открывается Team Create Modal.
- В модалке доступны поля: название команды (уникальное), prefix (уникальный), возможность добавить участников.
