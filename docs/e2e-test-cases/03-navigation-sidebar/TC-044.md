# TC-044: Кнопка открытия модалки личных меток

**Предусловие:** Пользователь авторизован в системе. Открыт любой экран приложения.

**Шаги:**
1. Нажать кнопку открытия модалки личных меток в Navigation Sidebar (расположена под кнопкой автоматизаций).

**Ожидаемый результат:**
- Открывается модальное окно управления личными метками пользователя.
- В модалке отображается список личных меток с возможностью создания, редактирования названия и цвета, удаления меток.
