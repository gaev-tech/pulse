# TC-040: Отображение списка командных фильтров под командой

**Предусловие:** Пользователь авторизован в системе. Пользователь является участником команды "ACME". У команды "ACME" есть фильтры "Все открытые" и "Баги". Команда "ACME" развёрнута в Navigation Sidebar.

**Шаги:**
1. Просмотреть список фильтров под командой "ACME" в Navigation Sidebar.

**Ожидаемый результат:**
- Под командой "ACME" отображаются командные фильтры: "Все открытые" и "Баги".
- Каждый фильтр отображается в виде строки с названием.
- Рядом с каждым фильтром отображается кнопка перехода на Filter Activity Screen.
