# TC-047: Кнопка поиска — открытие Search Modal

**Предусловие:** Пользователь авторизован в системе. Открыт любой экран приложения.

**Шаги:**
1. Нажать кнопку поиска в Navigation Sidebar (расположена под заголовком системы).

**Ожидаемый результат:**
- Открывается Search Modal.
- В модалке отображается поле ввода для поиска и область для списка результатов.
- Поле ввода получает фокус автоматически.
