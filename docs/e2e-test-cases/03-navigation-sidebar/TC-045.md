# TC-045: Переход на About Screen

**Предусловие:** Пользователь авторизован в системе. Открыт любой экран приложения.

**Шаги:**
1. Нажать кнопку перехода на About Screen в Navigation Sidebar.

**Ожидаемый результат:**
- Открывается About Screen.
- Отображается описание продукта Pulse, его ценность, маркетинговая информация со скриншотами.
