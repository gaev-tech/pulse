# TC-038: Клик по команде — переход на Team Activity Screen

**Предусловие:** Пользователь авторизован в системе. Пользователь является участником команды "ACME". Раздел "Команды" развёрнут.

**Шаги:**
1. Нажать на название команды "ACME" в разделе "Команды" Navigation Sidebar.

**Ожидаемый результат:**
- Открывается Team Activity Screen команды "ACME".
- Отображается лента событий по задачам команды "ACME".
