# TC-039: Кнопка создания командного фильтра в строке команды

**Предусловие:** Пользователь авторизован в системе. Пользователь является участником команды "ACME" и имеет право `team.manage_filters`. Раздел "Команды" развёрнут.

**Шаги:**
1. Навести курсор на строку команды "ACME" в Navigation Sidebar.
2. Нажать кнопку добавления командного фильтра в строке команды.

**Ожидаемый результат:**
- Открывается Filter Create Modal.
- В модалке тип фильтра предустановлен как командный, команда "ACME" выбрана.
