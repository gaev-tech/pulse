# TC-029: Отображение списка личных фильтров

**Предусловие:** Пользователь авторизован в системе. У пользователя созданы личные фильтры (например, "Мои открытые задачи", "Закрытые задачи"). Раздел "Личные фильтры" развёрнут.

**Шаги:**
1. Просмотреть раздел "Личные фильтры" в Navigation Sidebar.

**Ожидаемый результат:**
- В разделе "Личные фильтры" отображается список всех личных фильтров пользователя.
- Каждый фильтр отображается в виде строки с названием.
- Рядом с каждым фильтром отображается кнопка перехода на Filter Activity Screen.
