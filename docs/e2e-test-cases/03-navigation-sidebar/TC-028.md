# TC-028: Создание личного фильтра по кнопке "+"

**Предусловие:** Пользователь авторизован в системе. Раздел "Личные фильтры" в Navigation Sidebar развёрнут.

**Шаги:**
1. Нажать кнопку "+" (создание фильтра) в разделе "Личные фильтры" Navigation Sidebar.

**Ожидаемый результат:**
- Открывается Filter Create Modal.
- В модалке доступны поля: название фильтра, тип (личный/командный), режим фильтрации (простой/RSQL) и критерии.
