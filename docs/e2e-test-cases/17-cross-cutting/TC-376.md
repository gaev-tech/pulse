# TC-376: Zero state — пустой список автоматизаций в Automation Modal

**Предусловие:** Пользователь авторизован. Существует фильтр «Без автоматизаций», для которого не создано ни одной автоматизации.

**Шаги:**
1. Открыть Navigation Sidebar.
2. Нажать на фильтр «Без автоматизаций».
3. Открыть Automation Modal для данного фильтра.
4. Наблюдать за содержимым списка автоматизаций.

**Ожидаемый результат:**
- Список автоматизаций не содержит ни одного элемента.
- Вместо пустого пространства отображается zero-state — сообщение об отсутствии автоматизаций.
