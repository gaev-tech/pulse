# TC-356: Last opened — фиксация при открытии Task Modal

**Предусловие:** Пользователь авторизован. Существует задача с ключом `ACME-18`. Пользователь имеет право `view` на задачу. Задача ранее не открывалась пользователем. Пользователь находится на Filter Results Screen, где задача `ACME-18` отображается в результатах.

**Шаги:**
1. Нажать на задачу `ACME-18` в таблице результатов фильтра для открытия Task Modal.
2. Закрыть Task Modal.
3. Открыть Search Modal.
4. Ввести поисковый запрос, по которому задача `ACME-18` и другие задачи соответствуют критериям.
5. Проверить позицию задачи `ACME-18` в результатах поиска.

**Ожидаемый результат:**
- После открытия Task Modal дата последнего открытия задачи `ACME-18` зафиксирована на бэкенде.
- В результатах поиска (шаг 5) задача `ACME-18` отображается выше задач, которые не открывались или открывались раньше (при отсутствии точного совпадения ключа/названия).
