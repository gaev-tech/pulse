# TC-368: Zero state — пустой список задач в таблице фильтра

**Предусловие:** Пользователь авторизован. Существует фильтр «Пустой фильтр», критериям которого не соответствует ни одна задача.

**Шаги:**
1. Открыть Navigation Sidebar.
2. Нажать на фильтр «Пустой фильтр» в списке фильтров.
3. Наблюдать за содержимым таблицы задач на Filter Screen.

**Ожидаемый результат:**
- Таблица задач не содержит строк.
- Вместо пустого пространства отображается zero-state — сообщение об отсутствии задач, соответствующих критериям фильтра.
