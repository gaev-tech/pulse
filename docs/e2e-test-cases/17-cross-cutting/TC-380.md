# TC-380: Подтверждение потери данных при закрытии Team Edit Modal с несохранёнными данными

**Предусловие:** Пользователь авторизован. Существует команда «ACME». Пользователь является владельцем команды.

**Шаги:**
1. Открыть Navigation Sidebar.
2. Нажать на команду «ACME».
3. Открыть Team Edit Modal для команды «ACME».
4. Изменить название команды на «ACME Updated».
5. Не сохраняя изменения, нажать кнопку закрытия модального окна (крестик) или кликнуть вне области модального окна.

**Ожидаемый результат:**
- Появляется Unsaved Changes Confirmation Modal с текстом «Вы уверены? Несохранённые данные будут потеряны».
- Модальное окно содержит кнопку «Да, закрыть» и кнопку «Отмена».
- При нажатии «Отмена» — Confirmation Modal закрывается, Team Edit Modal остаётся открытым с изменёнными данными.
- При нажатии «Да, закрыть» — оба модальных окна закрываются, изменения не сохраняются.
