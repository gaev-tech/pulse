# TC-359: Переключение аккаунта — применение темы выбранного пользователя

**Предусловие:** На данном клиенте авторизованы два пользователя: user1@example.com (тема — светлая, текущий активный) и user2@example.com (тема — тёмная).

**Шаги:**
1. Убедиться, что интерфейс отображается в светлой теме.
2. Нажать на аватарку и имя пользователя в нижней части Navigation Sidebar.
3. В Profile Popup нажать на user2@example.com в списке аккаунтов.
4. Проверить тему интерфейса после переключения.

**Ожидаемый результат:**
- Происходит переключение на аккаунт user2@example.com.
- Тема интерфейса меняется на тёмную (тема, установленная у user2).
- Все элементы интерфейса отображаются в тёмной цветовой схеме.
- Пользователь перенаправлен на Personal Activity Screen аккаунта user2.
