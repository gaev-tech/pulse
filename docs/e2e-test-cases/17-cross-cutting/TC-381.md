# TC-381: 403 Screen — открытие задачи без доступа

**Предусловие:** Пользователь авторизован. Существует задача `ACME-25` в команде «ACME». Пользователь не является участником команды «ACME» и не имеет доступа к задаче.

**Шаги:**
1. Ввести в адресную строку браузера прямой URL задачи `ACME-25`.
2. Нажать Enter для перехода.

**Ожидаемый результат:**
- Вместо Task Screen отображается 403 Screen — экран отсутствия доступа.
- Экран содержит сообщение о том, что у пользователя нет доступа к данной задаче.
