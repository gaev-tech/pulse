# TC-378: Подтверждение потери данных при закрытии Filter Create Modal с несохранёнными данными

**Предусловие:** Пользователь авторизован и находится на любом экране системы.

**Шаги:**
1. Открыть Filter Create Modal (нажать кнопку создания фильтра в Navigation Sidebar).
2. Заполнить поле названия фильтра текстом «Тестовый фильтр».
3. Не сохраняя фильтр, нажать кнопку закрытия модального окна (крестик) или кликнуть вне области модального окна.

**Ожидаемый результат:**
- Появляется Unsaved Changes Confirmation Modal с текстом «Вы уверены? Несохранённые данные будут потеряны».
- Модальное окно содержит кнопку «Да, закрыть» и кнопку «Отмена».
- При нажатии «Отмена» — Confirmation Modal закрывается, Filter Create Modal остаётся открытым с введёнными данными.
- При нажатии «Да, закрыть» — оба модальных окна закрываются, данные не сохраняются.
