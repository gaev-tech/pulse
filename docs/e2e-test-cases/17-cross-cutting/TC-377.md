# TC-377: Подтверждение потери данных при закрытии Task Create Modal с несохранёнными данными

**Предусловие:** Пользователь авторизован и находится на любом экране системы.

**Шаги:**
1. Открыть Task Create Modal (нажать кнопку создания задачи).
2. Заполнить поле названия задачи текстом «Тестовая задача».
3. Не сохраняя задачу, нажать кнопку закрытия модального окна (крестик) или кликнуть вне области модального окна.

**Ожидаемый результат:**
- Появляется Unsaved Changes Confirmation Modal с текстом «Вы уверены? Несохранённые данные будут потеряны».
- Модальное окно содержит кнопку «Да, закрыть» и кнопку «Отмена».
- При нажатии «Отмена» — Confirmation Modal закрывается, Task Create Modal остаётся открытым с введёнными данными.
- При нажатии «Да, закрыть» — оба модальных окна закрываются, данные не сохраняются.
