# TC-361: Переход по ссылке на задачу (с ключом) — открытие задачи

**Предусловие:** Пользователь авторизован. Существует командная задача с ключом `ACME-33`. Пользователь имеет право `view` на задачу. Пользователь ранее скопировал ссылку на задачу, содержащую ключ.

**Шаги:**
1. Вставить ссылку на задачу `ACME-33` в адресную строку браузера и перейти по ней.

**Ожидаемый результат:**
- Открывается Task Screen задачи `ACME-33`.
- Отображаются все поля задачи: название, описание, статус, ответственный, метки, ссылки, связи, блокирование, родитель, вложения, лента событий.
- Задача корректно определена по ключу из URL.
