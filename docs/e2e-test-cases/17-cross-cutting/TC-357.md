# TC-357: Управление несколькими аккаунтами — список в Profile Popup

**Предусловие:** На данном клиенте авторизованы два пользователя: user1@example.com (текущий активный) и user2@example.com. Пользователь user1 находится на любом экране системы.

**Шаги:**
1. Нажать на аватарку и имя пользователя в нижней части Navigation Sidebar.
2. Проверить содержимое Profile Popup.

**Ожидаемый результат:**
- Открывается Profile Popup.
- В списке авторизованных пользователей отображаются оба аккаунта: user1@example.com и user2@example.com.
- Текущий активный пользователь (user1@example.com) визуально выделен.
- В конце списка отображается кнопка «Добавить пользователя».
- Popup также содержит переключатель языка и переключатель темы.
