# TC-362: Стрелка раскрытия иерархии в таблице фильтра

**Предусловие:** Пользователь авторизован. Существует фильтр, результаты которого содержат родительскую задачу `ACME-40` и её дочернюю задачу `ACME-41`. Пользователь находится на Filter Results Screen данного фильтра. Задача `ACME-40` отображается в таблице результатов.

**Шаги:**
1. Найти строку задачи `ACME-40` в таблице результатов.
2. Нажать на стрелку раскрытия иерархии в строке задачи `ACME-40`.
3. Проверить, что дочерние задачи отобразились.
4. Нажать на стрелку раскрытия иерархии повторно.

**Ожидаемый результат:**
- Стрелка раскрытия иерархии расположена в обязательной колонке таблицы (перед ключом задачи).
- После шага 2: под строкой `ACME-40` отображается дочерняя задача `ACME-41` с визуальным отступом, обозначающим вложенность.
- После шага 4: дочерние задачи скрываются, иерархия свёрнута.
