# TC-355: Last opened — фиксация при открытии Task Screen

**Предусловие:** Пользователь авторизован. Существует задача с ключом `ACME-12`. Пользователь имеет право `view` на задачу. Задача ранее не открывалась пользователем.

**Шаги:**
1. Перейти на Task Screen задачи `ACME-12` (например, по прямой ссылке).
2. Открыть Search Modal.
3. Ввести поисковый запрос, по которому задача `ACME-12` и другие задачи соответствуют критериям (например, общее слово из описания).
4. Проверить позицию задачи `ACME-12` в результатах поиска.

**Ожидаемый результат:**
- После открытия Task Screen дата последнего открытия задачи `ACME-12` зафиксирована на бэкенде.
- В результатах поиска (шаг 4) задача `ACME-12` отображается выше задач, которые не открывались или открывались раньше (при отсутствии точного совпадения ключа/названия).
