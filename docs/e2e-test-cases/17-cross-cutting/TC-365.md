# TC-365: Documentation Screen — отображение документации

**Предусловие:** Пользователь авторизован и находится на любом экране системы. Navigation Sidebar отображается.

**Шаги:**
1. Найти кнопку перехода на Documentation Screen в Navigation Sidebar.
2. Нажать на кнопку.
3. Проверить содержимое Documentation Screen.

**Ожидаемый результат:**
- Открывается Documentation Screen.
- Экран содержит документацию по системе Pulse.
- Документация доступна для чтения и навигации.
