# TC-449: About Screen доступен без авторизации

**Предусловие:** Пользователь не авторизован (сессия отсутствует).

**Шаги:**
1. Открыть страницу `/about` напрямую в браузере.

**Ожидаемый результат:**
- Страница `/about` открывается без редиректа на `/auth`.
- Отображается описание продукта, ключевые функции, скриншоты и CTA-кнопка «Попробовать бесплатно».
