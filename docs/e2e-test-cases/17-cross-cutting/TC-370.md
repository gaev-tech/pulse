# TC-370: Zero state — пустая лента событий Team Activity

**Предусловие:** Пользователь авторизован. Существует команда «Новая команда», в которой не было выполнено ни одного действия. Пользователь является участником команды.

**Шаги:**
1. Открыть Navigation Sidebar.
2. Нажать на команду «Новая команда».
3. Перейти на Team Activity Screen команды «Новая команда».
4. Наблюдать за содержимым ленты событий.

**Ожидаемый результат:**
- Лента событий команды не содержит ни одного события.
- Вместо пустого пространства отображается zero-state — сообщение об отсутствии событий.
