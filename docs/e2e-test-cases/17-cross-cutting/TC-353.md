# TC-353: Динамическое обновление ленты — получение сигнала от бэкенда

**Предусловие:** Пользователь A и пользователь B авторизованы. Оба состоят в команде «ACME». Пользователь A находится на Team Activity Screen команды «ACME».

**Шаги:**
1. Пользователь B создаёт новую задачу `ACME-50` в команде «ACME» (из другого браузера или сессии).
2. Наблюдать за лентой событий у пользователя A на Team Activity Screen.

**Ожидаемый результат:**
- Лента событий у пользователя A обновляется автоматически без перезагрузки страницы.
- В ленте появляется новое событие `task.created` для задачи `ACME-50`.
- Фронтенд получает сигнал от бэкенда о появлении нового события и запрашивает обновлённые данные.
