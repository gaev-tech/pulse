# TC-364: About Screen — отображение описания продукта

**Предусловие:** Пользователь авторизован и находится на любом экране системы. Navigation Sidebar отображается.

**Шаги:**
1. Найти кнопку перехода на About Screen в Navigation Sidebar.
2. Нажать на кнопку.
3. Проверить содержимое About Screen.

**Ожидаемый результат:**
- Открывается About Screen.
- Экран содержит описание продукта Pulse.
- Отображается информация о ценности продукта.
- Присутствует маркетинговая информация со скриншотами системы.
