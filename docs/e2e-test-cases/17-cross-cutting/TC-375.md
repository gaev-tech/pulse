# TC-375: Zero state — пустой результат поиска в Search Modal

**Предусловие:** Пользователь авторизован и находится на любом экране системы.

**Шаги:**
1. Открыть Search Modal (через кнопку поиска или горячую клавишу).
2. Ввести в поле поиска текст, которому не соответствует ни одна задача (например, «zzzнесуществующийзапросzzz»).
3. Наблюдать за результатами поиска.

**Ожидаемый результат:**
- Список результатов поиска не содержит ни одного элемента.
- Вместо пустого пространства отображается zero-state — сообщение об отсутствии результатов поиска.
