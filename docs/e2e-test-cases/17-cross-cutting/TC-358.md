# TC-358: Переключение аккаунта — применение языка выбранного пользователя

**Предусловие:** На данном клиенте авторизованы два пользователя: user1@example.com (язык интерфейса — русский, текущий активный) и user2@example.com (язык интерфейса — English).

**Шаги:**
1. Нажать на аватарку и имя пользователя в нижней части Navigation Sidebar.
2. В Profile Popup нажать на user2@example.com в списке аккаунтов.
3. Проверить язык интерфейса после переключения.

**Ожидаемый результат:**
- Происходит переключение на аккаунт user2@example.com.
- Язык интерфейса меняется на English (язык, установленный у user2).
- Все элементы интерфейса (Navigation Sidebar, кнопки, надписи) отображаются на английском языке.
- Пользователь перенаправлен на Personal Activity Screen аккаунта user2.
