# TC-379: Подтверждение потери данных при закрытии Team Create Modal с несохранёнными данными

**Предусловие:** Пользователь авторизован и находится на любом экране системы.

**Шаги:**
1. Открыть Team Create Modal (нажать кнопку создания команды в Navigation Sidebar).
2. Заполнить поле названия команды текстом «Тестовая команда» и указать ключ «TEST».
3. Не сохраняя команду, нажать кнопку закрытия модального окна (крестик) или кликнуть вне области модального окна.

**Ожидаемый результат:**
- Появляется Unsaved Changes Confirmation Modal с текстом «Вы уверены? Несохранённые данные будут потеряны».
- Модальное окно содержит кнопку «Да, закрыть» и кнопку «Отмена».
- При нажатии «Отмена» — Confirmation Modal закрывается, Team Create Modal остаётся открытым с введёнными данными.
- При нажатии «Да, закрыть» — оба модальных окна закрываются, данные не сохраняются.
