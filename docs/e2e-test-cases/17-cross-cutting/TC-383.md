# TC-383: 404 Screen — открытие несуществующего или недоступного фильтра или команды

**Предусловие:** Пользователь авторизован. Фильтр с ID `99999` и команда с ID `99999` не существуют в системе.

**Шаги:**
1. Ввести в адресную строку браузера прямой URL фильтра с несуществующим ID (например, `/filters/99999`).
2. Нажать Enter для перехода.
3. Убедиться, что отображается 404 Screen.
4. Ввести в адресную строку браузера прямой URL команды с несуществующим ID (например, `/teams/99999`).
5. Нажать Enter для перехода.

**Ожидаемый результат:**
- При переходе на несуществующий фильтр отображается 404 Screen — экран «Не найдено» с сообщением о том, что фильтр не найден.
- При переходе на несуществующую команду отображается 404 Screen — экран «Не найдено» с сообщением о том, что команда не найдена.
- Аналогичный результат при попытке открыть фильтр или команду, к которым у пользователя нет доступа.
