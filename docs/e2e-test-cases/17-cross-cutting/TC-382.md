# TC-382: 404 Screen — открытие несуществующей задачи

**Предусловие:** Пользователь авторизован. Задача с ключом `ACME-99999` не существует в системе.

**Шаги:**
1. Ввести в адресную строку браузера прямой URL задачи `ACME-99999`.
2. Нажать Enter для перехода.

**Ожидаемый результат:**
- Вместо Task Screen отображается 404 Screen — экран «Не найдено».
- Экран содержит сообщение о том, что запрашиваемая задача не найдена.
