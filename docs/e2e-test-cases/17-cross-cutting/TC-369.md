# TC-369: Zero state — пустая лента событий Personal Activity

**Предусловие:** Пользователь авторизован. У пользователя нет ни одного события в личной ленте (новый пользователь без действий).

**Шаги:**
1. Перейти на Personal Activity Screen.
2. Наблюдать за содержимым ленты событий.

**Ожидаемый результат:**
- Лента событий не содержит ни одного события.
- Вместо пустого пространства отображается zero-state — сообщение об отсутствии событий.
