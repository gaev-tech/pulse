# TC-363: Чекбокс выбора задач в таблице фильтра

**Предусловие:** Пользователь авторизован. Существует фильтр с несколькими задачами в результатах: `ACME-50`, `ACME-51`, `ACME-52`. Пользователь находится на Filter Results Screen данного фильтра.

**Шаги:**
1. Найти чекбоксы выбора в строках задач таблицы результатов.
2. Нажать на чекбокс в строке задачи `ACME-50`.
3. Нажать на чекбокс в строке задачи `ACME-52`.
4. Проверить состояние выбора.
5. Нажать на чекбокс в строке задачи `ACME-50` повторно.

**Ожидаемый результат:**
- Чекбоксы расположены в первой обязательной колонке таблицы.
- После шага 2: задача `ACME-50` выделена (чекбокс отмечен).
- После шага 3: задачи `ACME-50` и `ACME-52` выделены, задача `ACME-51` не выделена.
- После шага 5: выделение с задачи `ACME-50` снято (чекбокс снят), задача `ACME-52` остаётся выделенной.
