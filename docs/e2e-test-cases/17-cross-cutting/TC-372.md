# TC-372: Zero state — пустой список фильтров в Navigation Sidebar

**Предусловие:** Пользователь авторизован. У пользователя нет ни одного созданного фильтра.

**Шаги:**
1. Открыть Navigation Sidebar.
2. Найти секцию фильтров в боковой панели.
3. Наблюдать за содержимым списка фильтров.

**Ожидаемый результат:**
- Список фильтров в Navigation Sidebar не содержит ни одного элемента.
- Вместо пустого пространства отображается zero-state — сообщение об отсутствии фильтров.
