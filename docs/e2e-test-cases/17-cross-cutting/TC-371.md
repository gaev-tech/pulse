# TC-371: Zero state — пустая лента событий Filter Activity

**Предусловие:** Пользователь авторизован. Существует фильтр «Пустой фильтр», по задачам которого не было выполнено ни одного действия.

**Шаги:**
1. Открыть Navigation Sidebar.
2. Нажать на фильтр «Пустой фильтр» в списке фильтров.
3. Перейти на Filter Activity Screen фильтра «Пустой фильтр».
4. Наблюдать за содержимым ленты событий.

**Ожидаемый результат:**
- Лента событий фильтра не содержит ни одного события.
- Вместо пустого пространства отображается zero-state — сообщение об отсутствии событий.
