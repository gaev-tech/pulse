# TC-373: Zero state — пустой список команд в Navigation Sidebar

**Предусловие:** Пользователь авторизован. Пользователь не состоит ни в одной команде.

**Шаги:**
1. Открыть Navigation Sidebar.
2. Найти секцию команд в боковой панели.
3. Наблюдать за содержимым списка команд.

**Ожидаемый результат:**
- Список команд в Navigation Sidebar не содержит ни одного элемента.
- Вместо пустого пространства отображается zero-state — сообщение об отсутствии команд.
