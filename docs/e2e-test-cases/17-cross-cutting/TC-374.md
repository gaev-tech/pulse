# TC-374: Zero state — пустой список участников команды в Team Edit Modal

**Предусловие:** Пользователь авторизован. Существует команда «Пустая команда», в которой пользователь является единственным участником (владельцем). Других участников нет.

**Шаги:**
1. Открыть Navigation Sidebar.
2. Нажать на команду «Пустая команда».
3. Открыть Team Edit Modal для команды «Пустая команда».
4. Перейти к секции участников команды.

**Ожидаемый результат:**
- Список участников команды содержит только текущего пользователя (владельца).
- Если удалить всех участников кроме владельца, вместо пустого пространства отображается zero-state — сообщение об отсутствии дополнительных участников.
