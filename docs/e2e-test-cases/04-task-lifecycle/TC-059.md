# TC-059: Открытие задачи в Task Modal (модальное окно)

**Предусловие:** Пользователь авторизован в системе. Открыт Filter Results Screen. В таблице результатов отображается задача с ключом #1 и названием "Подготовить отчёт".

**Шаги:**
1. Нажать на строку задачи #1 "Подготовить отчёт" в таблице результатов фильтра.

**Ожидаемый результат:**
- Открывается Task Modal поверх текущего экрана (Filter Results Screen).
- В модалке отображается та же структура, что и в Task Screen: название, описание, лента событий, поля задачи.
- Модалка содержит кнопку "Открыть задачу в отдельной странице".
- Filter Results Screen остаётся видимым под модалкой.
