# TC-071: Добавление метки к задаче

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи без меток. У пользователя есть право `edit.labels`. Существует метка «Срочно» красного цвета.

**Шаги:**
1. В правой части Task Screen найти поле «Метки» (пустое).
2. Кликнуть на поле меток.
3. Выбрать метку «Срочно» из списка доступных меток.

**Ожидаемый результат:**
- В поле «Метки» отображается метка «Срочно» красного цвета.
- В ленте событий задачи появилось событие `task.labels_changed`.
