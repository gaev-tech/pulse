# TC-077: Удаление родительской задачи

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи с родительской задачей «ACME-10: Эпик проекта». У пользователя есть право `edit.parent`.

**Шаги:**
1. В правой части Task Screen найти поле «Родительская задача» со значением «ACME-10: Эпик проекта».
2. Кликнуть на кнопку очистки поля родительской задачи.

**Ожидаемый результат:**
- Поле «Родительская задача» отображается пустым.
- В ленте событий задачи появилось событие `task.parent_changed`.
