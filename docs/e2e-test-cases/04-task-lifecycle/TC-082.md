# TC-082: Удаление задачи через меню "Ещё" -- Confirmation Popup

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи «ACME-42: Тестовая задача». Пользователь является владельцем задачи.

**Шаги:**
1. В верхней части Task Screen нажать кнопку «Ещё» (меню дополнительных действий).
2. В открывшемся меню выбрать пункт «Удалить задачу».

**Ожидаемый результат:**
- Отображается Confirmation Popup с текстом «Вы подтверждаете удаление задачи?».
- Popup содержит красную кнопку «Да, подтверждаю» и кнопку «Отмена».
