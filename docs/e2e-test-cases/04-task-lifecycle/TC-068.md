# TC-068: Назначение ответственного задачи

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи без ответственного. У пользователя есть право `edit.assignee`. В системе зарегистрирован пользователь «Иван Петров».

**Шаги:**
1. В правой части Task Screen найти поле «Ответственный» (пустое).
2. Кликнуть на поле ответственного.
3. Выбрать пользователя «Иван Петров» из списка.

**Ожидаемый результат:**
- В поле «Ответственный» отображается «Иван Петров».
- В ленте событий задачи появилось событие `task.assignee_changed`.
