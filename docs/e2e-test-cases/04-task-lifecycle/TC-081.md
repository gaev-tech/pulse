# TC-081: Копирование ссылки на задачу -- отображение тоста

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи с ключом «ACME-42».

**Шаги:**
1. В верхней части Task Screen нажать кнопку «Скопировать ссылку на задачу».

**Ожидаемый результат:**
- Отображается Task Link Toast с сообщением о том, что ссылка на задачу скопирована.
- В буфер обмена скопирована ссылка, содержащая ключ задачи «ACME-42».
