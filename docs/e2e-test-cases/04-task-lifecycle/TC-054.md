# TC-054: Переключение между личной и командной задачей при создании

**Предусловие:** Пользователь авторизован в системе. Пользователь является участником команды "ACME" с правом `task.create`. Открыт Task Create Modal.

**Шаги:**
1. Выбрать вариант "Личная задача" в качестве владельца.
2. Убедиться, что выбрана личная задача.
3. Переключить владельца на команду "ACME".
4. Убедиться, что выбрана командная задача.
5. Переключить обратно на "Личная задача".

**Ожидаемый результат:**
- Переключение между личной и командной задачей работает корректно.
- При выборе "Личная задача" задача будет создана с владельцем — текущий пользователь.
- При выборе команды "ACME" задача будет создана с владельцем — команда "ACME".
