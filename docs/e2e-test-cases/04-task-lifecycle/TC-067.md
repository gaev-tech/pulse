# TC-067: Изменение статуса задачи (closed -> opened)

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи со статусом `closed`. У пользователя есть право `edit.status`.

**Шаги:**
1. В правой части Task Screen найти поле «Статус» со значением `closed`.
2. Кликнуть на поле статуса.
3. Выбрать значение `opened`.

**Ожидаемый результат:**
- Статус задачи отображается как `opened`.
- В ленте событий задачи появилось событие `task.status_changed`.
