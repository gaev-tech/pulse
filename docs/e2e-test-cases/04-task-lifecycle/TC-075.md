# TC-075: Установка родительской задачи

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи без родительской задачи. У пользователя есть право `edit.parent`. Существует задача «ACME-10: Эпик проекта».

**Шаги:**
1. В правой части Task Screen найти поле «Родительская задача» (пустое).
2. Кликнуть на поле родительской задачи.
3. В поиске ввести «ACME-10».
4. Выбрать задачу «ACME-10: Эпик проекта» из результатов поиска.

**Ожидаемый результат:**
- В поле «Родительская задача» отображается «ACME-10: Эпик проекта».
- В ленте событий задачи появилось событие `task.parent_changed`.
