# TC-050: Создание личной задачи с названием и описанием

**Предусловие:** Пользователь авторизован в системе. Открыт Task Create Modal.

**Шаги:**
1. Убедиться, что выбран вариант "Личная задача" в качестве владельца.
2. Ввести название задачи: "Исправить баг в авторизации".
3. Ввести описание задачи в формате markdown: "При входе через **magic link** возникает ошибка 500. Нужно проверить обработку истёкших токенов.".
4. Нажать кнопку создания задачи.

**Ожидаемый результат:**
- Задача создана с указанным названием и описанием.
- Описание сохранено в формате markdown.
- Task Create Modal закрывается.
- Открывается Task Modal созданной задачи — в левой части отображаются название и описание с корректным markdown-рендерингом.
