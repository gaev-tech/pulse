# TC-051: Валидация обязательного поля "Название" при создании задачи

**Предусловие:** Пользователь авторизован в системе. Открыт Task Create Modal.

**Шаги:**
1. Оставить поле "Название" пустым.
2. Нажать кнопку создания задачи.

**Ожидаемый результат:**
- Задача не создаётся.
- Отображается сообщение об ошибке — поле "Название" обязательно для заполнения.
- Task Create Modal остаётся открытым.
