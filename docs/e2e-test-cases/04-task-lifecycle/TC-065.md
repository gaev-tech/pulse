# TC-065: Редактирование описания задачи

**Предусловие:** Пользователь авторизован, открыта Task Screen существующей задачи с описанием «Старое описание». У пользователя есть право `edit.description`.

**Шаги:**
1. Кликнуть на поле описания задачи в левой части Task Screen.
2. Очистить поле и ввести новое описание «Новое описание задачи с подробностями».
3. Кликнуть вне поля описания для сохранения.

**Ожидаемый результат:**
- Описание задачи отображается как «Новое описание задачи с подробностями».
- В ленте событий задачи появилось событие `task.description_changed`.
