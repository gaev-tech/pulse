# TC-058: Открытие задачи в Task Screen (полная страница)

**Предусловие:** Пользователь авторизован в системе. Существует задача с ключом #1 и названием "Подготовить отчёт".

**Шаги:**
1. Открыть в браузере прямую ссылку на задачу #1 (URL, содержащий ключ задачи).

**Ожидаемый результат:**
- Открывается Task Screen — полноэкранная страница задачи.
- В левой (широкой) части отображаются: название "Подготовить отчёт", описание, лента событий.
- В правой части отображаются: элементы управления, поля задачи (статус, ответственный, метки, ссылки, связи, блокирование, родитель, вложения).
