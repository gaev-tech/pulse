# TC-052: Создание командной задачи — выбор команды

**Предусловие:** Пользователь авторизован в системе. Пользователь является участником команды "ACME" с правом `task.create`. Открыт Task Create Modal.

**Шаги:**
1. В поле выбора владельца задачи выбрать команду "ACME".
2. Ввести название задачи: "Настроить CI/CD".
3. Нажать кнопку создания задачи.

**Ожидаемый результат:**
- Задача создана с владельцем — команда "ACME".
- Задаче присвоен ключ в формате PREFIX-NNN (например, ACME-1).
- Task Create Modal закрывается.
- Открывается Task Modal созданной задачи.
