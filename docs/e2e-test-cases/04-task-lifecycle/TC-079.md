# TC-079: Добавление вложения к задаче

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи без вложений. У пользователя есть право `edit.attachments`.

**Шаги:**
1. В правой части Task Screen найти раздел «Вложения» (пустой).
2. Кликнуть на кнопку добавления вложения.
3. Выбрать файл «report.pdf» в диалоге выбора файла.
4. Дождаться завершения загрузки.

**Ожидаемый результат:**
- В разделе «Вложения» отображается файл «report.pdf».
- В ленте событий задачи появилось событие `task.attachments_changed`.
