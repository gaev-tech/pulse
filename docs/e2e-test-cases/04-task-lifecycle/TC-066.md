# TC-066: Изменение статуса задачи (opened -> closed)

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи со статусом `opened`. У пользователя есть право `edit.status`.

**Шаги:**
1. В правой части Task Screen найти поле «Статус» со значением `opened`.
2. Кликнуть на поле статуса.
3. Выбрать значение `closed`.

**Ожидаемый результат:**
- Статус задачи отображается как `closed`.
- В ленте событий задачи появилось событие `task.status_changed`.
