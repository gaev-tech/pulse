# TC-060: Кнопка "Открыть задачу в отдельной странице" в Task Modal

**Предусловие:** Пользователь авторизован в системе. Открыт Task Modal задачи #1 "Подготовить отчёт" поверх Filter Results Screen.

**Шаги:**
1. Нажать кнопку "Открыть задачу в отдельной странице" в Task Modal.

**Ожидаемый результат:**
- Task Modal закрывается.
- Открывается Task Screen — полноэкранная страница задачи #1 "Подготовить отчёт".
- URL браузера изменяется на адрес задачи.
