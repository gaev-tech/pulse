# TC-069: Изменение ответственного задачи

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи с ответственным «Иван Петров». У пользователя есть право `edit.assignee`. В системе зарегистрирован пользователь «Мария Сидорова».

**Шаги:**
1. В правой части Task Screen найти поле «Ответственный» со значением «Иван Петров».
2. Кликнуть на поле ответственного.
3. Выбрать пользователя «Мария Сидорова» из списка.

**Ожидаемый результат:**
- В поле «Ответственный» отображается «Мария Сидорова».
- В ленте событий задачи появилось событие `task.assignee_changed`.
