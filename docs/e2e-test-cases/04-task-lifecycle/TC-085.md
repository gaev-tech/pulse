# TC-085: Отображение ленты событий задачи на Task Screen

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи «ACME-42», у которой ранее были выполнены изменения: создание, изменение названия, изменение статуса.

**Шаги:**
1. В левой части Task Screen прокрутить вниз до раздела ленты событий.
2. Просмотреть список событий.

**Ожидаемый результат:**
- Лента событий отображает записи: `task.created`, `task.title_changed`, `task.status_changed`.
- События отсортированы по времени (от новых к старым).
- Каждое событие содержит информацию об авторе изменения и времени.
