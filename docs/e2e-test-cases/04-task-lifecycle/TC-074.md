# TC-074: Удаление всех меток с задачи

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи с метками «Срочно» и «Баг». У пользователя есть право `edit.labels`.

**Шаги:**
1. В правой части Task Screen найти поле «Метки» с метками «Срочно» и «Баг».
2. Кликнуть на поле меток.
3. Снять выбор со всех меток (передать пустой массив).

**Ожидаемый результат:**
- Поле «Метки» отображается пустым.
- В ленте событий задачи появилось событие `task.labels_changed`.
