# TC-086: Динамическое обновление ленты событий задачи

**Предусловие:** Пользователь A авторизован, открыта Task Screen задачи «ACME-42». Пользователь B также авторизован и имеет право редактирования задачи.

**Шаги:**
1. Пользователь A наблюдает ленту событий задачи «ACME-42» на Task Screen.
2. Пользователь B в другой сессии открывает ту же задачу и изменяет её название на «Изменённое название».
3. Пользователь A наблюдает за обновлением ленты событий.

**Ожидаемый результат:**
- Лента событий у Пользователя A динамически обновляется без перезагрузки страницы.
- В ленте появляется новое событие `task.title_changed` с указанием Пользователя B как автора.
