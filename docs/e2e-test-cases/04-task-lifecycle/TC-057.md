# TC-057: Генерация ключа командной задачи в формате PREFIX-NNN

**Предусловие:** Пользователь авторизован в системе. Пользователь является участником команды "ACME" с правом `task.create`. У команды уже есть задачи (последний ключ — ACME-10).

**Шаги:**
1. Открыть Task Create Modal.
2. Выбрать команду "ACME" в качестве владельца.
3. Ввести название задачи: "Новая командная задача".
4. Нажать кнопку создания задачи.

**Ожидаемый результат:**
- Задача создана с ключом ACME-11 (следующий порядковый номер после ACME-10).
- Ключ отображается в Task Modal созданной задачи.
- Ключ уникален в рамках задач команды "ACME".
