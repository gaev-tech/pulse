# TC-064: Редактирование названия задачи

**Предусловие:** Пользователь авторизован, открыта Task Screen существующей задачи с текущим названием «Исходное название». У пользователя есть право `edit.title`.

**Шаги:**
1. Кликнуть на название задачи в левой части Task Screen.
2. Очистить поле и ввести новое название «Обновлённое название».
3. Кликнуть вне поля названия для сохранения.

**Ожидаемый результат:**
- Название задачи отображается как «Обновлённое название».
- В ленте событий задачи появилось событие `task.title_changed`.
