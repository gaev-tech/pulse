# TC-076: Изменение родительской задачи

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи с родительской задачей «ACME-10: Эпик проекта». У пользователя есть право `edit.parent`. Существует задача «ACME-20: Новый эпик».

**Шаги:**
1. В правой части Task Screen найти поле «Родительская задача» со значением «ACME-10: Эпик проекта».
2. Кликнуть на поле родительской задачи.
3. В поиске ввести «ACME-20».
4. Выбрать задачу «ACME-20: Новый эпик» из результатов поиска.

**Ожидаемый результат:**
- В поле «Родительская задача» отображается «ACME-20: Новый эпик».
- В ленте событий задачи появилось событие `task.parent_changed`.
