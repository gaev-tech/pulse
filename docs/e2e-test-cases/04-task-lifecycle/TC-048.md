# TC-048: Открытие Task Create Modal

**Предусловие:** Пользователь авторизован в системе. Открыт любой экран приложения.

**Шаги:**
1. Нажать кнопку создания задачи в интерфейсе.

**Ожидаемый результат:**
- Открывается Task Create Modal.
- В модалке доступны поля: название (обязательное), описание, выбор владельца (личная задача или одна из команд).
