# TC-055: После создания задачи — закрытие Create Modal и открытие Task Modal

**Предусловие:** Пользователь авторизован в системе. Открыт Filter Results Screen с результатами фильтра. Открыт Task Create Modal.

**Шаги:**
1. Ввести название задачи: "Тестовая задача".
2. Нажать кнопку создания задачи.

**Ожидаемый результат:**
- Task Create Modal закрывается.
- Открывается Task Modal с данными только что созданной задачи "Тестовая задача".
- Под модалкой остаётся видимым Filter Results Screen — пользователь не покидает текущую страницу.
