# TC-080: Удаление вложения из задачи

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи с вложением «report.pdf». У пользователя есть право `edit.attachments`.

**Шаги:**
1. В правой части Task Screen найти раздел «Вложения» с файлом «report.pdf».
2. Кликнуть на кнопку удаления вложения «report.pdf».
3. В Confirmation Popup нажать кнопку «Да, подтверждаю».

**Ожидаемый результат:**
- Раздел «Вложения» отображается пустым.
- В ленте событий задачи появилось событие `task.attachments_changed`.
