# TC-061: Закрытие Task Modal по крестику

**Предусловие:** Пользователь авторизован в системе. Открыт Task Modal задачи #1 поверх Filter Results Screen.

**Шаги:**
1. Нажать кнопку закрытия (крестик) в Task Modal.

**Ожидаемый результат:**
- Task Modal закрывается.
- Пользователь возвращается на Filter Results Screen.
- URL браузера соответствует Filter Results Screen.
