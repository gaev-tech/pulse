# TC-078: Отображение дочерних задач (только чтение)

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи «ACME-10: Эпик проекта». У задачи есть дочерние задачи «ACME-11: Подзадача 1» и «ACME-12: Подзадача 2».

**Шаги:**
1. В правой части Task Screen найти раздел «Дочерние задачи».
2. Убедиться, что отображаются задачи «ACME-11: Подзадача 1» и «ACME-12: Подзадача 2».

**Ожидаемый результат:**
- В разделе «Дочерние задачи» отображаются «ACME-11: Подзадача 1» и «ACME-12: Подзадача 2».
- Раздел доступен только для чтения — нет кнопок добавления или удаления дочерних задач.
