# TC-087: Фиксация факта открытия задачи (last opened)

**Предусловие:** Пользователь авторизован. Существуют задачи «ACME-10» и «ACME-20». Пользователь ранее не открывал ни одну из них.

**Шаги:**
1. Открыть Task Screen задачи «ACME-20».
2. Вернуться на предыдущий экран.
3. Открыть Task Screen задачи «ACME-10».
4. Вернуться на предыдущий экран.
5. Открыть Search Modal и ввести «ACME».

**Ожидаемый результат:**
- В результатах поиска задача «ACME-10» отображается выше «ACME-20», так как она была открыта позже.
- Факт открытия задачи зафиксирован на бэкенде при каждом открытии Task Screen.
