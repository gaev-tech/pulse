# TC-056: Генерация ключа личной задачи в формате #NNN

**Предусловие:** Пользователь авторизован в системе. У пользователя уже есть личные задачи (последний ключ — #5).

**Шаги:**
1. Открыть Task Create Modal.
2. Убедиться, что выбран вариант "Личная задача".
3. Ввести название задачи: "Новая личная задача".
4. Нажать кнопку создания задачи.

**Ожидаемый результат:**
- Задача создана с ключом #6 (следующий порядковый номер после #5).
- Ключ отображается в Task Modal созданной задачи.
- Ключ уникален в рамках задач пользователя.
