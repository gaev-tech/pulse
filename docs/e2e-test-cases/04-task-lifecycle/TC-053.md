# TC-053: Отображение только команд с правом task.create в Task Create Modal

**Предусловие:** Пользователь авторизован в системе. Пользователь является участником команды "ACME" с правом `task.create` и участником команды "BETA" без права `task.create`. Открыт Task Create Modal.

**Шаги:**
1. Открыть список доступных команд в поле выбора владельца задачи.

**Ожидаемый результат:**
- В списке команд отображается только "ACME" (команда с правом `task.create`).
- Команда "BETA" (без права `task.create`) не отображается в списке.
