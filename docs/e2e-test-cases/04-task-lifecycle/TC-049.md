# TC-049: Создание личной задачи с названием

**Предусловие:** Пользователь авторизован в системе. Открыт Task Create Modal.

**Шаги:**
1. Убедиться, что выбран вариант "Личная задача" в качестве владельца.
2. Ввести название задачи: "Подготовить отчёт за квартал".
3. Нажать кнопку создания задачи.

**Ожидаемый результат:**
- Задача создана с названием "Подготовить отчёт за квартал".
- Задаче присвоен ключ в формате #NNN (например, #1).
- Task Create Modal закрывается.
- Открывается Task Modal созданной задачи.
- Пользователь остаётся на текущей странице (под модалкой).
