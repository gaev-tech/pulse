# TC-084: Подтверждение удаления задачи

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи «ACME-42: Тестовая задача». Отображается Confirmation Popup удаления задачи.

**Шаги:**
1. В Confirmation Popup нажать красную кнопку «Да, подтверждаю».

**Ожидаемый результат:**
- Confirmation Popup закрывается.
- Задача «ACME-42: Тестовая задача» удалена.
- Пользователь перенаправлен с Task Screen на предыдущий экран.
- Задача больше не отображается в результатах фильтров и поиска.
