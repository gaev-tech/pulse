# TC-070: Снятие ответственного с задачи

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи с ответственным «Иван Петров». У пользователя есть право `edit.assignee`.

**Шаги:**
1. В правой части Task Screen найти поле «Ответственный» со значением «Иван Петров».
2. Кликнуть на кнопку очистки поля ответственного.

**Ожидаемый результат:**
- Поле «Ответственный» отображается пустым.
- В ленте событий задачи появилось событие `task.assignee_changed`.
