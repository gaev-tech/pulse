# TC-083: Отмена удаления задачи в Confirmation Popup

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи «ACME-42: Тестовая задача». Отображается Confirmation Popup удаления задачи.

**Шаги:**
1. В Confirmation Popup нажать кнопку «Отмена».

**Ожидаемый результат:**
- Confirmation Popup закрывается.
- Task Screen задачи «ACME-42: Тестовая задача» остаётся открытым.
- Задача не удалена, данные не изменились.
