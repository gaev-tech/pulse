# TC-072: Добавление нескольких меток к задаче

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи с меткой «Срочно». У пользователя есть право `edit.labels`. Существуют метки «Баг» синего цвета и «Документация» зелёного цвета.

**Шаги:**
1. В правой части Task Screen найти поле «Метки» с меткой «Срочно».
2. Кликнуть на поле меток.
3. Выбрать метку «Баг» из списка доступных меток.
4. Выбрать метку «Документация» из списка доступных меток.

**Ожидаемый результат:**
- В поле «Метки» отображаются три метки: «Срочно», «Баг», «Документация».
- В ленте событий задачи появилось событие `task.labels_changed`.
