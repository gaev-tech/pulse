# TC-017: Переключение темы на тёмную

**Предусловие:** Пользователь авторизован в системе. Текущая тема — светлая или системная.

**Шаги:**
1. Нажать на аватарку и имя пользователя в нижней части Navigation Sidebar — открывается Profile Popup.
2. В переключателе темы выбрать "Тёмная".

**Ожидаемый результат:**
- Интерфейс переключается на тёмную тему.
- Цветовая схема всех элементов соответствует тёмной теме.
- Настройка темы сохраняется на бэкенде для текущего пользователя.
