# TC-018: Переключение темы на системную

**Предусловие:** Пользователь авторизован в системе. Текущая тема — светлая или тёмная.

**Шаги:**
1. Нажать на аватарку и имя пользователя в нижней части Navigation Sidebar — открывается Profile Popup.
2. В переключателе темы выбрать "Системная".

**Ожидаемый результат:**
- Интерфейс переключается на тему, соответствующую настройкам операционной системы.
- Если ОС использует тёмную тему — отображается тёмная; если светлую — светлая.
- Настройка темы сохраняется на бэкенде для текущего пользователя.
