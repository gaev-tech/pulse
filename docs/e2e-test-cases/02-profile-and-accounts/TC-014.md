# TC-014: Переключение языка на русский

**Предусловие:** Пользователь авторизован в системе. Текущий язык интерфейса — English.

**Шаги:**
1. Нажать на аватарку и имя пользователя в нижней части Navigation Sidebar — открывается Profile Popup.
2. В переключателе языка выбрать "Русский".

**Ожидаемый результат:**
- Язык интерфейса переключается на русский.
- Все элементы интерфейса отображаются на русском языке.
- Настройка языка сохраняется на бэкенде для текущего пользователя.
