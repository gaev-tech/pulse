# TC-021: Добавление нового пользователя из Profile Popup — переход на Auth Screen

**Предусловие:** Пользователь авторизован в системе. Открыт любой экран приложения.

**Шаги:**
1. Нажать на аватарку и имя пользователя в нижней части Navigation Sidebar — открывается Profile Popup.
2. Нажать кнопку "Добавить пользователя" в конце списка авторизованных пользователей.

**Ожидаемый результат:**
- Пользователь перенаправлен на Auth Screen.
- На Auth Screen можно ввести email нового аккаунта для авторизации.
- Текущая сессия первого пользователя сохраняется в localStorage.
