# TC-023: Сохранение языка и темы при переключении аккаунта

**Предусловие:** На клиенте авторизованы два аккаунта: user1@example.com (язык — русский, тема — тёмная) и user2@example.com (язык — English, тема — светлая). Активная сессия — user1@example.com.

**Шаги:**
1. Нажать на аватарку и имя пользователя в нижней части Navigation Sidebar — открывается Profile Popup.
2. Переключиться на user2@example.com — язык меняется на English, тема на светлую.
3. Открыть Profile Popup повторно.
4. Переключиться обратно на user1@example.com.

**Ожидаемый результат:**
- После переключения на user1 язык интерфейса возвращается на русский.
- Тема интерфейса возвращается на тёмную.
- Настройки каждого пользователя сохраняются независимо и не влияют друг на друга.
