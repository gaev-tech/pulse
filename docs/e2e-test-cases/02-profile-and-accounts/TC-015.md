# TC-015: Переключение языка на английский

**Предусловие:** Пользователь авторизован в системе. Текущий язык интерфейса — русский.

**Шаги:**
1. Нажать на аватарку и имя пользователя в нижней части Navigation Sidebar — открывается Profile Popup.
2. В переключателе языка выбрать "English".

**Ожидаемый результат:**
- Язык интерфейса переключается на английский.
- Все элементы интерфейса отображаются на английском языке.
- Настройка языка сохраняется на бэкенде для текущего пользователя.
