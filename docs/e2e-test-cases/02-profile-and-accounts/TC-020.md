# TC-020: Переключение аккаунта — переход на Personal Activity и применение настроек

**Предусловие:** На клиенте авторизованы два аккаунта: user1@example.com (активный, язык — English, тема — светлая) и user2@example.com (язык — русский, тема — тёмная).

**Шаги:**
1. Нажать на аватарку и имя пользователя в нижней части Navigation Sidebar — открывается Profile Popup.
2. Нажать на user2@example.com в списке авторизованных пользователей.

**Ожидаемый результат:**
- Активная сессия переключается на user2@example.com.
- Открывается Personal Activity Screen пользователя user2@example.com.
- Язык интерфейса переключается на русский (настройка user2).
- Тема интерфейса переключается на тёмную (настройка user2).
- В Navigation Sidebar отображаются аватарка и имя user2.
