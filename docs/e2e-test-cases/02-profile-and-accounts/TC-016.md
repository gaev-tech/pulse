# TC-016: Переключение темы на светлую

**Предусловие:** Пользователь авторизован в системе. Текущая тема — тёмная или системная.

**Шаги:**
1. Нажать на аватарку и имя пользователя в нижней части Navigation Sidebar — открывается Profile Popup.
2. В переключателе темы выбрать "Светлая".

**Ожидаемый результат:**
- Интерфейс переключается на светлую тему.
- Цветовая схема всех элементов соответствует светлой теме.
- Настройка темы сохраняется на бэкенде для текущего пользователя.
