# TC-013: Открытие Profile Popup по клику на аватарку

**Предусловие:** Пользователь авторизован в системе. Открыт любой экран приложения.

**Шаги:**
1. Нажать на аватарку и имя пользователя в нижней части Navigation Sidebar.

**Ожидаемый результат:**
- Открывается Profile Popup.
- В Profile Popup отображаются: переключатель языка, переключатель темы, список авторизованных пользователей с кнопкой "Добавить пользователя", кнопка выхода из системы.
