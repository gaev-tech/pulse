# TC-019: Отображение списка авторизованных пользователей

**Предусловие:** На клиенте авторизованы два аккаунта: user1@example.com и user2@example.com. Активная сессия — user1@example.com.

**Шаги:**
1. Нажать на аватарку и имя пользователя в нижней части Navigation Sidebar — открывается Profile Popup.
2. Просмотреть список авторизованных пользователей в Profile Popup.

**Ожидаемый результат:**
- В списке отображаются оба пользователя: user1@example.com и user2@example.com.
- Текущий активный пользователь (user1@example.com) выделен визуально.
- В конце списка отображается кнопка "Добавить пользователя".
