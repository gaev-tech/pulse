# TC-307: Отображение события filter.deleted в ленте

**Предусловие:** Пользователь авторизован. Существует личный фильтр «Временный фильтр». Пользователь находится на Personal Activity Screen.

**Шаги:**
1. Открыть Filter Results Screen фильтра «Временный фильтр» через Navigation Sidebar.
2. Нажать кнопку «...» на верхней панели для открытия Filter Popup.
3. Выбрать пункт удаления фильтра.
4. Подтвердить удаление в Confirmation Popup, нажав кнопку «Да, подтверждаю».
5. Проверить ленту на Personal Activity Screen.

**Ожидаемый результат:**
- В ленте отображается событие filter.deleted.
- Событие содержит: название фильтра «Временный фильтр», автор — текущий пользователь, дата удаления.
