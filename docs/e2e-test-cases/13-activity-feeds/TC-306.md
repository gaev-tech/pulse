# TC-306: Отображение события filter.updated в ленте

**Предусловие:** Пользователь авторизован. Существует личный фильтр «Мои задачи» с критерием status==opened.

**Шаги:**
1. Открыть Filter Results Screen фильтра «Мои задачи» через Navigation Sidebar.
2. Изменить критерии: добавить фильтр по ответственному — выбрать текущего пользователя.
3. Нажать кнопку «Сохранить фильтр».
4. Перейти на Filter Activity Screen фильтра «Мои задачи».

**Ожидаемый результат:**
- В ленте отображается событие filter.updated.
- Событие содержит: название фильтра «Мои задачи», автор — текущий пользователь, информацию об изменении критериев.
