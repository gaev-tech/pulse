# TC-289: Отображение события task.created в ленте

**Предусловие:** Пользователь авторизован и находится на Personal Activity Screen.

**Шаги:**
1. Открыть Task Create Modal (кнопка создания задачи).
2. Выбрать владельца — личная задача.
3. Ввести название «Новая задача для теста».
4. Нажать кнопку создания.
5. Закрыть Task Modal созданной задачи.
6. Проверить ленту на Personal Activity Screen.

**Ожидаемый результат:**
- В ленте отображается событие task.created.
- Событие содержит: тип — создание задачи, ключ созданной задачи, название «Новая задача для теста», автор — текущий пользователь, дата и время создания.
