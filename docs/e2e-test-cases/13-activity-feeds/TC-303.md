# TC-303: Отображение события team.member_removed в ленте

**Предусловие:** Пользователь A авторизован, является владельцем команды «ACME». Пользователь B является участником команды «ACME». Пользователь A находится на Team Activity Screen команды «ACME».

**Шаги:**
1. Открыть Team Edit Modal команды «ACME».
2. Нажать кнопку удаления Пользователя B из команды.
3. Подтвердить удаление в Confirmation Popup, нажав кнопку «Да, подтверждаю».
4. Закрыть модалку.
5. Проверить ленту на Team Activity Screen.

**Ожидаемый результат:**
- В ленте отображается событие team.member_removed.
- Событие содержит: команда «ACME», автор — Пользователь A, удалённый участник — Пользователь B.
