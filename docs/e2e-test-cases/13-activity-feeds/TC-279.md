# TC-279: Team Activity: отображение всех событий по задачам команды

**Предусловие:** Пользователь является участником команды «ACME». В команде есть задачи ACME-1, ACME-2, ACME-3. По задачам совершены различные изменения несколькими участниками команды.

**Шаги:**
1. В Navigation Sidebar раскрыть раздел «Команды».
2. Нажать на команду «ACME» для перехода на Team Activity Screen.

**Ожидаемый результат:**
- Отображается лента событий команды «ACME».
- В ленте присутствуют все события по задачам ACME-1, ACME-2, ACME-3.
- Отображаются события всех типов: создание задач, изменения полей, добавление/удаление участников команды.
- События отсортированы по дате (от новых к старым).
