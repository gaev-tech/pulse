# TC-276: Personal Activity: отображение событий по доступным задачам

**Предусловие:** Пользователь авторизован. У пользователя есть доступ к личной задаче #1 и командной задаче ACME-10. По обеим задачам были совершены изменения (изменение статуса, назначение ответственного).

**Шаги:**
1. В Navigation Sidebar нажать кнопку перехода на Personal Activity Screen.

**Ожидаемый результат:**
- Отображается лента событий.
- В ленте присутствуют события по задаче #1 (личная) и задаче ACME-10 (командная).
- Каждое событие содержит информацию о типе изменения, авторе, дате и задаче.
- События отсортированы по дате (от новых к старым).
