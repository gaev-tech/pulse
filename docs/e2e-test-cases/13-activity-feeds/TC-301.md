# TC-301: Отображение события task.shared в ленте

**Предусловие:** Пользователь A авторизован, является владельцем задачи #5. Пользователь A имеет право share на задачу. Пользователь B зарегистрирован в системе. Пользователь A находится на Personal Activity Screen.

**Шаги:**
1. Открыть задачу #5 в Task Modal.
2. Открыть Task Share Modal.
3. В поле поиска найти Пользователя B по email.
4. Выбрать права для выдачи (например, edit.title).
5. Подтвердить шаринг.
6. Закрыть Task Share Modal и Task Modal.
7. Проверить ленту на Personal Activity Screen.

**Ожидаемый результат:**
- В ленте отображается событие task.shared.
- Событие содержит: задача #5, автор — Пользователь A, информацию о выдаче доступа Пользователю B.
