# TC-295: Отображение события task.labels_changed в ленте

**Предусловие:** Пользователь авторизован, имеет право edit.labels на задачу ACME-3. У задачи нет меток. В команде «ACME» существует метка «Баг». Пользователь находится на Personal Activity Screen.

**Шаги:**
1. Открыть задачу ACME-3 в Task Modal.
2. В правой панели найти поле «Метки».
3. Добавить метку «Баг».
4. Закрыть Task Modal.
5. Проверить ленту на Personal Activity Screen.

**Ожидаемый результат:**
- В ленте отображается событие task.labels_changed.
- Событие содержит: задача ACME-3, автор — текущий пользователь, информацию об изменении меток.
