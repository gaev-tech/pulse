# TC-300: Отображение события task.attachments_changed в ленте

**Предусловие:** Пользователь авторизован, имеет право edit.attachments на задачу ACME-3. У задачи нет вложений. Пользователь находится на Personal Activity Screen.

**Шаги:**
1. Открыть задачу ACME-3 в Task Modal.
2. В правой панели найти поле «Вложения».
3. Загрузить файл «report.pdf».
4. Закрыть Task Modal.
5. Проверить ленту на Personal Activity Screen.

**Ожидаемый результат:**
- В ленте отображается событие task.attachments_changed.
- Событие содержит: задача ACME-3, автор — текущий пользователь, информацию об изменении вложений.
