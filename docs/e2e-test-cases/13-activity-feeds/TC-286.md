# TC-286: Filter Activity: отображение событий задачи в период вхождения в результаты

**Предусловие:** Пользователь авторизован. Существует фильтр «Открытые задачи» с критерием status==opened. Задача ACME-3 имеет статус «opened» и входит в результаты фильтра.

**Шаги:**
1. Открыть Filter Activity Screen фильтра «Открытые задачи» через Navigation Sidebar.
2. В другой вкладке изменить название задачи ACME-3 с «Старое название» на «Новое название».
3. Вернуться на Filter Activity Screen.

**Ожидаемый результат:**
- В ленте отображается событие task.title_changed по задаче ACME-3.
- Событие показывает изменение названия задачи, автора и дату.
- Событие отображается, потому что задача ACME-3 находилась в результатах фильтра на момент изменения.
