# TC-293: Отображение события task.status_changed в ленте

**Предусловие:** Пользователь авторизован, имеет право edit.status на задачу ACME-3 со статусом «opened». Пользователь находится на Personal Activity Screen.

**Шаги:**
1. Открыть задачу ACME-3 в Task Modal.
2. В правой панели найти поле «Статус».
3. Изменить статус с «opened» на «closed».
4. Закрыть Task Modal.
5. Проверить ленту на Personal Activity Screen.

**Ожидаемый результат:**
- В ленте отображается событие task.status_changed.
- Событие содержит: задача ACME-3, автор — текущий пользователь, старый статус — opened, новый статус — closed.
