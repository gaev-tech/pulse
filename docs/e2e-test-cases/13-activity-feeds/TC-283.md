# TC-283: Filter Activity: отображение событий изменения критериев фильтра

**Предусловие:** Пользователь авторизован. Существует личный фильтр «Мои открытые». Пользователь ранее изменял критерии этого фильтра.

**Шаги:**
1. В Navigation Sidebar в разделе «Личные фильтры» нажать кнопку открытия Filter Activity Screen напротив фильтра «Мои открытые».

**Ожидаемый результат:**
- Открывается Filter Activity Screen.
- В ленте отображается событие filter.updated с информацией об изменении критериев фильтра.
- Событие содержит автора изменения и дату.
