# TC-305: Отображение события filter.created в ленте

**Предусловие:** Пользователь авторизован и находится на Personal Activity Screen.

**Шаги:**
1. В Navigation Sidebar в разделе «Личные фильтры» нажать кнопку создания фильтра.
2. В Filter Create Modal ввести название «Тестовый фильтр».
3. Выбрать тип — личный.
4. В простом режиме указать критерий: статус — opened.
5. Нажать кнопку создания фильтра.
6. Перейти на Filter Activity Screen созданного фильтра.

**Ожидаемый результат:**
- В ленте отображается событие filter.created.
- Событие содержит: название фильтра «Тестовый фильтр», автор — текущий пользователь, дата создания.
