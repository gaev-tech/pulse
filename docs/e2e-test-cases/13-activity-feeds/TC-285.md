# TC-285: Filter Activity: отображение событий task_left

**Предусловие:** Пользователь авторизован. Существует фильтр «Открытые задачи» с критерием status==opened. Задача ACME-7 имеет статус «opened» и входит в результаты фильтра.

**Шаги:**
1. Открыть Filter Activity Screen фильтра «Открытые задачи» через Navigation Sidebar.
2. В другой вкладке (или другим пользователем) изменить статус задачи ACME-7 с «opened» на «closed».
3. Вернуться на Filter Activity Screen.

**Ожидаемый результат:**
- В ленте появляется событие filter.task_left для задачи ACME-7.
- Событие указывает, что задача ACME-7 вышла из результатов фильтра «Открытые задачи».
- Событие содержит информацию о задаче (ключ, название) и авторе изменения.
