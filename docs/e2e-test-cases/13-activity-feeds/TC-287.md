# TC-287: Filter Activity: событие filter.updated при изменении критериев (без task_entered/task_left)

**Предусловие:** Пользователь авторизован. Существует фильтр «Мои задачи» с критерием status==opened. В результатах фильтра есть задачи ACME-1 и ACME-2.

**Шаги:**
1. Открыть Filter Activity Screen фильтра «Мои задачи» через Navigation Sidebar.
2. Перейти на Filter Results Screen того же фильтра.
3. Изменить критерии фильтра: добавить фильтр по ответственному.
4. Нажать кнопку «Сохранить фильтр».
5. Вернуться на Filter Activity Screen.

**Ожидаемый результат:**
- В ленте появляется событие filter.updated.
- События filter.task_entered и filter.task_left НЕ генерируются при изменении критериев фильтра.
- Событие filter.updated содержит информацию об авторе и дате изменения.
