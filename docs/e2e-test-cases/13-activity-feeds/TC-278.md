# TC-278: Personal Activity: динамическое обновление ленты

**Предусловие:** Пользователь A авторизован и находится на Personal Activity Screen. Пользователь B имеет доступ к задаче #3, принадлежащей Пользователю A.

**Шаги:**
1. Пользователь A открывает Personal Activity Screen и фиксирует текущий список событий.
2. Пользователь B изменяет статус задачи #3 с «opened» на «closed».

**Ожидаемый результат:**
- В ленте Пользователя A автоматически (без перезагрузки страницы) появляется новое событие task.status_changed по задаче #3.
- Новое событие отображается в верхней части ленты.
- Событие содержит информацию: автор — Пользователь B, задача — #3, старый статус — opened, новый статус — closed.
