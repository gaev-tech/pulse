# TC-299: Отображение события task.parent_changed в ленте

**Предусловие:** Пользователь авторизован, имеет право edit.parent на задачу ACME-3. У задачи нет родительской задачи. Существует задача ACME-1. Пользователь находится на Personal Activity Screen.

**Шаги:**
1. Открыть задачу ACME-3 в Task Modal.
2. В правой панели найти поле «Родительская задача».
3. Выбрать задачу ACME-1 в качестве родительской.
4. Закрыть Task Modal.
5. Проверить ленту на Personal Activity Screen.

**Ожидаемый результат:**
- В ленте отображается событие task.parent_changed.
- Событие содержит: задача ACME-3, автор — текущий пользователь, новый родитель — ACME-1.
