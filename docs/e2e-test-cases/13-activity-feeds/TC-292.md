# TC-292: Отображение события task.description_changed в ленте

**Предусловие:** Пользователь авторизован, имеет право edit.description на задачу ACME-3. Описание задачи пустое. Пользователь находится на Personal Activity Screen.

**Шаги:**
1. Открыть задачу ACME-3 в Task Modal.
2. Нажать на поле описания для редактирования.
3. Ввести описание: «Описание задачи в формате markdown».
4. Сохранить изменение.
5. Закрыть Task Modal.
6. Проверить ленту на Personal Activity Screen.

**Ожидаемый результат:**
- В ленте отображается событие task.description_changed.
- Событие содержит: задача ACME-3, автор — текущий пользователь, информацию об изменении описания.
