# TC-296: Отображение события task.links_changed в ленте

**Предусловие:** Пользователь авторизован, имеет право edit.links на задачу ACME-3. У задачи нет внешних ссылок. Пользователь находится на Personal Activity Screen.

**Шаги:**
1. Открыть задачу ACME-3 в Task Modal.
2. В правой панели найти поле «Ссылки» и открыть Related Links Popup.
3. Добавить внешнюю ссылку: URL — «https://example.com», название — «Документация».
4. Закрыть Related Links Popup и Task Modal.
5. Проверить ленту на Personal Activity Screen.

**Ожидаемый результат:**
- В ленте отображается событие task.links_changed.
- Событие содержит: задача ACME-3, автор — текущий пользователь, информацию об изменении ссылок.
