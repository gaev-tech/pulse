# TC-277: Personal Activity: учёт окна доступа -- события только за период доступа

**Предусловие:** Пользователь B получил доступ к задаче ACME-5 от Пользователя A в 10:00. До выдачи доступа (в 09:00) по задаче ACME-5 было событие изменения названия. После выдачи доступа (в 11:00) по задаче было событие изменения статуса.

**Шаги:**
1. Пользователь B переходит на Personal Activity Screen через Navigation Sidebar.
2. Просмотреть ленту событий, найти события по задаче ACME-5.

**Ожидаемый результат:**
- В ленте Пользователя B отображается событие изменения статуса задачи ACME-5 (произошло в 11:00, после выдачи доступа).
- Событие изменения названия задачи ACME-5 (произошло в 09:00, до выдачи доступа) НЕ отображается в ленте.
- Отображается событие выдачи доступа (task.shared) в 10:00.
