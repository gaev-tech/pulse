# TC-297: Отображение события task.relations_changed в ленте

**Предусловие:** Пользователь авторизован, имеет право edit.relations на задачу ACME-3. У задачи нет связанных задач. Существует задача ACME-7. Пользователь находится на Personal Activity Screen.

**Шаги:**
1. Открыть задачу ACME-3 в Task Modal.
2. В правой панели найти поле «Связи» и открыть Related Tasks Popup.
3. В поиске найти задачу ACME-7 и добавить её как связанную.
4. Закрыть Related Tasks Popup и Task Modal.
5. Проверить ленту на Personal Activity Screen.

**Ожидаемый результат:**
- В ленте отображается событие task.relations_changed.
- Событие содержит: задача ACME-3, автор — текущий пользователь, информацию об изменении связей.
