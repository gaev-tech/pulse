# TC-282: Team Activity: динамическое обновление ленты

**Предусловие:** Пользователь A является участником команды «ACME» и находится на Team Activity Screen. Пользователь B — также участник команды «ACME».

**Шаги:**
1. Пользователь A открывает Team Activity Screen команды «ACME» и фиксирует текущий список событий.
2. Пользователь B создаёт новую задачу ACME-15 в команде «ACME».

**Ожидаемый результат:**
- В ленте Пользователя A автоматически (без перезагрузки страницы) появляется новое событие task.created по задаче ACME-15.
- Новое событие отображается в верхней части ленты.
- Событие содержит информацию: автор — Пользователь B, задача — ACME-15.
