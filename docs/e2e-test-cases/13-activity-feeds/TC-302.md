# TC-302: Отображение события team.member_added в ленте

**Предусловие:** Пользователь A авторизован, является владельцем команды «ACME». Пользователь B зарегистрирован, но не является участником команды. Пользователь A находится на Team Activity Screen команды «ACME».

**Шаги:**
1. Открыть Team Edit Modal команды «ACME».
2. Добавить Пользователя B в команду по email.
3. Выдать право task.create.
4. Сохранить изменения и закрыть модалку.
5. Проверить ленту на Team Activity Screen.

**Ожидаемый результат:**
- В ленте отображается событие team.member_added.
- Событие содержит: команда «ACME», автор — Пользователь A, добавленный участник — Пользователь B, выданные права.
