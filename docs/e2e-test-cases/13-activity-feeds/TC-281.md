# TC-281: Team Activity: фильтр по авторам изменений (несколько авторов)

**Предусловие:** Пользователь является участником команды «ACME». В команде есть Пользователь A, Пользователь B и Пользователь C. Все трое совершали изменения по задачам команды.

**Шаги:**
1. Перейти на Team Activity Screen команды «ACME».
2. В фильтре по авторам изменений (Team Activity Filter By Member) выбрать Пользователя A.
3. Дополнительно выбрать Пользователя B.

**Ожидаемый результат:**
- В ленте отображаются события, автором которых является Пользователь A или Пользователь B.
- События Пользователя C не отображаются.
- Фильтр визуально показывает, что выбраны Пользователь A и Пользователь B.
