# TC-284: Filter Activity: отображение событий task_entered

**Предусловие:** Пользователь авторизован. Существует фильтр «Открытые задачи» с критерием status==opened. Задача ACME-5 имеет статус «closed».

**Шаги:**
1. Открыть Filter Activity Screen фильтра «Открытые задачи» через Navigation Sidebar.
2. В другой вкладке (или другим пользователем) изменить статус задачи ACME-5 с «closed» на «opened».
3. Вернуться на Filter Activity Screen.

**Ожидаемый результат:**
- В ленте появляется событие filter.task_entered для задачи ACME-5.
- Событие указывает, что задача ACME-5 вошла в результаты фильтра «Открытые задачи».
- Событие содержит информацию о задаче (ключ, название) и авторе изменения.
