# TC-304: Отображение события team.member_permissions_changed в ленте

**Предусловие:** Пользователь A авторизован, является владельцем команды «ACME». Пользователь B является участником команды с правом task.create. Пользователь A находится на Team Activity Screen команды «ACME».

**Шаги:**
1. Открыть Team Edit Modal команды «ACME».
2. Найти Пользователя B в списке участников.
3. Добавить ему право team.manage_filters.
4. Сохранить изменения и закрыть модалку.
5. Проверить ленту на Team Activity Screen.

**Ожидаемый результат:**
- В ленте отображается событие team.member_permissions_changed.
- Событие содержит: команда «ACME», автор — Пользователь A, участник — Пользователь B, старые права, новые права.
