# TC-290: Отображение события task.deleted в ленте

**Предусловие:** Пользователь авторизован, является владельцем личной задачи #5 «Задача для удаления». Пользователь находится на Personal Activity Screen.

**Шаги:**
1. Открыть задачу #5 в Task Modal.
2. Удалить задачу (через кнопку удаления).
3. Подтвердить удаление в Confirmation Popup, нажав кнопку «Да, подтверждаю».
4. Проверить ленту на Personal Activity Screen.

**Ожидаемый результат:**
- В ленте отображается событие task.deleted.
- Событие содержит: тип — удаление задачи, ключ #5, название «Задача для удаления», автор — текущий пользователь, дата и время удаления.
