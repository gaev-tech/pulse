# TC-298: Отображение события task.blocking_changed в ленте

**Предусловие:** Пользователь авторизован, имеет право edit.blocking на задачу ACME-3. У задачи нет блокирующих задач. Существует задача ACME-8, которая не образует цикл блокирования. Пользователь находится на Personal Activity Screen.

**Шаги:**
1. Открыть задачу ACME-3 в Task Modal.
2. В правой панели найти поле «Блокирование» и открыть Blocking Tasks Popup.
3. В поиске найти задачу ACME-8 и добавить её как блокирующую.
4. Закрыть Blocking Tasks Popup и Task Modal.
5. Проверить ленту на Personal Activity Screen.

**Ожидаемый результат:**
- В ленте отображается событие task.blocking_changed.
- Событие содержит: задача ACME-3, автор — текущий пользователь, информацию об изменении блокирующих задач.
