# TC-280: Team Activity: фильтр по авторам изменений (один автор)

**Предусловие:** Пользователь является участником команды «ACME». В команде есть Пользователь A и Пользователь B. Оба совершали изменения по задачам команды.

**Шаги:**
1. Перейти на Team Activity Screen команды «ACME».
2. В фильтре по авторам изменений (Team Activity Filter By Member) выбрать Пользователя A.

**Ожидаемый результат:**
- В ленте отображаются только события, автором которых является Пользователь A.
- События Пользователя B не отображаются.
- Фильтр визуально показывает, что выбран Пользователь A.
