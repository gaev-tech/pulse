# TC-288: Filter Activity: динамическое обновление ленты

**Предусловие:** Пользователь A авторизован и находится на Filter Activity Screen фильтра «Открытые задачи» (критерий status==opened). Задача ACME-10 имеет статус «closed».

**Шаги:**
1. Пользователь A находится на Filter Activity Screen и фиксирует текущий список событий.
2. Пользователь B изменяет статус задачи ACME-10 с «closed» на «opened».

**Ожидаемый результат:**
- В ленте Пользователя A автоматически (без перезагрузки страницы) появляется новое событие filter.task_entered для задачи ACME-10.
- Новое событие отображается в верхней части ленты.
