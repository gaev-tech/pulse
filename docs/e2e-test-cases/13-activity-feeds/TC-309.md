# TC-309: Отображение события label.deleted в ленте

**Предусловие:** Пользователь авторизован. Существует личная метка «Устаревшая». Пользователь находится на Personal Activity Screen.

**Шаги:**
1. В Navigation Sidebar нажать кнопку открытия модалки личных меток.
2. Найти метку «Устаревшая» в списке.
3. Нажать кнопку удаления метки.
4. Подтвердить удаление в Confirmation Popup, нажав кнопку «Да, подтверждаю».
5. Закрыть модалку меток.
6. Проверить ленту на Personal Activity Screen.

**Ожидаемый результат:**
- В ленте отображается событие label.deleted.
- Событие содержит: название удалённой метки «Устаревшая», автор — текущий пользователь, дата удаления.
