# TC-308: Отображение события label.renamed в ленте

**Предусловие:** Пользователь авторизован. Существует личная метка «Срочно». Пользователь находится на Personal Activity Screen.

**Шаги:**
1. В Navigation Sidebar нажать кнопку открытия модалки личных меток.
2. Найти метку «Срочно» в списке.
3. Изменить название метки с «Срочно» на «Критично».
4. Сохранить изменение.
5. Закрыть модалку меток.
6. Проверить ленту на Personal Activity Screen.

**Ожидаемый результат:**
- В ленте отображается событие label.renamed.
- Событие содержит: старое название — «Срочно», новое название — «Критично», автор — текущий пользователь.
