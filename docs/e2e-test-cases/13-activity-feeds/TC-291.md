# TC-291: Отображение события task.title_changed в ленте

**Предусловие:** Пользователь авторизован, имеет право edit.title на задачу ACME-3 с текущим названием «Старое название». Пользователь находится на Personal Activity Screen.

**Шаги:**
1. Открыть задачу ACME-3 в Task Modal.
2. Нажать на название задачи для редактирования.
3. Изменить название с «Старое название» на «Новое название».
4. Сохранить изменение.
5. Закрыть Task Modal.
6. Проверить ленту на Personal Activity Screen.

**Ожидаемый результат:**
- В ленте отображается событие task.title_changed.
- Событие содержит: задача ACME-3, автор — текущий пользователь, старое название — «Старое название», новое название — «Новое название».
