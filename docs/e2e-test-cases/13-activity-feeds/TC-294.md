# TC-294: Отображение события task.assignee_changed в ленте

**Предусловие:** Пользователь авторизован, имеет право edit.assignee на задачу ACME-3. Ответственный не назначен. Пользователь B является участником команды «ACME». Пользователь находится на Personal Activity Screen.

**Шаги:**
1. Открыть задачу ACME-3 в Task Modal.
2. В правой панели найти поле «Ответственный».
3. Назначить Пользователя B ответственным.
4. Закрыть Task Modal.
5. Проверить ленту на Personal Activity Screen.

**Ожидаемый результат:**
- В ленте отображается событие task.assignee_changed.
- Событие содержит: задача ACME-3, автор — текущий пользователь, новый ответственный — Пользователь B.
