# TC-395: Тулбар редактора: кнопка «Изображение» — вставка изображения по URL

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи. У пользователя есть право `edit.description`. Поле описания пустое.

**Шаги:**
1. Кликнуть на поле описания задачи для перехода в режим редактирования.
2. Нажать кнопку «Изображение» (иконка с картинкой) на тулбаре редактора.
3. В появившемся поле ввести URL изображения: `https://example.com/diagram.png`.
4. В поле альтернативного текста ввести: «Диаграмма архитектуры».
5. Подтвердить вставку изображения.
6. Сохранить описание, кликнув вне поля.

**Ожидаемый результат:**
- В редактор вставляется markdown-разметка: `![Диаграмма архитектуры](https://example.com/diagram.png)`.
- После сохранения изображение отображается в описании задачи.
