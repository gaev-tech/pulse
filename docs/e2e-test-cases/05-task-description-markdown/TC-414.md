# TC-414: Завершение списка при двойном нажатии Enter на пустом пункте

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи. У пользователя есть право `edit.description`. Поле описания пустое.

**Шаги:**
1. Кликнуть на поле описания задачи для перехода в режим редактирования.
2. Ввести `- Первый пункт` и нажать Enter.
3. Ввести «Второй пункт» и нажать Enter.
4. На новой строке с автоматическим префиксом `- ` не вводить текст и нажать Enter.
5. Убедиться, что пустой пункт списка удалён и курсор перешёл на обычную строку.
6. Ввести обычный текст «Абзац после списка».
7. Сохранить описание, кликнув вне поля.

**Ожидаемый результат:**
- При нажатии Enter на пустом пункте списка (строка содержит только префикс `- `) список завершается.
- Пустой пункт списка удаляется.
- Курсор переходит на новую строку без префикса списка.
- После сохранения список содержит два пункта, за которыми следует обычный текст.
