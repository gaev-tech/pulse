# TC-388: Тулбар редактора: кнопка «Заголовок» — выбор уровня (1–5)

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи. У пользователя есть право `edit.description`. Поле описания содержит текст «Текст заголовка».

**Шаги:**
1. Кликнуть на поле описания задачи для перехода в режим редактирования.
2. Установить курсор на строку «Текст заголовка».
3. Нажать кнопку «Заголовок» (иконка H) на тулбаре редактора.
4. В выпадающем меню выбрать «Заголовок 1».
5. Убедиться, что строка превратилась в заголовок первого уровня.
6. Повторно нажать кнопку «Заголовок» и выбрать «Заголовок 3».
7. Сохранить описание, кликнув вне поля.

**Ожидаемый результат:**
- При выборе «Заголовок 1» строка получает префикс `# `.
- При смене на «Заголовок 3» префикс заменяется на `### `.
- В выпадающем меню доступны уровни от 1 до 5.
- После сохранения заголовок отображается соответствующим размером шрифта.
