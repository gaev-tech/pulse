# TC-100: Markdown: блок кода

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи с пустым описанием. У пользователя есть право `edit.description`.

**Шаги:**
1. Кликнуть на поле описания задачи.
2. Ввести текст:
   ````
   ```javascript
   function hello() {
     console.log("Hello, world!");
   }
   ```
   ````
3. Сохранить описание, кликнув вне поля.
4. Просмотреть отрендеренное описание.

**Ожидаемый результат:**
- Отображается блок кода с моноширинным шрифтом и фоновой подсветкой.
- Код отображается с сохранением форматирования и отступов.
- Блок кода визуально выделен от остального текста описания.
