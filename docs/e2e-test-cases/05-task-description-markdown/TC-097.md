# TC-097: Markdown: чеклист

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи с пустым описанием. У пользователя есть право `edit.description`.

**Шаги:**
1. Кликнуть на поле описания задачи.
2. Ввести текст:
   ```
   - [x] Выполненный пункт
   - [ ] Невыполненный пункт
   - [ ] Ещё один пункт
   ```
3. Сохранить описание, кликнув вне поля.
4. Просмотреть отрендеренное описание.

**Ожидаемый результат:**
- Отображается чеклист с тремя пунктами.
- Первый пункт отображается с отмеченным чекбоксом.
- Второй и третий пункты отображаются с пустыми чекбоксами.
