# TC-405: Горячие клавиши: снятие форматирования повторным нажатием

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи. У пользователя есть право `edit.description`. Поле описания содержит текст с курсивом: `Текст *курсивный* конец`.

**Шаги:**
1. Кликнуть на поле описания задачи для перехода в режим редактирования.
2. Выделить слово «курсивный» (включая маркеры `*`).
3. Нажать сочетание клавиш Ctrl+I / Cmd+I.
4. Убедиться, что маркеры `*` удалены.
5. Сохранить описание, кликнув вне поля.

**Ожидаемый результат:**
- Маркеры курсива `*` удаляются с выделенного текста.
- Текст отображается обычным шрифтом: «Текст курсивный конец».
- Кнопка «Курсив» на тулбаре переходит из активного состояния в неактивное.
