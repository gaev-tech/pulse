# TC-398: Горячая клавиша Ctrl+B / Cmd+B — жирный текст

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи. У пользователя есть право `edit.description`. Поле описания содержит текст «Обычный текст для теста».

**Шаги:**
1. Кликнуть на поле описания задачи для перехода в режим редактирования.
2. Выделить слово «текст» в описании.
3. Нажать сочетание клавиш Ctrl+B (Windows/Linux) или Cmd+B (macOS).
4. Сохранить описание, кликнув вне поля.

**Ожидаемый результат:**
- Выделенное слово «текст» оборачивается маркерами `**текст**`.
- После сохранения слово «текст» отображается жирным шрифтом.
- Кнопка «Жирный» на тулбаре подсвечивается как активная при установке курсора на отформатированный текст.
