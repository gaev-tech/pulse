# TC-411: Уменьшение вложенности списка клавишей Shift+Tab

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи. У пользователя есть право `edit.description`. Поле описания содержит вложенный список:
```
- Первый пункт
  - Вложенный пункт
    - Глубоко вложенный пункт
```

**Шаги:**
1. Кликнуть на поле описания задачи для перехода в режим редактирования.
2. Установить курсор в строку «Глубоко вложенный пункт».
3. Нажать Shift+Tab.
4. Убедиться, что строка сдвинулась на один уровень вложенности вверх.
5. Нажать Shift+Tab ещё раз.
6. Убедиться, что строка вышла на корневой уровень списка.
7. Сохранить описание, кликнув вне поля.

**Ожидаемый результат:**
- Первое нажатие Shift+Tab уменьшает вложенность на один уровень (строка выравнивается по уровню «Вложенный пункт»).
- Второе нажатие Shift+Tab выводит строку на корневой уровень списка.
- На корневом уровне Shift+Tab не производит дальнейших изменений.
