# TC-413: Автоматическое продолжение списка при нажатии Enter

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи. У пользователя есть право `edit.description`. Поле описания пустое.

**Шаги:**
1. Кликнуть на поле описания задачи для перехода в режим редактирования.
2. Ввести `- Первый пункт` и нажать Enter.
3. Убедиться, что на новой строке автоматически появился префикс `- `.
4. Ввести «Второй пункт» и нажать Enter.
5. Убедиться, что на новой строке снова появился префикс `- `.
6. Очистить текущую строку и ввести `1. Элемент один` и нажать Enter.
7. Убедиться, что на новой строке автоматически появился префикс `2. `.
8. Сохранить описание, кликнув вне поля.

**Ожидаемый результат:**
- При нажатии Enter в ненумерованном списке новая строка автоматически получает префикс `- `.
- При нажатии Enter в нумерованном списке новая строка получает следующий номер (`2. `, `3. ` и т.д.).
- Автопродолжение работает для чеклистов: новая строка получает префикс `- [ ] `.
