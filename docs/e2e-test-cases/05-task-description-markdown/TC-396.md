# TC-396: Тулбар редактора: применение форматирования без выделения — вставка маркеров в позицию курсора

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи. У пользователя есть право `edit.description`. Поле описания содержит текст «Начало конец».

**Шаги:**
1. Кликнуть на поле описания задачи для перехода в режим редактирования.
2. Установить курсор между словами «Начало» и «конец» (не выделяя текст).
3. Нажать кнопку «Жирный» (иконка **B**) на тулбаре редактора.
4. Ввести текст «важное слово».
5. Повторно нажать кнопку «Жирный» для завершения форматирования.
6. Сохранить описание, кликнув вне поля.

**Ожидаемый результат:**
- При нажатии кнопки без выделения в позицию курсора вставляются парные маркеры `****` с курсором между ними.
- Введённый текст «важное слово» размещается между маркерами: `**важное слово**`.
- После сохранения текст «важное слово» отображается жирным шрифтом.
