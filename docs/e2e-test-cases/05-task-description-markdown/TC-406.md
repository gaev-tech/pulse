# TC-406: Drag-and-drop изображения в редактор описания

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи. У пользователя есть право `edit.description`. Поле описания в режиме редактирования. На рабочем столе подготовлен файл изображения `screenshot.png` (размер < 5 МБ).

**Шаги:**
1. Кликнуть на поле описания задачи для перехода в режим редактирования.
2. Перетащить файл `screenshot.png` с рабочего стола в область редактора описания.
3. Дождаться завершения загрузки (индикатор прогресса).
4. Сохранить описание, кликнув вне поля.

**Ожидаемый результат:**
- Во время перетаскивания область редактора подсвечивается зоной для drop.
- Файл загружается на сервер, в редактор вставляется markdown-разметка изображения: `![screenshot.png](URL_загруженного_файла)`.
- После сохранения изображение отображается в описании задачи.
