# TC-407: Вставка изображения из буфера обмена (Ctrl+V / Cmd+V)

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи. У пользователя есть право `edit.description`. В буфере обмена находится скриншот (сделан через PrintScreen или Cmd+Shift+4).

**Шаги:**
1. Кликнуть на поле описания задачи для перехода в режим редактирования.
2. Установить курсор в нужную позицию в редакторе.
3. Нажать Ctrl+V (Windows/Linux) или Cmd+V (macOS).
4. Дождаться завершения загрузки изображения (индикатор прогресса).
5. Сохранить описание, кликнув вне поля.

**Ожидаемый результат:**
- Изображение из буфера обмена загружается на сервер.
- В редактор вставляется markdown-разметка: `![image](URL_загруженного_файла)`.
- После сохранения изображение отображается в описании задачи в позиции курсора.
