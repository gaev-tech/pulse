# TC-089: Markdown: изображения в описании задачи

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи с пустым описанием. У пользователя есть право `edit.description`.

**Шаги:**
1. Кликнуть на поле описания задачи.
2. Ввести текст: `![Скриншот ошибки](https://example.com/screenshot.png)`.
3. Сохранить описание, кликнув вне поля.
4. Просмотреть отрендеренное описание.

**Ожидаемый результат:**
- В описании отображается изображение, загруженное по URL `https://example.com/screenshot.png`.
- Alt-текст «Скриншот ошибки» используется при недоступности изображения.
