# TC-412: Переключение чекбокса в чеклисте кликом

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи. У пользователя есть право `edit.description`. Поле описания содержит чеклист:
```
- [ ] Первая задача
- [ ] Вторая задача
- [x] Третья задача (выполнена)
```

**Шаги:**
1. Кликнуть на поле описания задачи для перехода в режим редактирования.
2. Кликнуть на чекбокс рядом с «Первая задача».
3. Убедиться, что чекбокс стал отмеченным.
4. Кликнуть на чекбокс рядом с «Третья задача (выполнена)».
5. Убедиться, что чекбокс стал неотмеченным.
6. Сохранить описание, кликнув вне поля.

**Ожидаемый результат:**
- Клик по пустому чекбоксу меняет `- [ ]` на `- [x]`, чекбокс отображается отмеченным.
- Клик по отмеченному чекбоксу меняет `- [x]` на `- [ ]`, чекбокс отображается пустым.
- После сохранения состояния чекбоксов сохраняются.
