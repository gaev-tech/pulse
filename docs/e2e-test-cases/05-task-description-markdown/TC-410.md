# TC-410: Увеличение вложенности списка клавишей Tab

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи. У пользователя есть право `edit.description`. Поле описания содержит ненумерованный список:
```
- Первый пункт
- Второй пункт
- Третий пункт
```

**Шаги:**
1. Кликнуть на поле описания задачи для перехода в режим редактирования.
2. Установить курсор в начало строки «Второй пункт».
3. Нажать клавишу Tab.
4. Убедиться, что строка сдвинулась на один уровень вложенности.
5. Установить курсор в начало строки «Третий пункт».
6. Нажать клавишу Tab дважды.
7. Сохранить описание, кликнув вне поля.

**Ожидаемый результат:**
- «Второй пункт» становится вложенным элементом первого уровня (отступ в 2 или 4 пробела).
- «Третий пункт» становится вложенным элементом второго уровня.
- После сохранения список отображается с визуальной иерархией вложенности.
