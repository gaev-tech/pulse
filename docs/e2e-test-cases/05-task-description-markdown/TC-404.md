# TC-404: Горячие клавиши: применение к выделенному тексту

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи. У пользователя есть право `edit.description`. Поле описания содержит текст «Первое слово второе слово третье слово».

**Шаги:**
1. Кликнуть на поле описания задачи для перехода в режим редактирования.
2. Выделить фразу «второе слово» в описании.
3. Нажать Ctrl+B / Cmd+B для применения жирного форматирования.
4. Не снимая выделение, нажать Ctrl+I / Cmd+I для применения курсива.
5. Сохранить описание, кликнув вне поля.

**Ожидаемый результат:**
- Фраза «второе слово» оборачивается комбинированными маркерами: `***второе слово***`.
- После сохранения фраза отображается одновременно жирным шрифтом и курсивом.
- Текст до и после выделенной фразы остаётся без изменений.
