# TC-094: Markdown: нумерованный список

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи с пустым описанием. У пользователя есть право `edit.description`.

**Шаги:**
1. Кликнуть на поле описания задачи.
2. Ввести текст:
   ```
   1. Первый пункт
   2. Второй пункт
   3. Третий пункт
   ```
3. Сохранить описание, кликнув вне поля.
4. Просмотреть отрендеренное описание.

**Ожидаемый результат:**
- Отображается нумерованный список с тремя пунктами, пронумерованными 1, 2, 3.
- Каждый пункт отображается на отдельной строке с соответствующим номером.
