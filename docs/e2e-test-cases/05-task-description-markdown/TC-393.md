# TC-393: Тулбар редактора: кнопка «Блок кода»

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи. У пользователя есть право `edit.description`. Поле описания пустое.

**Шаги:**
1. Кликнуть на поле описания задачи для перехода в режим редактирования.
2. Нажать кнопку «Блок кода» (иконка с фигурными скобками `{}`) на тулбаре редактора.
3. Ввести многострочный текст:
   ```
   const user = getUser(id);
   console.log(user.name);
   ```
4. Сохранить описание, кликнув вне поля.

**Ожидаемый результат:**
- В редакторе вставляются открывающие и закрывающие маркеры блока кода (` ``` `).
- Введённый текст размещается между маркерами.
- После сохранения код отображается в блоке с моноширинным шрифтом и фоновой заливкой.
