# TC-402: Горячая клавиша Ctrl+Shift+K / Cmd+Shift+K — inline код

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи. У пользователя есть право `edit.description`. Поле описания содержит текст «Вызовите функцию getUser для получения данных».

**Шаги:**
1. Кликнуть на поле описания задачи для перехода в режим редактирования.
2. Выделить слово «getUser» в описании.
3. Нажать сочетание клавиш Ctrl+Shift+K (Windows/Linux) или Cmd+Shift+K (macOS).
4. Сохранить описание, кликнув вне поля.

**Ожидаемый результат:**
- Выделенное слово «getUser» оборачивается обратными кавычками: `` `getUser` ``.
- После сохранения слово «getUser» отображается моноширинным шрифтом с фоновой заливкой.
- Кнопка «Код» на тулбаре подсвечивается как активная при установке курсора на отформатированный текст.
