# TC-392: Тулбар редактора: кнопка «Код» (inline)

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи. У пользователя есть право `edit.description`. Поле описания содержит текст «Используйте функцию getUser для получения данных».

**Шаги:**
1. Кликнуть на поле описания задачи для перехода в режим редактирования.
2. Выделить слово «getUser» в описании.
3. Нажать кнопку «Код» (иконка `<>` или моноширинный шрифт) на тулбаре редактора.
4. Сохранить описание, кликнув вне поля.

**Ожидаемый результат:**
- Выделенное слово «getUser» оборачивается обратными кавычками: `` `getUser` ``.
- После сохранения слово «getUser» отображается моноширинным шрифтом с фоновой заливкой.
- Остальной текст описания не изменён.
