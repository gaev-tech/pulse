# TC-099: Markdown: моноширинный текст (inline code)

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи с пустым описанием. У пользователя есть право `edit.description`.

**Шаги:**
1. Кликнуть на поле описания задачи.
2. Ввести текст: ``Используйте команду `npm install` для установки``.
3. Сохранить описание, кликнув вне поля.
4. Просмотреть отрендеренное описание.

**Ожидаемый результат:**
- Текст «npm install» отображается моноширинным шрифтом с фоновой подсветкой.
- Остальной текст отображается обычным шрифтом.
