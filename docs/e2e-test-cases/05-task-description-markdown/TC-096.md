# TC-096: Markdown: вложенные списки

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи с пустым описанием. У пользователя есть право `edit.description`.

**Шаги:**
1. Кликнуть на поле описания задачи.
2. Ввести текст:
   ```
   - Первый пункт
     - Вложенный пункт 1
     - Вложенный пункт 2
   - Второй пункт
     1. Нумерованный вложенный 1
     2. Нумерованный вложенный 2
   ```
3. Сохранить описание, кликнув вне поля.
4. Просмотреть отрендеренное описание.

**Ожидаемый результат:**
- Отображается список с двумя уровнями вложенности.
- Вложенные пункты отображаются с отступом относительно родительских.
- Нумерованные и ненумерованные вложенные списки рендерятся корректно.
