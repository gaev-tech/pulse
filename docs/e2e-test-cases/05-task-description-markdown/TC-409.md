# TC-409: Вставка текста из буфера обмена с сохранением форматирования

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи. У пользователя есть право `edit.description`. В буфере обмена находится форматированный текст, скопированный из веб-страницы (содержит жирный текст, ссылку и список).

**Шаги:**
1. Кликнуть на поле описания задачи для перехода в режим редактирования.
2. Установить курсор в нужную позицию в редакторе.
3. Нажать Ctrl+V (Windows/Linux) или Cmd+V (macOS).
4. Просмотреть вставленный текст в редакторе.
5. Сохранить описание, кликнув вне поля.

**Ожидаемый результат:**
- Форматированный текст из буфера обмена конвертируется в markdown-разметку.
- Жирный текст оборачивается маркерами `**`, ссылки преобразуются в формат `[текст](url)`, списки получают соответствующие префиксы.
- После сохранения описание отображается с сохранённым форматированием.
