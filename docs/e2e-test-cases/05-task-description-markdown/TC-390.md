# TC-390: Тулбар редактора: кнопка «Ненумерованный список»

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи. У пользователя есть право `edit.description`. Поле описания пустое.

**Шаги:**
1. Кликнуть на поле описания задачи для перехода в режим редактирования.
2. Нажать кнопку «Ненумерованный список» (иконка с буллитами) на тулбаре редактора.
3. Ввести текст «Первый пункт» и нажать Enter.
4. Ввести текст «Второй пункт» и нажать Enter.
5. Ввести текст «Третий пункт».
6. Сохранить описание, кликнув вне поля.

**Ожидаемый результат:**
- Каждая строка получает префикс `- `.
- После сохранения описание отображается как маркированный список с тремя пунктами.
