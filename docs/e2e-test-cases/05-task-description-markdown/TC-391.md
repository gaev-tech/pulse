# TC-391: Тулбар редактора: кнопка «Чеклист»

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи. У пользователя есть право `edit.description`. Поле описания пустое.

**Шаги:**
1. Кликнуть на поле описания задачи для перехода в режим редактирования.
2. Нажать кнопку «Чеклист» (иконка с чекбоксами) на тулбаре редактора.
3. Ввести текст «Подготовить документацию» и нажать Enter.
4. Ввести текст «Провести ревью» и нажать Enter.
5. Ввести текст «Задеплоить на прод».
6. Сохранить описание, кликнув вне поля.

**Ожидаемый результат:**
- Каждая строка получает префикс `- [ ] `.
- После сохранения описание отображается как чеклист с тремя невыполненными пунктами (пустые чекбоксы).
