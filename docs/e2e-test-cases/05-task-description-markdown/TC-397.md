# TC-397: Тулбар редактора: снятие форматирования повторным нажатием кнопки

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи. У пользователя есть право `edit.description`. Поле описания содержит текст с жирным словом: `Текст **жирный** конец`.

**Шаги:**
1. Кликнуть на поле описания задачи для перехода в режим редактирования.
2. Выделить слово «жирный» (включая маркеры `**`).
3. Нажать кнопку «Жирный» (иконка **B**) на тулбаре редактора.
4. Убедиться, что маркеры `**` удалены.
5. Сохранить описание, кликнув вне поля.

**Ожидаемый результат:**
- Маркеры жирного форматирования `**` удаляются с выделенного текста.
- Текст отображается обычным шрифтом: «Текст жирный конец».
- Кнопка «Жирный» на тулбаре переходит из активного состояния в неактивное.
