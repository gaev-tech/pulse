# TC-394: Тулбар редактора: кнопка «Ссылка» — вставка ссылки

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи. У пользователя есть право `edit.description`. Поле описания содержит текст «Смотрите документацию проекта».

**Шаги:**
1. Кликнуть на поле описания задачи для перехода в режим редактирования.
2. Выделить слово «документацию» в описании.
3. Нажать кнопку «Ссылка» (иконка со звеном цепи) на тулбаре редактора.
4. В появившемся поле ввести URL: `https://docs.example.com`.
5. Подтвердить вставку ссылки.
6. Сохранить описание, кликнув вне поля.

**Ожидаемый результат:**
- Выделенный текст оборачивается в markdown-ссылку: `[документацию](https://docs.example.com)`.
- После сохранения слово «документацию» отображается как кликабельная ссылка.
- При клике на ссылку открывается указанный URL.
