# TC-098: Markdown: заголовки 5 уровней

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи с пустым описанием. У пользователя есть право `edit.description`.

**Шаги:**
1. Кликнуть на поле описания задачи.
2. Ввести текст:
   ```
   # Заголовок 1
   ## Заголовок 2
   ### Заголовок 3
   #### Заголовок 4
   ##### Заголовок 5
   ```
3. Сохранить описание, кликнув вне поля.
4. Просмотреть отрендеренное описание.

**Ожидаемый результат:**
- Отображаются пять заголовков с последовательно уменьшающимся размером шрифта.
- Заголовок 1 — самый крупный, заголовок 5 — самый мелкий.
- Каждый заголовок визуально отличается от остальных уровней.
