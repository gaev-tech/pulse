# TC-415: Предпросмотр описания в режиме чтения (переключение редактирование/просмотр)

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи. У пользователя есть право `edit.description`. Поле описания содержит markdown-разметку:
```
# Заголовок
Текст с **жирным** и *курсивом*.
- Пункт списка
- [ ] Чеклист
```

**Шаги:**
1. Кликнуть на поле описания задачи для перехода в режим редактирования.
2. Убедиться, что отображается исходная markdown-разметка и тулбар редактора.
3. Кликнуть вне поля описания для выхода из режима редактирования.
4. Убедиться, что описание переключилось в режим просмотра.
5. Повторно кликнуть на описание для возврата в режим редактирования.

**Ожидаемый результат:**
- В режиме редактирования отображается тулбар и исходная markdown-разметка с подсветкой синтаксиса.
- В режиме просмотра markdown-разметка рендерится: заголовок отображается крупным шрифтом, текст форматируется жирным и курсивом, список отображается с буллитами, чеклист — с чекбоксами.
- Переключение между режимами происходит без потери данных.
- Тулбар редактора скрывается в режиме просмотра.
