# TC-408: Drag-and-drop файла в редактор описания

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи. У пользователя есть право `edit.description`. Поле описания в режиме редактирования. На рабочем столе подготовлен файл `report.pdf` (размер < 10 МБ).

**Шаги:**
1. Кликнуть на поле описания задачи для перехода в режим редактирования.
2. Перетащить файл `report.pdf` с рабочего стола в область редактора описания.
3. Дождаться завершения загрузки (индикатор прогресса).
4. Сохранить описание, кликнув вне поля.

**Ожидаемый результат:**
- Во время перетаскивания область редактора подсвечивается зоной для drop.
- Файл загружается на сервер, в редактор вставляется markdown-ссылка на файл: `[report.pdf](URL_загруженного_файла)`.
- После сохранения имя файла отображается как кликабельная ссылка для скачивания.
