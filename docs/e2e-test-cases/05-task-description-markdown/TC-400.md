# TC-400: Горячая клавиша Ctrl+U / Cmd+U — подчёркивание

**Предусловие:** Пользователь авторизован, открыта Task Screen задачи. У пользователя есть право `edit.description`. Поле описания содержит текст «Обычный текст для теста».

**Шаги:**
1. Кликнуть на поле описания задачи для перехода в режим редактирования.
2. Выделить слово «текст» в описании.
3. Нажать сочетание клавиш Ctrl+U (Windows/Linux) или Cmd+U (macOS).
4. Сохранить описание, кликнув вне поля.

**Ожидаемый результат:**
- Выделенное слово «текст» оборачивается маркерами `<u>текст</u>`.
- После сохранения слово «текст» отображается с подчёркиванием.
- Кнопка «Подчёркивание» на тулбаре подсвечивается как активная при установке курсора на отформатированный текст.
