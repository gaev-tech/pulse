# TC-160: Открытие модалки личных меток

**Предусловие:** Пользователь авторизован и находится на любом экране с Navigation Sidebar.

**Шаги:**
1. В нижней части Navigation Sidebar нажать кнопку открытия модалки личных меток.

**Ожидаемый результат:**
- Открывается модальное окно управления личными метками.
- Отображается список существующих личных меток пользователя (если они есть).
- Доступна возможность создания новой метки.
