# TC-170: Применение личной метки к личной задаче

**Предусловие:** Пользователь авторизован. Существует личная метка "Срочно". Существует личная задача #1 без меток.

**Шаги:**
1. Открыть задачу #1 (Task Screen или Task Modal).
2. В правой панели найти поле "Метки".
3. Нажать на поле для добавления метки.
4. Выбрать метку "Срочно" из списка доступных меток.
5. Подтвердить выбор.

**Ожидаемый результат:**
- Метка "Срочно" успешно применена к задаче #1.
- Метка отображается в поле "Метки" задачи.
- В ленте событий задачи появилось событие `task.labels_changed`.
