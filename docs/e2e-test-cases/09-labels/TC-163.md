# TC-163: Распространение переименования метки на все задачи

**Предусловие:** Пользователь авторизован. Существует личная метка "Срочно", применённая к личным задачам #1 и #2.

**Шаги:**
1. Открыть модалку личных меток.
2. Переименовать метку "Срочно" в "Критично".
3. Сохранить изменения.
4. Открыть задачу #1.
5. Проверить метки задачи.
6. Открыть задачу #2.
7. Проверить метки задачи.

**Ожидаемый результат:**
- В задаче #1 метка отображается с новым названием "Критично".
- В задаче #2 метка отображается с новым названием "Критично".
- Переименование применилось ко всем задачам, где была метка "Срочно".
