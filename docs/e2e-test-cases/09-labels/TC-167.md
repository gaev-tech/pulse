# TC-167: Создание командной метки (требует team.manage_labels)

**Предусловие:** Пользователь авторизован и является участником команды "Alpha Team" с правом `team.manage_labels`.

**Шаги:**
1. Открыть управление метками команды "Alpha Team".
2. Нажать кнопку создания новой командной метки.
3. Ввести название "Баг".
4. Выбрать оранжевый цвет.
5. Подтвердить создание метки.

**Ожидаемый результат:**
- Командная метка "Баг" с оранжевым цветом успешно создана.
- Метка отображается в списке командных меток "Alpha Team".
- Метка доступна для применения к задачам команды "Alpha Team".
