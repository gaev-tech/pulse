# TC-165: Удаление личной метки — Confirmation Popup

**Предусловие:** Пользователь авторизован. Открыта модалка личных меток. Существует личная метка "Критично".

**Шаги:**
1. Найти метку "Критично" в списке личных меток.
2. Нажать кнопку удаления метки.

**Ожидаемый результат:**
- Появляется Confirmation Popup с текстом подтверждения удаления метки.
- Popup содержит красную кнопку "Да, подтверждаю" и кнопку "Отмена".
- Метка ещё не удалена.
