# TC-172: Недоступность управления командными метками без права team.manage_labels

**Предусловие:** Пользователь авторизован и является участником команды "Alpha Team" без права `team.manage_labels` (есть только `view` и `task.create`).

**Шаги:**
1. Перейти к управлению метками команды "Alpha Team".
2. Попытаться создать новую командную метку.
3. Попытаться удалить существующую командную метку.

**Ожидаемый результат:**
- Кнопка создания новой командной метки недоступна или отсутствует.
- Кнопка удаления существующих командных меток недоступна или отсутствует.
- Пользователь не может создавать или удалять командные метки без права `team.manage_labels`.
