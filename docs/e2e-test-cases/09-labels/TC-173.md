# TC-173: Недоступность редактирования командных меток без права team.edit_labels

**Предусловие:** Пользователь авторизован и является участником команды "Alpha Team" без права `team.edit_labels` (есть только `view` и `task.create`). В команде существует метка "Баг".

**Шаги:**
1. Перейти к управлению метками команды "Alpha Team".
2. Попытаться переименовать метку "Баг".

**Ожидаемый результат:**
- Возможность переименования метки недоступна.
- Пользователь не может редактировать командные метки без права `team.edit_labels`.
