# TC-166: Удаление метки — удаление из всех задач

**Предусловие:** Пользователь авторизован. Существует личная метка "Критично", применённая к задачам #1 и #2. Открыта модалка личных меток. Нажата кнопка удаления метки "Критично". Отображается Confirmation Popup.

**Шаги:**
1. В Confirmation Popup нажать красную кнопку "Да, подтверждаю".
2. Открыть задачу #1 и проверить метки.
3. Открыть задачу #2 и проверить метки.

**Ожидаемый результат:**
- Метка "Критично" удалена из списка личных меток.
- В задаче #1 метка "Критично" отсутствует.
- В задаче #2 метка "Критично" отсутствует.
- Метка удалена из всех задач, к которым была применена.
