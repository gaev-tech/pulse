# TC-171: Применение командной метки к командной задаче

**Предусловие:** Пользователь авторизован и является участником команды "Alpha Team" с правом `edit.labels`. В команде существует метка "Баг". Существует командная задача ALPHA-1 без меток.

**Шаги:**
1. Открыть задачу ALPHA-1 (Task Screen или Task Modal).
2. В правой панели найти поле "Метки".
3. Нажать на поле для добавления метки.
4. Выбрать командную метку "Баг" из списка доступных меток.
5. Подтвердить выбор.

**Ожидаемый результат:**
- Командная метка "Баг" успешно применена к задаче ALPHA-1.
- Метка отображается в поле "Метки" задачи.
- В ленте событий задачи появилось событие `task.labels_changed`.
