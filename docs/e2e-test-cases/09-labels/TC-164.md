# TC-164: Событие в каждой задаче при переименовании метки

**Предусловие:** Пользователь авторизован. Существует личная метка "Срочно", применённая к личным задачам #1 и #2.

**Шаги:**
1. Открыть модалку личных меток.
2. Переименовать метку "Срочно" в "Критично".
3. Сохранить изменения.
4. Открыть задачу #1 и просмотреть ленту событий.
5. Открыть задачу #2 и просмотреть ленту событий.

**Ожидаемый результат:**
- В ленте событий задачи #1 появилось событие `task.labels_changed` о переименовании метки.
- В ленте событий задачи #2 появилось событие `task.labels_changed` о переименовании метки.
- События зафиксированы в каждой задаче, где была применена переименованная метка.
