# TC-162: Переименование личной метки

**Предусловие:** Пользователь авторизован. Открыта модалка личных меток. Существует личная метка "Срочно".

**Шаги:**
1. Найти метку "Срочно" в списке личных меток.
2. Нажать на метку для редактирования.
3. Изменить название с "Срочно" на "Критично".
4. Сохранить изменения.

**Ожидаемый результат:**
- Метка переименована с "Срочно" на "Критично".
- В списке личных меток отображается обновлённое название "Критично".
