# TC-161: Создание личной метки (название + цвет)

**Предусловие:** Пользователь авторизован. Открыта модалка личных меток.

**Шаги:**
1. Нажать кнопку создания новой метки.
2. Ввести название метки "Срочно".
3. Выбрать красный цвет для метки.
4. Подтвердить создание метки.

**Ожидаемый результат:**
- Личная метка "Срочно" с красным цветом успешно создана.
- Метка отображается в списке личных меток.
- Метка доступна для применения к личным задачам.
