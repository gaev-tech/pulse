# TC-169: Удаление командной метки

**Предусловие:** Пользователь авторизован и является участником команды "Alpha Team" с правом `team.manage_labels`. В команде существует метка "Дефект", применённая к задаче ALPHA-1.

**Шаги:**
1. Открыть управление метками команды "Alpha Team".
2. Найти метку "Дефект" в списке командных меток.
3. Нажать кнопку удаления метки.
4. В Confirmation Popup нажать красную кнопку "Да, подтверждаю".

**Ожидаемый результат:**
- Командная метка "Дефект" удалена из списка меток команды.
- В задаче ALPHA-1 метка "Дефект" отсутствует.
- Метка удалена из всех командных задач.
