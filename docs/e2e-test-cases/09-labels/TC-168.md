# TC-168: Переименование командной метки (требует team.edit_labels)

**Предусловие:** Пользователь авторизован и является участником команды "Alpha Team" с правом `team.edit_labels`. В команде существует метка "Баг".

**Шаги:**
1. Открыть управление метками команды "Alpha Team".
2. Найти метку "Баг" в списке командных меток.
3. Нажать на метку для редактирования.
4. Изменить название с "Баг" на "Дефект".
5. Сохранить изменения.

**Ожидаемый результат:**
- Командная метка переименована с "Баг" на "Дефект".
- Во всех задачах команды "Alpha Team", где была метка "Баг", отображается новое название "Дефект".
