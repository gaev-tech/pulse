# TC-226: Отображение строки критериев

**Предусловие:** пользователь авторизован, создан личный фильтр в простом режиме с критериями: статус `opened`, выбрана метка «Баг».

**Шаги:**
1. Открыть Filter Results Screen созданного фильтра через Navigation Sidebar.
2. Осмотреть строку критериев под названием фильтра.

**Ожидаемый результат:**
- Строка критериев отображается под названием фильтра.
- В строке присутствуют: инпут поиска (по ключу и названию), тег ответственного, тег статуса (установлен `opened`), тег меток (выбрана «Баг»), тег команды.
- Установленные критерии визуально выделены (статус `opened` и метка «Баг» отображаются как активные теги).
