# TC-249: Shift+клик по заголовку первичной -- смена направления без сброса вторичной

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Установлена первичная сортировка по колонке «Дата создания» (по возрастанию) и вторичная сортировка по колонке «Название».

**Шаги:**
1. Зажать клавишу Shift и нажать на заголовок колонки «Дата создания» (первичная сортировка).
2. Осмотреть заголовки колонок и таблицу результатов.

**Ожидаемый результат:**
- Направление первичной сортировки по колонке «Дата создания» меняется на противоположное (по убыванию).
- Вторичная сортировка по колонке «Название» сохраняется без изменений (стрелка с цифрой «2» остаётся).
- Задачи пересортированы с учётом нового направления первичной сортировки.
