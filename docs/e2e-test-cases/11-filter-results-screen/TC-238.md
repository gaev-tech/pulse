# TC-238: Отображение колонки "Наличие вложений"

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Колонка «Наличие вложений» включена в настройках таблицы. Среди результатов есть задача с вложениями и задача без вложений.

**Шаги:**
1. Осмотреть заголовки таблицы — убедиться в наличии колонки «Наличие вложений».
2. Найти строки задач с вложениями и без вложений в таблице.

**Ожидаемый результат:**
- В таблице отображается колонка «Наличие вложений».
- Для задачи с вложениями в колонке отображается индикатор наличия.
- Для задачи без вложений в колонке отображается индикатор отсутствия.
