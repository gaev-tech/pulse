# TC-229: Сохранение изменённых критериев фильтра

**Предусловие:** пользователь авторизован, открыт Filter Results Screen личного фильтра в простом режиме. Критерии были изменены (статус с `opened` на `closed`), кнопка «Сохранить фильтр» отображается.

**Шаги:**
1. Нажать кнопку «Сохранить фильтр».
2. Дождаться сохранения.
3. Перейти на другой экран (например, Personal Activity Screen).
4. Вернуться на Filter Results Screen того же фильтра через Navigation Sidebar.

**Ожидаемый результат:**
- После шага 1 кнопка «Сохранить фильтр» исчезает.
- После шага 4 фильтр отображает сохранённые критерии: статус `closed`.
- Результаты таблицы соответствуют сохранённым критериям.
