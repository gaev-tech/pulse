# TC-236: Отображение колонки "Статус"

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Колонка «Статус» включена в настройках таблицы. Среди результатов есть задачи со статусами `opened` и `closed`.

**Шаги:**
1. Осмотреть заголовки таблицы — убедиться в наличии колонки «Статус».
2. Найти строки задач с разными статусами в таблице.

**Ожидаемый результат:**
- В таблице отображается колонка «Статус».
- В строках задач в колонке «Статус» отображаются значения `opened` или `closed` соответственно.
