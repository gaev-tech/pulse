# TC-259: Сортировка по ответственному -- А->Я / Я->А

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Колонка «Ответственный» включена. Среди результатов есть задачи с ответственными «Алексей», «Борис», «Дмитрий».

**Шаги:**
1. Нажать на заголовок колонки «Ответственный» для сортировки А->Я.
2. Запомнить порядок задач в таблице.
3. Нажать повторно на заголовок колонки «Ответственный» для сортировки Я->А.
4. Запомнить порядок задач в таблице.

**Ожидаемый результат:**
- После шага 2 задачи отсортированы по имени ответственного по алфавиту: «Алексей», «Борис», «Дмитрий».
- После шага 4 задачи отсортированы по имени ответственного в обратном порядке: «Дмитрий», «Борис», «Алексей».
