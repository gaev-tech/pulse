# TC-267: Колонка "Метки" не поддерживает сортировку

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Колонка «Метки» включена в настройках таблицы.

**Шаги:**
1. Нажать на заголовок колонки «Метки».
2. Осмотреть заголовок колонки и таблицу результатов.

**Ожидаемый результат:**
- Клик по заголовку колонки «Метки» не устанавливает сортировку.
- В заголовке колонки «Метки» не появляется стрелка направления сортировки.
- Порядок задач в таблице не изменяется.
