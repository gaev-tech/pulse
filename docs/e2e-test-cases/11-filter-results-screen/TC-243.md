# TC-243: Отображение колонки "Дата изменения"

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Колонка «Дата изменения» включена в настройках таблицы. Среди результатов есть задачи, изменённые в разное время.

**Шаги:**
1. Осмотреть заголовки таблицы — убедиться в наличии колонки «Дата изменения».
2. Найти строки задач и проверить значения в колонке «Дата изменения».

**Ожидаемый результат:**
- В таблице отображается колонка «Дата изменения».
- В строках задач в колонке отображается дата и время последнего изменения каждой задачи.
