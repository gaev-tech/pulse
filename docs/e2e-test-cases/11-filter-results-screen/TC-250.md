# TC-250: Сброс вторичной сортировки при установке новой первичной

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Установлена первичная сортировка по колонке «Дата создания» и вторичная по колонке «Название». Колонка «Ответственный» отображается.

**Шаги:**
1. Нажать (без Shift) на заголовок колонки «Ответственный».
2. Осмотреть заголовки колонок и таблицу результатов.

**Ожидаемый результат:**
- Первичная сортировка переключается на колонку «Ответственный» (стрелка в заголовке).
- Вторичная сортировка по колонке «Название» сбрасывается (стрелка с цифрой «2» исчезает).
- Предыдущая первичная сортировка по «Дата создания» сбрасывается.
- Задачи отсортированы только по ответственному.
