# TC-225: Редактирование названия фильтра по клику

**Предусловие:** пользователь авторизован, открыт Filter Results Screen личного фильтра с названием «Мои задачи».

**Шаги:**
1. Нажать на название фильтра «Мои задачи» в верхней панели экрана.
2. Очистить текущее название и ввести новое: «Активные задачи».
3. Подтвердить изменение (нажать Enter или кликнуть вне поля ввода).

**Ожидаемый результат:**
- После шага 1 название фильтра становится редактируемым полем ввода.
- После шага 3 название фильтра обновляется на «Активные задачи».
- В Navigation Sidebar в разделе «Личные фильтры» отображается обновлённое название «Активные задачи».
