# TC-253: Сортировка по ключу -- по возрастанию/убыванию номера

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Среди результатов есть задачи с ключами `ACME-1`, `ACME-5`, `ACME-12`, `ACME-3`.

**Шаги:**
1. Нажать на заголовок колонки «Ключ» для сортировки по возрастанию.
2. Запомнить порядок задач в таблице.
3. Нажать повторно на заголовок колонки «Ключ» для сортировки по убыванию.
4. Запомнить порядок задач в таблице.

**Ожидаемый результат:**
- После шага 2 задачи отсортированы по возрастанию номера: `ACME-1`, `ACME-3`, `ACME-5`, `ACME-12`.
- После шага 4 задачи отсортированы по убыванию номера: `ACME-12`, `ACME-5`, `ACME-3`, `ACME-1`.
