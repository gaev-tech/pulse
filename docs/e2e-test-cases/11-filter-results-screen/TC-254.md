# TC-254: Сортировка по названию -- А->Я / Я->А

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Среди результатов есть задачи с названиями «Авторизация», «Баги», «Деплой», «Рефакторинг».

**Шаги:**
1. Нажать на заголовок колонки «Название» для сортировки А->Я.
2. Запомнить порядок задач в таблице.
3. Нажать повторно на заголовок колонки «Название» для сортировки Я->А.
4. Запомнить порядок задач в таблице.

**Ожидаемый результат:**
- После шага 2 задачи отсортированы по алфавиту: «Авторизация», «Баги», «Деплой», «Рефакторинг».
- После шага 4 задачи отсортированы в обратном алфавитном порядке: «Рефакторинг», «Деплой», «Баги», «Авторизация».
