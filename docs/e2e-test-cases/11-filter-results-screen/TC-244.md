# TC-244: Отображение колонки "Автор создания"

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Колонка «Автор создания» включена в настройках таблицы. Среди результатов есть задачи, созданные разными пользователями.

**Шаги:**
1. Осмотреть заголовки таблицы — убедиться в наличии колонки «Автор создания».
2. Найти строки задач и проверить значения в колонке «Автор создания».

**Ожидаемый результат:**
- В таблице отображается колонка «Автор создания».
- В строках задач в колонке отображается имя пользователя, создавшего задачу.
