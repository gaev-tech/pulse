# TC-246: Первичная сортировка по клику на заголовок колонки

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра с результатами. Колонка «Дата создания» включена в настройках таблицы. Сортировка не установлена.

**Шаги:**
1. Нажать на заголовок колонки «Дата создания».
2. Осмотреть таблицу результатов и заголовок колонки.

**Ожидаемый результат:**
- В заголовке колонки «Дата создания» появляется стрелка направления сортировки.
- Задачи в таблице отсортированы по дате создания.
- Никакая другая колонка не имеет индикатора сортировки.
