# TC-235: Отображение колонки "Ответственный"

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Колонка «Ответственный» включена в настройках таблицы. Среди результатов есть задача с назначенным ответственным «Иван Петров».

**Шаги:**
1. Осмотреть заголовки таблицы — убедиться в наличии колонки «Ответственный».
2. Найти строку задачи с ответственным в таблице.

**Ожидаемый результат:**
- В таблице отображается колонка «Ответственный».
- В строке задачи в колонке «Ответственный» отображается имя «Иван Петров».
