# TC-263: Сортировка по наличию блокеров -- сначала есть/сначала нет

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Колонка «Наличие блокеров» включена. Среди результатов есть задачи с блокерами и без блокеров.

**Шаги:**
1. Нажать на заголовок колонки «Наличие блокеров» для сортировки «сначала есть».
2. Запомнить порядок задач в таблице.
3. Нажать повторно на заголовок колонки «Наличие блокеров» для сортировки «сначала нет».
4. Запомнить порядок задач в таблице.

**Ожидаемый результат:**
- После шага 2 задачи с блокерами отображаются выше задач без блокеров.
- После шага 4 задачи без блокеров отображаются выше задач с блокерами.
