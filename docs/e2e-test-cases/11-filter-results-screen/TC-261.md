# TC-261: Сортировка по родительской задаче -- А->Я / Я->А

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Колонка «Родительская задача» включена. Среди результатов есть задачи с родительскими задачами «Авторизация», «Деплой», «Тестирование».

**Шаги:**
1. Нажать на заголовок колонки «Родительская задача» для сортировки А->Я.
2. Запомнить порядок задач в таблице.
3. Нажать повторно на заголовок колонки «Родительская задача» для сортировки Я->А.
4. Запомнить порядок задач в таблице.

**Ожидаемый результат:**
- После шага 2 задачи отсортированы по названию родительской задачи по алфавиту: «Авторизация», «Деплой», «Тестирование».
- После шага 4 задачи отсортированы в обратном порядке: «Тестирование», «Деплой», «Авторизация».
