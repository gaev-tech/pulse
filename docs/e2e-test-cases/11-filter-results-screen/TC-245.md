# TC-245: Отображение колонки "Автор изменения"

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Колонка «Автор изменения» включена в настройках таблицы. Среди результатов есть задачи, последний раз изменённые разными пользователями.

**Шаги:**
1. Осмотреть заголовки таблицы — убедиться в наличии колонки «Автор изменения».
2. Найти строки задач и проверить значения в колонке «Автор изменения».

**Ожидаемый результат:**
- В таблице отображается колонка «Автор изменения».
- В строках задач в колонке отображается имя пользователя, последним изменившего задачу.
