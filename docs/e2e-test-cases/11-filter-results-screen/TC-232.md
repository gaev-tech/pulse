# TC-232: Обязательные колонки таблицы: чекбокс, стрелка, ключ, название

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра с результатами. Среди результатов есть задача с ключом `ACME-5` и названием «Настроить CI/CD».

**Шаги:**
1. Осмотреть заголовки таблицы результатов.
2. Осмотреть строку задачи `ACME-5` в таблице.

**Ожидаемый результат:**
- Первые четыре колонки таблицы (слева направо): чекбокс выбора, стрелка раскрытия иерархии, ключ задачи, название задачи.
- Порядок обязательных колонок фиксирован и не может быть изменён.
- В строке задачи отображается: чекбокс, стрелка, ключ `ACME-5`, название «Настроить CI/CD».
