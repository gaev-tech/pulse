# TC-237: Отображение колонки "Родительская задача"

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Колонка «Родительская задача» включена в настройках таблицы. Среди результатов есть задача с родительской задачей `ACME-1` («Эпик: Авторизация»).

**Шаги:**
1. Осмотреть заголовки таблицы — убедиться в наличии колонки «Родительская задача».
2. Найти строку задачи с родительской задачей в таблице.

**Ожидаемый результат:**
- В таблице отображается колонка «Родительская задача».
- В строке задачи в колонке «Родительская задача» отображается информация о родительской задаче.
