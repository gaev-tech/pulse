# TC-262: Сортировка по наличию вложений -- сначала есть/сначала нет

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Колонка «Наличие вложений» включена. Среди результатов есть задачи с вложениями и без вложений.

**Шаги:**
1. Нажать на заголовок колонки «Наличие вложений» для сортировки «сначала есть».
2. Запомнить порядок задач в таблице.
3. Нажать повторно на заголовок колонки «Наличие вложений» для сортировки «сначала нет».
4. Запомнить порядок задач в таблице.

**Ожидаемый результат:**
- После шага 2 задачи с вложениями отображаются выше задач без вложений.
- После шага 4 задачи без вложений отображаются выше задач с вложениями.
