# TC-247: Смена направления первичной сортировки повторным кликом

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Установлена первичная сортировка по колонке «Дата создания» по возрастанию (стрелка вверх).

**Шаги:**
1. Нажать повторно на заголовок колонки «Дата создания».
2. Осмотреть таблицу результатов и заголовок колонки.

**Ожидаемый результат:**
- Направление стрелки в заголовке колонки «Дата создания» меняется на противоположное (стрелка вниз).
- Задачи в таблице отсортированы по дате создания в обратном порядке (по убыванию).
