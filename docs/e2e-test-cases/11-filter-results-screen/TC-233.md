# TC-233: Отображение необязательной колонки "Метки"

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Колонка «Метки» включена в настройках таблицы (Table Settings Popup). Среди результатов есть задача с метками «Баг» и «Срочно».

**Шаги:**
1. Осмотреть заголовки таблицы — убедиться в наличии колонки «Метки».
2. Найти строку задачи с метками в таблице.

**Ожидаемый результат:**
- В таблице отображается колонка «Метки» после обязательных колонок.
- В строке задачи в колонке «Метки» отображаются метки «Баг» и «Срочно».
