# TC-258: Сортировка по автору изменения -- А->Я / Я->А

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Колонка «Автор изменения» включена. Среди результатов есть задачи, последний раз изменённые пользователями «Алексей», «Борис», «Дмитрий».

**Шаги:**
1. Нажать на заголовок колонки «Автор изменения» для сортировки А->Я.
2. Запомнить порядок задач в таблице.
3. Нажать повторно на заголовок колонки «Автор изменения» для сортировки Я->А.
4. Запомнить порядок задач в таблице.

**Ожидаемый результат:**
- После шага 2 задачи отсортированы по имени автора изменения по алфавиту: «Алексей», «Борис», «Дмитрий».
- После шага 4 задачи отсортированы по имени автора изменения в обратном порядке: «Дмитрий», «Борис», «Алексей».
