# TC-231: Кнопка меню "..." -- удаление фильтра с подтверждением

**Предусловие:** пользователь авторизован, открыт Filter Results Screen личного фильтра с названием «Тестовый фильтр».

**Шаги:**
1. Нажать кнопку «...» в верхней панели Filter Results Screen.
2. В открывшемся Filter Popup нажать на пункт удаления фильтра.
3. В Confirmation Popup нажать кнопку «Да, подтверждаю».

**Ожидаемый результат:**
- После шага 2 появляется Confirmation Popup с текстом подтверждения удаления фильтра.
- В Confirmation Popup отображается красная кнопка «Да, подтверждаю» и кнопка «Отмена».
- После шага 3 фильтр удаляется, пользователь перенаправляется с Filter Results Screen.
- В Navigation Sidebar в разделе «Личные фильтры» фильтр «Тестовый фильтр» больше не отображается.
