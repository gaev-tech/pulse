# TC-266: Сортировка по статусу -- сначала opened/сначала closed

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Колонка «Статус» включена. Среди результатов есть задачи со статусами `opened` и `closed`.

**Шаги:**
1. Нажать на заголовок колонки «Статус» для сортировки «сначала opened».
2. Запомнить порядок задач в таблице.
3. Нажать повторно на заголовок колонки «Статус» для сортировки «сначала closed».
4. Запомнить порядок задач в таблице.

**Ожидаемый результат:**
- После шага 2 задачи со статусом `opened` отображаются выше задач со статусом `closed`.
- После шага 4 задачи со статусом `closed` отображаются выше задач со статусом `opened`.
