# TC-234: Отображение колонки "Команда" только в личных фильтрах

**Предусловие:** пользователь авторизован, существует личный фильтр и командный фильтр. Колонка «Команда» включена в настройках таблицы обоих фильтров.

**Шаги:**
1. Открыть Filter Results Screen личного фильтра через Navigation Sidebar.
2. Осмотреть заголовки таблицы — проверить наличие колонки «Команда».
3. Открыть Filter Results Screen командного фильтра через Navigation Sidebar.
4. Осмотреть заголовки таблицы — проверить наличие колонки «Команда».

**Ожидаемый результат:**
- В личном фильтре (шаг 2) колонка «Команда» отображается в таблице.
- В командном фильтре (шаг 4) колонка «Команда» не отображается.
