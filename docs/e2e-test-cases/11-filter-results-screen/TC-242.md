# TC-242: Отображение колонки "Дата создания"

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Колонка «Дата создания» включена в настройках таблицы. Среди результатов есть задачи, созданные в разное время.

**Шаги:**
1. Осмотреть заголовки таблицы — убедиться в наличии колонки «Дата создания».
2. Найти строки задач и проверить значения в колонке «Дата создания».

**Ожидаемый результат:**
- В таблице отображается колонка «Дата создания».
- В строках задач в колонке отображается дата и время создания каждой задачи.
