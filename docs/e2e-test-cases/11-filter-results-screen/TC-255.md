# TC-255: Сортировка по дате создания -- по возрастанию/убыванию

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Колонка «Дата создания» включена. Среди результатов есть задачи, созданные в разное время.

**Шаги:**
1. Нажать на заголовок колонки «Дата создания» для сортировки по возрастанию.
2. Запомнить порядок задач в таблице.
3. Нажать повторно на заголовок колонки «Дата создания» для сортировки по убыванию.
4. Запомнить порядок задач в таблице.

**Ожидаемый результат:**
- После шага 2 задачи отсортированы от самой старой к самой новой по дате создания.
- После шага 4 задачи отсортированы от самой новой к самой старой по дате создания.
