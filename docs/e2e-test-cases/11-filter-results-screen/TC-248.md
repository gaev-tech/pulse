# TC-248: Вторичная сортировка по Shift+клик на другой заголовок

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Установлена первичная сортировка по колонке «Дата создания». Колонка «Название» отображается.

**Шаги:**
1. Зажать клавишу Shift и нажать на заголовок колонки «Название».
2. Осмотреть заголовки колонок и таблицу результатов.

**Ожидаемый результат:**
- В заголовке колонки «Дата создания» сохраняется стрелка первичной сортировки.
- В заголовке колонки «Название» появляется стрелка с цифрой «2», обозначающая вторичную сортировку.
- Задачи отсортированы: сначала по дате создания, при равенстве — по названию.
