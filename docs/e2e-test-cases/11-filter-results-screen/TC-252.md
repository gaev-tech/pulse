# TC-252: Отображение стрелки с цифрой "2" для вторичной сортировки

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Установлена первичная сортировка по колонке «Ключ». Колонка «Название» отображается.

**Шаги:**
1. Зажать клавишу Shift и нажать на заголовок колонки «Название».
2. Осмотреть заголовок колонки «Название».

**Ожидаемый результат:**
- В заголовке колонки «Название» отображается стрелка направления сортировки с цифрой «2».
- Цифра «2» визуально отличает вторичную сортировку от первичной.
- В заголовке колонки «Ключ» сохраняется обычная стрелка первичной сортировки (без цифры).
