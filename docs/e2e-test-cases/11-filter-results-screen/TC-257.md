# TC-257: Сортировка по автору создания -- А->Я / Я->А

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Колонка «Автор создания» включена. Среди результатов есть задачи, созданные пользователями «Алексей», «Борис», «Дмитрий».

**Шаги:**
1. Нажать на заголовок колонки «Автор создания» для сортировки А->Я.
2. Запомнить порядок задач в таблице.
3. Нажать повторно на заголовок колонки «Автор создания» для сортировки Я->А.
4. Запомнить порядок задач в таблице.

**Ожидаемый результат:**
- После шага 2 задачи отсортированы по имени автора создания по алфавиту: «Алексей», «Борис», «Дмитрий».
- После шага 4 задачи отсортированы по имени автора создания в обратном порядке: «Дмитрий», «Борис», «Алексей».
