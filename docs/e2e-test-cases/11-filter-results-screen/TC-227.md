# TC-227: Изменение критериев -- немедленное обновление результатов

**Предусловие:** пользователь авторизован, открыт Filter Results Screen личного фильтра в простом режиме с критерием статуса `opened`. В таблице отображаются задачи.

**Шаги:**
1. В строке критериев изменить статус с `opened` на `closed`.
2. Наблюдать за таблицей результатов.

**Ожидаемый результат:**
- Таблица результатов обновляется немедленно после изменения критерия.
- В таблице отображаются только задачи со статусом `closed`.
- Задачи со статусом `opened` больше не отображаются.
