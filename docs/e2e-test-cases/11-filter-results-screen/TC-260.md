# TC-260: Сортировка по команде -- А->Я / Я->А

**Предусловие:** пользователь авторизован, открыт Filter Results Screen личного фильтра. Колонка «Команда» включена. Среди результатов есть задачи команд «ACME», «Backend», «Frontend».

**Шаги:**
1. Нажать на заголовок колонки «Команда» для сортировки А->Я.
2. Запомнить порядок задач в таблице.
3. Нажать повторно на заголовок колонки «Команда» для сортировки Я->А.
4. Запомнить порядок задач в таблице.

**Ожидаемый результат:**
- После шага 2 задачи отсортированы по названию команды по алфавиту: «ACME», «Backend», «Frontend».
- После шага 4 задачи отсортированы по названию команды в обратном порядке: «Frontend», «Backend», «ACME».
