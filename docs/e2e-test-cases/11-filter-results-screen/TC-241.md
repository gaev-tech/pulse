# TC-241: Отображение колонки "Наличие ссылок"

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Колонка «Наличие ссылок» включена в настройках таблицы. Среди результатов есть задача с внешними ссылками и задача без ссылок.

**Шаги:**
1. Осмотреть заголовки таблицы — убедиться в наличии колонки «Наличие ссылок».
2. Найти строки задач со ссылками и без ссылок в таблице.

**Ожидаемый результат:**
- В таблице отображается колонка «Наличие ссылок».
- Для задачи с внешними ссылками в колонке отображается индикатор наличия.
- Для задачи без ссылок в колонке отображается индикатор отсутствия.
