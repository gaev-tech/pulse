# TC-268: Сортировка доступна только для отображаемых колонок

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. В настройках таблицы включены колонки «Ответственный» и «Дата создания». Колонка «Дата изменения» выключена.

**Шаги:**
1. Нажать на заголовок колонки «Ответственный» — установить первичную сортировку.
2. Зажать Shift и нажать на заголовок колонки «Дата создания» — установить вторичную сортировку.
3. Открыть Table Settings Popup и отключить колонку «Ответственный».
4. Закрыть Table Settings Popup и осмотреть таблицу.

**Ожидаемый результат:**
- После шага 2 в таблице установлена первичная сортировка по «Ответственный» и вторичная по «Дата создания».
- После шага 4 колонка «Ответственный» не отображается в таблице.
- Сортировка по скрытой колонке «Ответственный» сброшена.
- Колонка «Дата изменения» не отображается в таблице и недоступна для сортировки.
