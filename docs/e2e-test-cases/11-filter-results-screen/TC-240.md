# TC-240: Отображение колонки "Наличие связей"

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Колонка «Наличие связей» включена в настройках таблицы. Среди результатов есть задача со связанными задачами и задача без связей.

**Шаги:**
1. Осмотреть заголовки таблицы — убедиться в наличии колонки «Наличие связей».
2. Найти строки задач со связями и без связей в таблице.

**Ожидаемый результат:**
- В таблице отображается колонка «Наличие связей».
- Для задачи со связями в колонке отображается индикатор наличия.
- Для задачи без связей в колонке отображается индикатор отсутствия.
