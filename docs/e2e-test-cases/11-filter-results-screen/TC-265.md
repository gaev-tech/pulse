# TC-265: Сортировка по наличию ссылок -- сначала есть/сначала нет

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Колонка «Наличие ссылок» включена. Среди результатов есть задачи с внешними ссылками и без ссылок.

**Шаги:**
1. Нажать на заголовок колонки «Наличие ссылок» для сортировки «сначала есть».
2. Запомнить порядок задач в таблице.
3. Нажать повторно на заголовок колонки «Наличие ссылок» для сортировки «сначала нет».
4. Запомнить порядок задач в таблице.

**Ожидаемый результат:**
- После шага 2 задачи с внешними ссылками отображаются выше задач без ссылок.
- После шага 4 задачи без ссылок отображаются выше задач с внешними ссылками.
