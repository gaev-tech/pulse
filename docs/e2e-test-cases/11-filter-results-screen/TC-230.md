# TC-230: Кнопка меню "..." -- переход на Filter Activity Screen

**Предусловие:** пользователь авторизован, открыт Filter Results Screen личного фильтра.

**Шаги:**
1. Нажать кнопку «...» в верхней панели Filter Results Screen.
2. В открывшемся Filter Popup нажать на пункт перехода на Filter Activity Screen.

**Ожидаемый результат:**
- После шага 1 открывается Filter Popup с доступными действиями.
- После шага 2 открывается Filter Activity Screen для данного фильтра.
- На Filter Activity Screen отображается лента событий фильтра.
