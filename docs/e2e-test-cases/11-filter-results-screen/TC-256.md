# TC-256: Сортировка по дате изменения -- по возрастанию/убыванию

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Колонка «Дата изменения» включена. Среди результатов есть задачи, изменённые в разное время.

**Шаги:**
1. Нажать на заголовок колонки «Дата изменения» для сортировки по возрастанию.
2. Запомнить порядок задач в таблице.
3. Нажать повторно на заголовок колонки «Дата изменения» для сортировки по убыванию.
4. Запомнить порядок задач в таблице.

**Ожидаемый результат:**
- После шага 2 задачи отсортированы от самой давно изменённой к самой недавно изменённой.
- После шага 4 задачи отсортированы от самой недавно изменённой к самой давно изменённой.
