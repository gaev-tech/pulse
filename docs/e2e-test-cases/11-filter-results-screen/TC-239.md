# TC-239: Отображение колонки "Наличие блокеров"

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Колонка «Наличие блокеров» включена в настройках таблицы. Среди результатов есть задача с блокирующими задачами и задача без блокеров.

**Шаги:**
1. Осмотреть заголовки таблицы — убедиться в наличии колонки «Наличие блокеров».
2. Найти строки задач с блокерами и без блокеров в таблице.

**Ожидаемый результат:**
- В таблице отображается колонка «Наличие блокеров».
- Для задачи с блокерами в колонке отображается индикатор наличия.
- Для задачи без блокеров в колонке отображается индикатор отсутствия.
