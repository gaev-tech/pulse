# TC-228: Появление кнопки "Сохранить фильтр" при изменении критериев

**Предусловие:** пользователь авторизован, открыт Filter Results Screen личного фильтра в простом режиме. Сохранённый критерий: статус `opened`.

**Шаги:**
1. Убедиться, что кнопка «Сохранить фильтр» не отображается.
2. В строке критериев изменить статус с `opened` на `closed`.
3. Осмотреть верхнюю панель экрана.

**Ожидаемый результат:**
- До изменения критериев кнопка «Сохранить фильтр» отсутствует.
- После изменения критерия появляется кнопка «Сохранить фильтр».
