# TC-224: Отображение Filter Results Screen с результатами

**Предусловие:** пользователь авторизован, создан личный фильтр с критериями, которым соответствует хотя бы 3 задачи.

**Шаги:**
1. В Navigation Sidebar в разделе «Личные фильтры» нажать на название созданного фильтра.
2. Дождаться загрузки Filter Results Screen.

**Ожидаемый результат:**
- Открывается Filter Results Screen.
- В верхней панели отображается название фильтра.
- Под названием отображается строка критериев фильтра.
- Отображается таблица задач с результатами, соответствующими критериям фильтра.
- Каждая строка таблицы содержит обязательные колонки: чекбокс выбора, стрелка раскрытия иерархии, ключ задачи, название задачи.
