# TC-251: Отображение стрелки направления первичной сортировки

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра с результатами. Сортировка не установлена. Колонка «Ключ» отображается.

**Шаги:**
1. Нажать на заголовок колонки «Ключ».
2. Осмотреть заголовок колонки «Ключ».
3. Нажать повторно на заголовок колонки «Ключ».
4. Осмотреть заголовок колонки «Ключ».

**Ожидаемый результат:**
- После шага 2 в заголовке колонки «Ключ» отображается стрелка, указывающая направление сортировки (вверх или вниз).
- После шага 4 направление стрелки меняется на противоположное.
- Стрелка отображается только у колонки с активной первичной сортировкой.
