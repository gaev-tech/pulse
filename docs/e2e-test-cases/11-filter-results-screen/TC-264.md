# TC-264: Сортировка по наличию связей -- сначала есть/сначала нет

**Предусловие:** пользователь авторизован, открыт Filter Results Screen фильтра. Колонка «Наличие связей» включена. Среди результатов есть задачи со связями и без связей.

**Шаги:**
1. Нажать на заголовок колонки «Наличие связей» для сортировки «сначала есть».
2. Запомнить порядок задач в таблице.
3. Нажать повторно на заголовок колонки «Наличие связей» для сортировки «сначала нет».
4. Запомнить порядок задач в таблице.

**Ожидаемый результат:**
- После шага 2 задачи со связями отображаются выше задач без связей.
- После шага 4 задачи без связей отображаются выше задач со связями.
