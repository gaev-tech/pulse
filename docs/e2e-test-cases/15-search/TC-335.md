# TC-335: Сортировка результатов: точное совпадение ключа/названия первым

**Предусловие:** Пользователь авторизован. Существуют задачи: задача A с ключом `ACME-10` и названием «Рефакторинг», задача B с названием «ACME-10 — описание бага» (ключ другой), задача C с названием «Рефакторинг модуля авторизации». Пользователь имеет право `view` на все задачи. Search Modal открыт.

**Шаги:**
1. Ввести в поле поиска текст `ACME-10`.
2. Дождаться отображения результатов.
3. Очистить поле поиска.
4. Ввести в поле поиска текст «Рефакторинг».
5. Дождаться отображения результатов.

**Ожидаемый результат:**
- После шага 2: задача A (ключ `ACME-10`) отображается первой в списке, так как ключ точно совпадает с запросом.
- После шага 5: задача A с названием «Рефакторинг» отображается первой, так как название точно совпадает с запросом. Задача C «Рефакторинг модуля авторизации» отображается ниже.
