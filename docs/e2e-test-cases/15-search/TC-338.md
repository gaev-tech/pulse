# TC-338: Пустой результат поиска

**Предусловие:** Пользователь авторизован. В системе нет задач, содержащих текст «xyz999nonexistent» в ключе, названии или описании. Search Modal открыт.

**Шаги:**
1. Ввести в поле поиска текст «xyz999nonexistent».
2. Дождаться обработки запроса.

**Ожидаемый результат:**
- Список результатов пуст.
- Отображается сообщение об отсутствии результатов.
