# TC-331: Поиск задачи по ключу

**Предусловие:** Пользователь авторизован. Существует личная задача с ключом `#42` и командная задача с ключом `ACME-15`. Пользователь имеет право `view` на обе задачи. Search Modal открыт.

**Шаги:**
1. Ввести в поле поиска текст `#42`.
2. Дождаться отображения результатов.
3. Очистить поле поиска.
4. Ввести в поле поиска текст `ACME-15`.
5. Дождаться отображения результатов.

**Ожидаемый результат:**
- После шага 2: в списке результатов отображается задача с ключом `#42`.
- После шага 5: в списке результатов отображается задача с ключом `ACME-15`.
- Найденные задачи отображаются с ключом и названием.
