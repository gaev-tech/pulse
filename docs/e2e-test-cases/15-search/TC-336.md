# TC-336: Сортировка результатов: по дате последнего открытия задачи

**Предусловие:** Пользователь авторизован. Существуют три задачи с одинаковым словом «интеграция» в описании: задача A (открывалась пользователем вчера), задача B (открывалась пользователем сегодня), задача C (никогда не открывалась пользователем). Ни одна из задач не имеет точного совпадения ключа или названия с запросом. Search Modal открыт.

**Шаги:**
1. Ввести в поле поиска текст «интеграция».
2. Дождаться отображения результатов.

**Ожидаемый результат:**
- Задачи отсортированы по дате последнего открытия пользователем от новых к старым.
- Задача B (открыта сегодня) отображается первой.
- Задача A (открыта вчера) отображается второй.
- Задача C (никогда не открывалась) отображается последней.
