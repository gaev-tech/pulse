# TC-337: Клик по результату поиска — открытие Task Modal

**Предусловие:** Пользователь авторизован. Существует задача с ключом `ACME-5` и названием «Добавить логирование». Пользователь имеет право `view` на эту задачу. Search Modal открыт, в поле поиска введён текст «логирование», задача отображается в результатах.

**Шаги:**
1. Нажать на строку задачи «Добавить логирование» в списке результатов.

**Ожидаемый результат:**
- Search Modal закрывается.
- Открывается Task Modal с задачей `ACME-5` «Добавить логирование».
- Task Modal отображает все поля задачи: название, описание, статус, ответственного, метки, ссылки, связи, блокирование, родитель, вложения.
- Пользователь остаётся на том же экране, с которого был открыт поиск.
