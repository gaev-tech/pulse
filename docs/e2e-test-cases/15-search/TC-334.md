# TC-334: Результаты поиска обновляются в реальном времени при вводе

**Предусловие:** Пользователь авторизован. Существуют задачи с названиями «Настройка CI», «Настройка CD», «Настройка мониторинга». Пользователь имеет право `view` на все задачи. Search Modal открыт.

**Шаги:**
1. Ввести в поле поиска букву «Н».
2. Наблюдать за обновлением списка результатов.
3. Дополнить текст до «Настройка».
4. Наблюдать за обновлением списка результатов.
5. Дополнить текст до «Настройка CI».
6. Наблюдать за обновлением списка результатов.

**Ожидаемый результат:**
- После каждого ввода символа список результатов обновляется автоматически без нажатия дополнительных кнопок.
- После шага 3: в результатах отображаются все три задачи с «Настройка» в названии.
- После шага 5: в результатах остаётся только задача «Настройка CI».
