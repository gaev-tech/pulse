# TC-332: Поиск задачи по названию

**Предусловие:** Пользователь авторизован. Существует задача с названием «Исправить баг авторизации». Пользователь имеет право `view` на эту задачу. Search Modal открыт.

**Шаги:**
1. Ввести в поле поиска текст «Исправить баг авторизации».
2. Дождаться отображения результатов.

**Ожидаемый результат:**
- В списке результатов отображается задача с названием «Исправить баг авторизации».
- Задача отображается с ключом и названием.
