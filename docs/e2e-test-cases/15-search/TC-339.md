# TC-339: Закрытие Search Modal

**Предусловие:** Пользователь авторизован. Search Modal открыт, в поле поиска введён произвольный текст.

**Шаги:**
1. Нажать клавишу Escape или кликнуть вне области Search Modal.

**Ожидаемый результат:**
- Search Modal закрывается.
- Пользователь возвращается к экрану, с которого был открыт поиск.
- Состояние предыдущего экрана не изменилось.
