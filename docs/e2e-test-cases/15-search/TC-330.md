# TC-330: Открытие Search Modal по кнопке поиска

**Предусловие:** Пользователь авторизован и находится на любом экране системы. Navigation Sidebar отображается.

**Шаги:**
1. Найти кнопку поиска в Navigation Sidebar (расположена под заголовком системы).
2. Нажать на кнопку поиска.

**Ожидаемый результат:**
- Открывается Search Modal поверх текущего экрана.
- Search Modal содержит поле ввода для поискового запроса.
- Под полем ввода отображается область для списка результатов.
- Фокус автоматически установлен на поле ввода.
