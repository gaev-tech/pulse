# TC-333: Поиск задачи по описанию (contains)

**Предусловие:** Пользователь авторизован. Существует задача с описанием, содержащим текст «необходимо обновить зависимости до последней версии». Пользователь имеет право `view` на эту задачу. Search Modal открыт.

**Шаги:**
1. Ввести в поле поиска текст «обновить зависимости».
2. Дождаться отображения результатов.

**Ожидаемый результат:**
- В списке результатов отображается задача, в описании которой содержится текст «обновить зависимости».
- Поиск работает по принципу contains — достаточно частичного совпадения с описанием.
