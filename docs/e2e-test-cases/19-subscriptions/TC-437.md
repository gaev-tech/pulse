# TC-437: После апгрейда до Pro план обновляется в Profile Popup

**Предусловие:** Пользователь авторизован с планом Free; открыт Subscription Purchase Modal.

**Шаги:**
1. Нажать «Оформить» на карточке плана Pro.

**Ожидаемый результат:**
- Subscription Purchase Modal закрывается.
- В Profile Popup отображается план Pro.
