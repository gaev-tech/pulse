# TC-440: Раздел подписки команды в Team Edit Modal (план Free)

**Предусловие:** Пользователь авторизован как владелец команды с правом team.manage_owners; у команды план Free.

**Шаги:**
1. Открыть Team Edit Modal.
2. Найти раздел подписки команды.

**Ожидаемый результат:**
- Отображается текущий план команды (Free).
- Присутствует кнопка Upgrade.
