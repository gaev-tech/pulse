# TC-441: Апгрейд подписки команды Free → Team

**Предусловие:** Пользователь авторизован как владелец команды с правом team.manage_owners; у команды план Free; в Team Edit Modal открыт раздел подписки.

**Шаги:**
1. Нажать кнопку Upgrade.
2. В Subscription Purchase Modal выбрать план Team.
3. Нажать «Оформить».

**Ожидаемый результат:**
- Subscription Purchase Modal закрывается.
- В разделе подписки Team Edit Modal отображается план Team.
- Квотные лимиты команды (задачи, метки, фильтры, автоматизации, участники) сняты.
