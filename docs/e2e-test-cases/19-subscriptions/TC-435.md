# TC-435: Текущий план (Free) отображается в Profile Popup

**Предусловие:** Пользователь авторизован с планом Free.

**Шаги:**
1. Нажать на аватарку / имя пользователя в нижней части Navigation Sidebar.

**Ожидаемый результат:**
- Открывается Profile Popup.
- В Profile Popup отображается текущий план пользователя (Free).
- Присутствует кнопка Upgrade.
