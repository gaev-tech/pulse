# TC-436: Открытие Subscription Purchase Modal из Profile Popup

**Предусловие:** Пользователь авторизован с планом Free; Profile Popup открыт.

**Шаги:**
1. Нажать кнопку «Upgrade» в Profile Popup.

**Ожидаемый результат:**
- Открывается Subscription Purchase Modal.
- Отображаются карточки планов Free и Pro; карточка Free выделена как текущий.
- Карточка Pro содержит кнопку «Оформить».
