# TC-439: Quota Exceeded State — кнопка создания метки заменяется на CTA Upgrade

**Блокируется:** TC-438

**Предусловие:** Пользователь авторизован с планом Free; создано 20 личных меток (лимит исчерпан).

**Шаги:**
1. Открыть модальное окно личных меток.

**Ожидаемый результат:**
- Кнопка создания метки заменена состоянием Quota Exceeded с CTA «Upgrade».
- Клик по CTA открывает Subscription Purchase Modal.
