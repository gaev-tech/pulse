# TC-438: Quota Exceeded State — кнопка создания задачи заменяется на CTA Upgrade

**Предусловие:** Пользователь авторизован с планом Free; создано 200 личных задач (лимит исчерпан).

**Шаги:**
1. Попытаться создать задачу (открыть Task Create Modal).

**Ожидаемый результат:**
- Кнопка создания задачи заменена состоянием Quota Exceeded с CTA «Upgrade».
- Клик по CTA открывает Subscription Purchase Modal.
