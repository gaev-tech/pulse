# TC-206: RSQL: фильтрация по полю label_id

**Предусловие:** пользователь авторизован, создан личный фильтр в RSQL-режиме, существуют задачи с различными метками. Известен UUID метки (например, `label-uuid-1`).

**Шаги:**
1. Открыть Filter Results Screen созданного фильтра через Navigation Sidebar.
2. В textarea RSQL-выражения ввести `label_id==label-uuid-1`.
3. Дождаться обновления результатов таблицы.
4. Очистить textarea и ввести `label_id=in=(label-uuid-1,label-uuid-2)`.
5. Дождаться обновления результатов таблицы.

**Ожидаемый результат:**
- После шага 3 в таблице отображаются только задачи с меткой `label-uuid-1`.
- После шага 5 в таблице отображаются задачи, имеющие метку `label-uuid-1` или `label-uuid-2`.
