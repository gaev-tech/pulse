# TC-197: RSQL: логический оператор ; (AND)

**Предусловие:** Пользователь авторизован. Существует фильтр в RSQL режиме. Открыт Filter Results Screen. Существуют задачи: #1 — статус "opened", название "Баг авторизации"; #2 — статус "closed", название "Баг поиска"; #3 — статус "opened", название "Улучшение UI".

**Шаги:**
1. В строке критериев (textarea) ввести выражение `status==opened;title==*Баг*`.
2. Дождаться обновления результатов.

**Ожидаемый результат:**
- В таблице результатов отображается только задача #1 — статус "opened" и название содержит "Баг".
- Задача #2 не отображается (статус "closed").
- Задача #3 не отображается (название не содержит "Баг").
- Оператор `;` (AND) корректно объединяет условия логическим "И".
