# TC-216: RSQL: фильтрация по полю blocking_id

**Предусловие:** пользователь авторизован, создан личный фильтр в RSQL-режиме, существуют задачи, блокирующие другие задачи. Известен UUID задачи, которую блокируют (например, `blocked-task-uuid-1`).

**Шаги:**
1. Открыть Filter Results Screen созданного фильтра через Navigation Sidebar.
2. В textarea RSQL-выражения ввести `blocking_id==blocked-task-uuid-1`.
3. Дождаться обновления результатов таблицы.
4. Очистить textarea и ввести `blocking_id=in=(blocked-task-uuid-1,blocked-task-uuid-2)`.
5. Дождаться обновления результатов таблицы.

**Ожидаемый результат:**
- После шага 3 в таблице отображаются только задачи, которые блокируют задачу `blocked-task-uuid-1`.
- После шага 5 в таблице отображаются задачи, которые блокируют `blocked-task-uuid-1` или `blocked-task-uuid-2`.
