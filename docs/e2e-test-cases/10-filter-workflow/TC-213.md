# TC-213: RSQL: фильтрация по полю is_blocked

**Предусловие:** пользователь авторизован, создан личный фильтр в RSQL-режиме, существуют задачи, которые заблокированы другими задачами, и незаблокированные задачи.

**Шаги:**
1. Открыть Filter Results Screen созданного фильтра через Navigation Sidebar.
2. В textarea RSQL-выражения ввести `is_blocked==true`.
3. Дождаться обновления результатов таблицы.
4. Очистить textarea и ввести `is_blocked==false`.
5. Дождаться обновления результатов таблицы.

**Ожидаемый результат:**
- После шага 3 в таблице отображаются только задачи, имеющие хотя бы одну блокирующую задачу.
- После шага 5 в таблице отображаются только задачи без блокирующих задач.
