# TC-193: RSQL: оператор =gt= (больше)

**Предусловие:** Пользователь авторизован. Существует фильтр в RSQL режиме. Открыт Filter Results Screen. Существуют задачи: #1 создана 2025-01-01, #2 создана 2025-06-01, #3 создана 2025-12-01.

**Шаги:**
1. В строке критериев (textarea) ввести выражение `created_at=gt=2025-06-01T00:00:00Z`.
2. Дождаться обновления результатов.

**Ожидаемый результат:**
- В таблице результатов отображается только задача #3, созданная после 2025-06-01.
- Задачи #1 и #2 не отображаются.
- Оператор `=gt=` корректно фильтрует по условию "строго больше".
