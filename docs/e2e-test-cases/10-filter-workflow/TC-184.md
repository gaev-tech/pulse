# TC-184: Простой режим: фильтр по меткам (несколько)

**Предусловие:** Пользователь авторизован. Открыта Filter Create Modal в простом режиме. Существуют задачи: #1 с меткой "Срочно", #2 с меткой "Улучшение", #3 с метками "Срочно" и "Улучшение", #4 без меток.

**Шаги:**
1. В поле "Название" ввести "Срочные и улучшения".
2. Выбрать тип фильтра "Личный".
3. В теге "Метки" выбрать метку "Срочно".
4. В теге "Метки" дополнительно выбрать метку "Улучшение".
5. Нажать кнопку создания фильтра.
6. Открыть созданный фильтр на Filter Results Screen.

**Ожидаемый результат:**
- В результатах фильтра отображаются задачи #1, #2 и #3, имеющие хотя бы одну из выбранных меток.
- Задача #4 без меток не отображается.
