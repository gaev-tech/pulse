# TC-220: RSQL: фильтрация по полю attachment_name

**Предусловие:** пользователь авторизован, создан личный фильтр в RSQL-режиме, существуют задачи с вложениями, в том числе задача с вложением «report-2025.pdf».

**Шаги:**
1. Открыть Filter Results Screen созданного фильтра через Navigation Sidebar.
2. В textarea RSQL-выражения ввести `attachment_name==*report*`.
3. Дождаться обновления результатов таблицы.

**Ожидаемый результат:**
- В таблице отображаются только задачи, имеющие вложение с именем, содержащим подстроку «report».
- Задача с вложением «report-2025.pdf» присутствует в результатах.
- Задачи без вложений или с вложениями, не содержащими «report» в имени, отсутствуют в результатах.
