# TC-191: RSQL: оператор =lt= (меньше)

**Предусловие:** Пользователь авторизован. Существует фильтр в RSQL режиме. Открыт Filter Results Screen. Существуют задачи: #1 создана 2025-01-01, #2 создана 2025-06-01, #3 создана 2025-12-01.

**Шаги:**
1. В строке критериев (textarea) ввести выражение `created_at=lt=2025-06-01T00:00:00Z`.
2. Дождаться обновления результатов.

**Ожидаемый результат:**
- В таблице результатов отображается только задача #1, созданная до 2025-06-01.
- Задачи #2 и #3 не отображаются.
- Оператор `=lt=` корректно фильтрует по условию "строго меньше".
