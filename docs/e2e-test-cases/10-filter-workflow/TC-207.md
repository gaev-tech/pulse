# TC-207: RSQL: фильтрация по полю team_id

**Предусловие:** пользователь авторизован, создан личный фильтр в RSQL-режиме, существуют задачи в разных командах. Известен UUID команды (например, `team-uuid-1`).

**Шаги:**
1. Открыть Filter Results Screen созданного фильтра через Navigation Sidebar.
2. В textarea RSQL-выражения ввести `team_id==team-uuid-1`.
3. Дождаться обновления результатов таблицы.

**Ожидаемый результат:**
- В таблице отображаются только задачи, принадлежащие команде с UUID `team-uuid-1`.
- Задачи из других команд и личные задачи отсутствуют в результатах.
