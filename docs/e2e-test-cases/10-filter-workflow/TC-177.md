# TC-177: Создание командного фильтра — выбор команды

**Предусловие:** Пользователь авторизован. Пользователь является участником команды "Alpha Team" с правом `team.manage_filters`. Открыта Filter Create Modal.

**Шаги:**
1. В поле "Название" ввести "Командные баги".
2. Выбрать тип фильтра "Командный".
3. В появившемся списке команд выбрать "Alpha Team".
4. В теге "Статус" выбрать значение "opened".
5. Нажать кнопку создания фильтра.

**Ожидаемый результат:**
- Командный фильтр "Командные баги" успешно создан для команды "Alpha Team".
- Filter Create Modal закрывается.
- В Navigation Sidebar фильтр "Командные баги" отображается под командой "Alpha Team".
