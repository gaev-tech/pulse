# TC-186: Простой режим: фильтр по команде (только для личных фильтров)

**Предусловие:** Пользователь авторизован. Открыта Filter Create Modal в простом режиме. Пользователь является участником команд "Alpha Team" и "Beta Team". Существуют задачи: ALPHA-1 (команда Alpha), BETA-1 (команда Beta), #1 (личная).

**Шаги:**
1. В поле "Название" ввести "Задачи Alpha".
2. Выбрать тип фильтра "Личный".
3. В теге "Команда" выбрать "Alpha Team".
4. Нажать кнопку создания фильтра.
5. Открыть созданный фильтр на Filter Results Screen.

**Ожидаемый результат:**
- В результатах фильтра отображается только задача ALPHA-1 команды "Alpha Team".
- Задачи BETA-1 и #1 не отображаются.
