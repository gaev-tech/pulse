# TC-218: RSQL: фильтрация по полю parent_id

**Предусловие:** пользователь авторизован, создан личный фильтр в RSQL-режиме, существуют задачи с родительскими задачами. Известен UUID родительской задачи (например, `parent-uuid-1`).

**Шаги:**
1. Открыть Filter Results Screen созданного фильтра через Navigation Sidebar.
2. В textarea RSQL-выражения ввести `parent_id==parent-uuid-1`.
3. Дождаться обновления результатов таблицы.
4. Очистить textarea и ввести `parent_id=in=(parent-uuid-1,parent-uuid-2)`.
5. Дождаться обновления результатов таблицы.

**Ожидаемый результат:**
- После шага 3 в таблице отображаются только задачи, чья родительская задача — `parent-uuid-1`.
- После шага 5 в таблице отображаются задачи с родительской задачей `parent-uuid-1` или `parent-uuid-2`.
