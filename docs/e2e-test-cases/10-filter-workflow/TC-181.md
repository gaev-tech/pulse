# TC-181: Простой режим: фильтр по ответственному (несколько)

**Предусловие:** Пользователь авторизован. Открыта Filter Create Modal в простом режиме. Существуют задачи: #1 — ответственный "Иванов", #2 — ответственный "Петров", #3 — ответственный "Сидоров".

**Шаги:**
1. В поле "Название" ввести "Задачи Иванова и Петрова".
2. Выбрать тип фильтра "Личный".
3. В теге "Ответственный" выбрать пользователя "Иванов".
4. В теге "Ответственный" дополнительно выбрать пользователя "Петров".
5. Нажать кнопку создания фильтра.
6. Открыть созданный фильтр на Filter Results Screen.

**Ожидаемый результат:**
- В результатах фильтра отображаются задачи #1 и #2.
- Задача #3 с ответственным "Сидоров" не отображается.
