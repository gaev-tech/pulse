# TC-203: RSQL: фильтрация по полю description

**Предусловие:** пользователь авторизован, создан личный фильтр в RSQL-режиме, существуют задачи с различными описаниями, в том числе задача с описанием, содержащим слово «urgent».

**Шаги:**
1. Открыть Filter Results Screen созданного фильтра через Navigation Sidebar.
2. В textarea RSQL-выражения ввести `description==*urgent*`.
3. Дождаться обновления результатов таблицы.

**Ожидаемый результат:**
- В таблице отображаются только задачи, описание которых содержит подстроку «urgent».
- Задачи без подстроки «urgent» в описании отсутствуют в результатах.
