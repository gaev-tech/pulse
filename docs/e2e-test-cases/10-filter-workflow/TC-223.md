# TC-223: RSQL: невалидное выражение -- отображение ошибки

**Предусловие:** пользователь авторизован, создан личный фильтр в RSQL-режиме.

**Шаги:**
1. Открыть Filter Results Screen созданного фильтра через Navigation Sidebar.
2. В textarea RSQL-выражения ввести невалидное выражение, например `status===opened` (три знака равенства).
3. Дождаться реакции интерфейса.
4. Очистить textarea и ввести другое невалидное выражение: `unknown_field==value`.
5. Дождаться реакции интерфейса.

**Ожидаемый результат:**
- После шага 3 отображается сообщение об ошибке, указывающее на невалидный синтаксис RSQL-выражения.
- Таблица результатов не содержит данных или сохраняет предыдущие результаты.
- После шага 5 отображается сообщение об ошибке, указывающее на неизвестное поле.
