# TC-178: Отображение только команд с правом team.manage_filters

**Предусловие:** Пользователь авторизован. Пользователь является участником команды "Alpha Team" с правом `team.manage_filters` и участником команды "Beta Team" без права `team.manage_filters`. Открыта Filter Create Modal.

**Шаги:**
1. Выбрать тип фильтра "Командный".
2. Просмотреть список доступных команд.

**Ожидаемый результат:**
- В списке команд отображается только "Alpha Team".
- Команда "Beta Team" отсутствует в списке, так как у пользователя нет права `team.manage_filters` в этой команде.
