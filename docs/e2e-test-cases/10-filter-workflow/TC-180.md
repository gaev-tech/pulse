# TC-180: Простой режим: фильтр по ответственному (один)

**Предусловие:** Пользователь авторизован. Открыта Filter Create Modal в простом режиме. Существуют задачи с разными ответственными: задача #1 — ответственный "Иванов", задача #2 — ответственный "Петров".

**Шаги:**
1. В поле "Название" ввести "Задачи Иванова".
2. Выбрать тип фильтра "Личный".
3. В теге "Ответственный" выбрать пользователя "Иванов".
4. Нажать кнопку создания фильтра.
5. Открыть созданный фильтр на Filter Results Screen.

**Ожидаемый результат:**
- В результатах фильтра отображается только задача #1 с ответственным "Иванов".
- Задача #2 с ответственным "Петров" не отображается.
