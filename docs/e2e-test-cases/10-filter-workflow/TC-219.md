# TC-219: RSQL: фильтрация по полю has_attachments

**Предусловие:** пользователь авторизован, создан личный фильтр в RSQL-режиме, существуют задачи с вложениями и без них.

**Шаги:**
1. Открыть Filter Results Screen созданного фильтра через Navigation Sidebar.
2. В textarea RSQL-выражения ввести `has_attachments==true`.
3. Дождаться обновления результатов таблицы.
4. Очистить textarea и ввести `has_attachments==false`.
5. Дождаться обновления результатов таблицы.

**Ожидаемый результат:**
- После шага 3 в таблице отображаются только задачи, имеющие хотя бы одно вложение.
- После шага 5 в таблице отображаются только задачи без вложений.
