# TC-174: Открытие Filter Create Modal

**Предусловие:** Пользователь авторизован и находится на любом экране с Navigation Sidebar.

**Шаги:**
1. В Navigation Sidebar найти раздел "Личные фильтры".
2. Нажать кнопку создания фильтра рядом с заголовком раздела.

**Ожидаемый результат:**
- Открывается модальное окно Filter Create Modal.
- Модалка содержит поле "Название".
- Доступен выбор типа фильтра: личный или командный.
- Отображаются критерии фильтрации в простом режиме по умолчанию.
- Доступна возможность переключения между простым и RSQL режимами.
