# TC-187: Отсутствие тега "Команда" в командном фильтре

**Предусловие:** Пользователь авторизован. Пользователь является участником команды "Alpha Team" с правом `team.manage_filters`. Открыта Filter Create Modal.

**Шаги:**
1. Выбрать тип фильтра "Командный".
2. Выбрать команду "Alpha Team".
3. Убедиться, что выбран простой режим фильтрации.
4. Просмотреть доступные теги критериев.

**Ожидаемый результат:**
- Среди тегов критериев отсутствует тег "Команда".
- Доступны теги: поиск, ответственный, статус, метки.
- Тег "Команда" не отображается, так как командный фильтр уже привязан к конкретной команде.
