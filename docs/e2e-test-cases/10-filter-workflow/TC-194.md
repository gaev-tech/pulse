# TC-194: RSQL: оператор =ge= (больше или равно)

**Предусловие:** Пользователь авторизован. Существует фильтр в RSQL режиме. Открыт Filter Results Screen. Существуют задачи: #1 создана 2025-01-01, #2 создана 2025-06-01, #3 создана 2025-12-01.

**Шаги:**
1. В строке критериев (textarea) ввести выражение `created_at=ge=2025-06-01T00:00:00Z`.
2. Дождаться обновления результатов.

**Ожидаемый результат:**
- В таблице результатов отображаются задачи #2 и #3.
- Задача #1 не отображается.
- Оператор `=ge=` корректно фильтрует по условию "больше или равно".
