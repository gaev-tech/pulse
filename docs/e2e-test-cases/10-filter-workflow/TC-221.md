# TC-221: Переключение режима с простого на RSQL -- сброс критериев

**Предусловие:** пользователь авторизован, создан личный фильтр в простом режиме с заполненными критериями (например, установлен статус `opened` и выбрана метка).

**Шаги:**
1. Открыть Filter Results Screen созданного фильтра через Navigation Sidebar.
2. Убедиться, что отображаются критерии простого режима: инпут поиска, теги ответственного, статуса, меток, команды.
3. Нажать кнопку настроек таблицы для открытия Table Settings Popup.
4. Переключить режим фильтрации на RSQL.
5. Закрыть Table Settings Popup.

**Ожидаемый результат:**
- Строка критериев заменяется на textarea для ввода RSQL-выражения.
- Textarea пуста — ранее введённые критерии простого режима сброшены.
- Результаты таблицы обновляются в соответствии с пустым RSQL-выражением.
