# TC-200: RSQL: фильтрация по полю status

**Предусловие:** пользователь авторизован, создан личный фильтр в RSQL-режиме, существуют задачи со статусами `opened` и `closed`.

**Шаги:**
1. Открыть Filter Results Screen созданного фильтра через Navigation Sidebar.
2. В textarea RSQL-выражения ввести `status==opened`.
3. Дождаться обновления результатов таблицы.
4. Очистить textarea и ввести `status==closed`.
5. Дождаться обновления результатов таблицы.

**Ожидаемый результат:**
- После шага 3 в таблице отображаются только задачи со статусом `opened`.
- После шага 5 в таблице отображаются только задачи со статусом `closed`.
