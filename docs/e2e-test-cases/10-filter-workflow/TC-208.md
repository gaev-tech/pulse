# TC-208: RSQL: фильтрация по полю created_at

**Предусловие:** пользователь авторизован, создан личный фильтр в RSQL-режиме, существуют задачи, созданные в разное время — до и после 2025-01-01.

**Шаги:**
1. Открыть Filter Results Screen созданного фильтра через Navigation Sidebar.
2. В textarea RSQL-выражения ввести `created_at=gt=2025-01-01T00:00:00Z`.
3. Дождаться обновления результатов таблицы.
4. Очистить textarea и ввести `created_at=lt=2025-01-01T00:00:00Z`.
5. Дождаться обновления результатов таблицы.

**Ожидаемый результат:**
- После шага 3 в таблице отображаются только задачи, созданные после 2025-01-01.
- После шага 5 в таблице отображаются только задачи, созданные до 2025-01-01.
