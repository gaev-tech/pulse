# TC-190: RSQL: оператор != (не равно)

**Предусловие:** Пользователь авторизован. Существует фильтр в RSQL режиме. Открыт Filter Results Screen. Существуют задачи: #1 со статусом "opened", #2 со статусом "closed".

**Шаги:**
1. В строке критериев (textarea) ввести выражение `status!=opened`.
2. Дождаться обновления результатов.

**Ожидаемый результат:**
- В таблице результатов отображается только задача #2 со статусом "closed".
- Задача #1 со статусом "opened" не отображается.
- Оператор `!=` корректно исключает задачи с указанным значением.
