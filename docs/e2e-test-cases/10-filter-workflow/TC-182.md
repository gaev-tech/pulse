# TC-182: Простой режим: фильтр по статусу

**Предусловие:** Пользователь авторизован. Открыта Filter Create Modal в простом режиме. Существуют задачи: #1 со статусом "opened", #2 со статусом "closed".

**Шаги:**
1. В поле "Название" ввести "Закрытые задачи".
2. Выбрать тип фильтра "Личный".
3. В теге "Статус" выбрать значение "closed".
4. Нажать кнопку создания фильтра.
5. Открыть созданный фильтр на Filter Results Screen.

**Ожидаемый результат:**
- В результатах фильтра отображается только задача #2 со статусом "closed".
- Задача #1 со статусом "opened" не отображается.
