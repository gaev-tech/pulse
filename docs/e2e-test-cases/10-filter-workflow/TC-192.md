# TC-192: RSQL: оператор =le= (меньше или равно)

**Предусловие:** Пользователь авторизован. Существует фильтр в RSQL режиме. Открыт Filter Results Screen. Существуют задачи: #1 создана 2025-01-01, #2 создана 2025-06-01, #3 создана 2025-12-01.

**Шаги:**
1. В строке критериев (textarea) ввести выражение `created_at=le=2025-06-01T00:00:00Z`.
2. Дождаться обновления результатов.

**Ожидаемый результат:**
- В таблице результатов отображаются задачи #1 и #2.
- Задача #3 не отображается.
- Оператор `=le=` корректно фильтрует по условию "меньше или равно".
