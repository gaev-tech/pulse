# TC-211: RSQL: фильтрация по полю has_relation

**Предусловие:** пользователь авторизован, создан личный фильтр в RSQL-режиме, существуют задачи со связями и без связей.

**Шаги:**
1. Открыть Filter Results Screen созданного фильтра через Navigation Sidebar.
2. В textarea RSQL-выражения ввести `has_relation==true`.
3. Дождаться обновления результатов таблицы.
4. Очистить textarea и ввести `has_relation==false`.
5. Дождаться обновления результатов таблицы.

**Ожидаемый результат:**
- После шага 3 в таблице отображаются только задачи, имеющие хотя бы одну связанную задачу.
- После шага 5 в таблице отображаются только задачи без связей.
