# TC-196: RSQL: оператор =out= (не входит в список)

**Предусловие:** Пользователь авторизован. Существует фильтр в RSQL режиме. Открыт Filter Results Screen. Существуют задачи: #1 — ответственный "Иванов" (uuid1), #2 — ответственный "Петров" (uuid2), #3 — ответственный "Сидоров" (uuid3).

**Шаги:**
1. В строке критериев (textarea) ввести выражение `assignee_id=out=(uuid1,uuid2)`, подставив реальные UUID пользователей "Иванов" и "Петров".
2. Дождаться обновления результатов.

**Ожидаемый результат:**
- В таблице результатов отображается только задача #3 с ответственным "Сидоров".
- Задачи #1 и #2 не отображаются.
- Оператор `=out=` корректно исключает задачи, значение которых входит в указанный список.
