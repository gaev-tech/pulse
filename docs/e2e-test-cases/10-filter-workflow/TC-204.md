# TC-204: RSQL: фильтрация по полю author_id

**Предусловие:** пользователь авторизован, создан личный фильтр в RSQL-режиме, существуют задачи, созданные разными пользователями. Известен UUID пользователя-автора (например, `user-uuid-1`).

**Шаги:**
1. Открыть Filter Results Screen созданного фильтра через Navigation Sidebar.
2. В textarea RSQL-выражения ввести `author_id==user-uuid-1`.
3. Дождаться обновления результатов таблицы.
4. Очистить textarea и ввести `author_id=in=(user-uuid-1,user-uuid-2)`.
5. Дождаться обновления результатов таблицы.

**Ожидаемый результат:**
- После шага 3 в таблице отображаются только задачи, созданные пользователем с UUID `user-uuid-1`.
- После шага 5 в таблице отображаются задачи, созданные пользователями с UUID `user-uuid-1` и `user-uuid-2`.
