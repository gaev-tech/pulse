# TC-199: RSQL: группировка скобками ()

**Предусловие:** Пользователь авторизован. Существует фильтр в RSQL режиме. Открыт Filter Results Screen. Существуют задачи: #1 — статус "opened", есть вложения; #2 — статус "opened", нет вложений; #3 — статус "closed", есть связи; #4 — статус "closed", нет связей и вложений.

**Шаги:**
1. В строке критериев (textarea) ввести выражение `status==opened;(has_attachments==true,has_relation==true)`.
2. Дождаться обновления результатов.

**Ожидаемый результат:**
- В таблице результатов отображается только задача #1 — статус "opened" и есть вложения.
- Задача #2 не отображается (статус "opened", но нет вложений и связей).
- Задачи #3 и #4 не отображаются (статус "closed").
- Скобки корректно группируют условия: сначала вычисляется выражение внутри скобок (OR), затем результат объединяется с внешним условием (AND).
