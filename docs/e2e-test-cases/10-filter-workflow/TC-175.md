# TC-175: Создание личного фильтра — валидация обязательного поля "Название"

**Предусловие:** Пользователь авторизован. Открыта Filter Create Modal.

**Шаги:**
1. Оставить поле "Название" пустым.
2. Выбрать тип фильтра "Личный".
3. Нажать кнопку создания фильтра.

**Ожидаемый результат:**
- Фильтр не создаётся.
- Отображается ошибка валидации для обязательного поля "Название".
