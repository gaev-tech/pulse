# TC-183: Простой режим: фильтр по меткам (одна)

**Предусловие:** Пользователь авторизован. Открыта Filter Create Modal в простом режиме. Существуют задачи: #1 с меткой "Срочно", #2 с меткой "Улучшение", #3 без меток.

**Шаги:**
1. В поле "Название" ввести "Срочные задачи".
2. Выбрать тип фильтра "Личный".
3. В теге "Метки" выбрать метку "Срочно".
4. Нажать кнопку создания фильтра.
5. Открыть созданный фильтр на Filter Results Screen.

**Ожидаемый результат:**
- В результатах фильтра отображается только задача #1 с меткой "Срочно".
- Задачи #2 и #3 не отображаются.
