# TC-214: RSQL: фильтрация по полю blocker_id

**Предусловие:** пользователь авторизован, создан личный фильтр в RSQL-режиме, существуют задачи с блокерами. Известен UUID блокирующей задачи (например, `blocker-uuid-1`).

**Шаги:**
1. Открыть Filter Results Screen созданного фильтра через Navigation Sidebar.
2. В textarea RSQL-выражения ввести `blocker_id==blocker-uuid-1`.
3. Дождаться обновления результатов таблицы.

**Ожидаемый результат:**
- В таблице отображаются только задачи, которые заблокированы задачей с UUID `blocker-uuid-1`.
- Задачи, не заблокированные этой задачей, отсутствуют в результатах.
