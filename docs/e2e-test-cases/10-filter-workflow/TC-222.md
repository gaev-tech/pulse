# TC-222: Переключение режима с RSQL на простой -- сброс критериев

**Предусловие:** пользователь авторизован, создан личный фильтр в RSQL-режиме с заполненным выражением (например, `status==opened;has_links==true`).

**Шаги:**
1. Открыть Filter Results Screen созданного фильтра через Navigation Sidebar.
2. Убедиться, что отображается textarea с RSQL-выражением.
3. Нажать кнопку настроек таблицы для открытия Table Settings Popup.
4. Переключить режим фильтрации на простой.
5. Закрыть Table Settings Popup.

**Ожидаемый результат:**
- Textarea заменяется на строку критериев простого режима: инпут поиска, теги ответственного, статуса, меток, команды.
- Все критерии простого режима пусты — ранее введённое RSQL-выражение сброшено.
- Результаты таблицы обновляются в соответствии с пустыми критериями.
