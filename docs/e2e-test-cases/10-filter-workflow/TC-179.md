# TC-179: Простой режим: поиск по ключу и названию

**Предусловие:** Пользователь авторизован. Открыта Filter Create Modal в простом режиме. Существуют задачи: #1 "Исправить авторизацию", #2 "Добавить поиск", ALPHA-1 "Исправить баг".

**Шаги:**
1. В поле "Название" ввести "Поиск задач".
2. Выбрать тип фильтра "Личный".
3. В инпуте поиска ввести "Исправить".
4. Нажать кнопку создания фильтра.
5. Открыть созданный фильтр на Filter Results Screen.

**Ожидаемый результат:**
- В результатах фильтра отображаются задачи, содержащие "Исправить" в ключе или названии: #1 "Исправить авторизацию" и ALPHA-1 "Исправить баг".
- Задача #2 "Добавить поиск" не отображается в результатах.
