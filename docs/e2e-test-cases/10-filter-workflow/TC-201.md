# TC-201: RSQL: фильтрация по полю title

**Предусловие:** пользователь авторизован, создан личный фильтр в RSQL-режиме, существуют задачи с различными названиями, в том числе задача с названием «Исправить баг авторизации».

**Шаги:**
1. Открыть Filter Results Screen созданного фильтра через Navigation Sidebar.
2. В textarea RSQL-выражения ввести `title==*баг*`.
3. Дождаться обновления результатов таблицы.

**Ожидаемый результат:**
- В таблице отображаются только задачи, название которых содержит подстроку «баг».
- Задача «Исправить баг авторизации» присутствует в результатах.
- Задачи без подстроки «баг» в названии отсутствуют в результатах.
