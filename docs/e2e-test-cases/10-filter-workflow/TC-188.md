# TC-188: Создание фильтра в RSQL режиме

**Предусловие:** Пользователь авторизован. Открыта Filter Create Modal.

**Шаги:**
1. В поле "Название" ввести "RSQL фильтр".
2. Выбрать тип фильтра "Личный".
3. Переключить режим фильтрации на RSQL.
4. В textarea ввести выражение `status==opened;assignee_id==<uuid пользователя>`.
5. Нажать кнопку создания фильтра.

**Ожидаемый результат:**
- Фильтр "RSQL фильтр" успешно создан в RSQL режиме.
- Filter Create Modal закрывается.
- Фильтр отображается в разделе "Личные фильтры" Navigation Sidebar.
- При открытии Filter Results Screen отображаются задачи, соответствующие RSQL-выражению.
