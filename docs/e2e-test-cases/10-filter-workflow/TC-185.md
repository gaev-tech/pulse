# TC-185: Простой режим: фильтр "без меток"

**Предусловие:** Пользователь авторизован. Открыта Filter Create Modal в простом режиме. Существуют задачи: #1 с меткой "Срочно", #2 без меток, #3 без меток.

**Шаги:**
1. В поле "Название" ввести "Задачи без меток".
2. Выбрать тип фильтра "Личный".
3. В теге "Метки" выбрать опцию "Без меток".
4. Нажать кнопку создания фильтра.
5. Открыть созданный фильтр на Filter Results Screen.

**Ожидаемый результат:**
- В результатах фильтра отображаются только задачи #2 и #3, не имеющие меток.
- Задача #1 с меткой "Срочно" не отображается.
