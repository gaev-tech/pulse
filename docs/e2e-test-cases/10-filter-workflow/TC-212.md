# TC-212: RSQL: фильтрация по полю relation_task_id

**Предусловие:** пользователь авторизован, создан личный фильтр в RSQL-режиме, существуют задачи со связями. Известен UUID задачи, являющейся связанной (например, `related-task-uuid-1`).

**Шаги:**
1. Открыть Filter Results Screen созданного фильтра через Navigation Sidebar.
2. В textarea RSQL-выражения ввести `relation_task_id==related-task-uuid-1`.
3. Дождаться обновления результатов таблицы.
4. Очистить textarea и ввести `relation_task_id=in=(related-task-uuid-1,related-task-uuid-2)`.
5. Дождаться обновления результатов таблицы.

**Ожидаемый результат:**
- После шага 3 в таблице отображаются только задачи, связанные с задачей `related-task-uuid-1`.
- После шага 5 в таблице отображаются задачи, связанные с `related-task-uuid-1` или `related-task-uuid-2`.
