# TC-176: Создание личного фильтра в простом режиме

**Предусловие:** Пользователь авторизован. Открыта Filter Create Modal.

**Шаги:**
1. В поле "Название" ввести "Мои открытые задачи".
2. Выбрать тип фильтра "Личный".
3. Убедиться, что выбран простой режим фильтрации.
4. В теге "Статус" выбрать значение "opened".
5. Нажать кнопку создания фильтра.

**Ожидаемый результат:**
- Фильтр "Мои открытые задачи" успешно создан.
- Filter Create Modal закрывается.
- В разделе "Личные фильтры" Navigation Sidebar появляется новый фильтр "Мои открытые задачи".
- При клике на фильтр открывается Filter Results Screen с задачами, соответствующими критерию status=opened.
