# TC-217: RSQL: фильтрация по полю has_parent

**Предусловие:** пользователь авторизован, создан личный фильтр в RSQL-режиме, существуют задачи с родительской задачей и без неё.

**Шаги:**
1. Открыть Filter Results Screen созданного фильтра через Navigation Sidebar.
2. В textarea RSQL-выражения ввести `has_parent==true`.
3. Дождаться обновления результатов таблицы.
4. Очистить textarea и ввести `has_parent==false`.
5. Дождаться обновления результатов таблицы.

**Ожидаемый результат:**
- После шага 3 в таблице отображаются только задачи, у которых назначена родительская задача.
- После шага 5 в таблице отображаются только задачи без родительской задачи.
