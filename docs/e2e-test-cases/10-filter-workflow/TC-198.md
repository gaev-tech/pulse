# TC-198: RSQL: логический оператор , (OR)

**Предусловие:** Пользователь авторизован. Существует фильтр в RSQL режиме. Открыт Filter Results Screen. Существуют задачи: #1 — статус "opened", #2 — статус "closed" с вложениями, #3 — статус "closed" без вложений.

**Шаги:**
1. В строке критериев (textarea) ввести выражение `status==opened,has_attachments==true`.
2. Дождаться обновления результатов.

**Ожидаемый результат:**
- В таблице результатов отображаются задачи #1 (статус "opened") и #2 (есть вложения).
- Задача #3 не отображается (статус "closed" и нет вложений).
- Оператор `,` (OR) корректно объединяет условия логическим "ИЛИ".
