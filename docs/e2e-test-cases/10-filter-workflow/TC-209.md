# TC-209: RSQL: фильтрация по полю updated_at

**Предусловие:** пользователь авторизован, создан личный фильтр в RSQL-режиме, существуют задачи, обновлённые в разное время, в том числе задачи, обновлённые за последние 7 дней.

**Шаги:**
1. Открыть Filter Results Screen созданного фильтра через Navigation Sidebar.
2. В textarea RSQL-выражения ввести `updated_at=ge=-7d`.
3. Дождаться обновления результатов таблицы.

**Ожидаемый результат:**
- В таблице отображаются только задачи, обновлённые за последние 7 дней.
- Задачи, не обновлявшиеся более 7 дней, отсутствуют в результатах.
