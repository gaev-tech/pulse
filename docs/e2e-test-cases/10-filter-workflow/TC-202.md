# TC-202: RSQL: фильтрация по полю key

**Предусловие:** пользователь авторизован, создан личный фильтр в RSQL-режиме, существуют личные задачи (`#1`, `#2`, `#3`) и командные задачи (`ACME-1`, `ACME-2`).

**Шаги:**
1. Открыть Filter Results Screen созданного фильтра через Navigation Sidebar.
2. В textarea RSQL-выражения ввести `key==ACME*`.
3. Дождаться обновления результатов таблицы.

**Ожидаемый результат:**
- В таблице отображаются только задачи команды ACME (`ACME-1`, `ACME-2`).
- Личные задачи (`#1`, `#2`, `#3`) отсутствуют в результатах.
