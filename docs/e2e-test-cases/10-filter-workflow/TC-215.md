# TC-215: RSQL: фильтрация по полю is_blocker

**Предусловие:** пользователь авторизован, создан личный фильтр в RSQL-режиме, существуют задачи, которые блокируют другие задачи, и задачи, которые никого не блокируют.

**Шаги:**
1. Открыть Filter Results Screen созданного фильтра через Navigation Sidebar.
2. В textarea RSQL-выражения ввести `is_blocker==true`.
3. Дождаться обновления результатов таблицы.
4. Очистить textarea и ввести `is_blocker==false`.
5. Дождаться обновления результатов таблицы.

**Ожидаемый результат:**
- После шага 3 в таблице отображаются только задачи, которые блокируют хотя бы одну другую задачу.
- После шага 5 в таблице отображаются только задачи, которые не блокируют никакие другие задачи.
