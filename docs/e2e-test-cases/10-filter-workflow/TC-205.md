# TC-205: RSQL: фильтрация по полю assignee_id

**Предусловие:** пользователь авторизован, создан личный фильтр в RSQL-режиме, существуют задачи с назначенными ответственными. Известен UUID ответственного (например, `assignee-uuid-1`).

**Шаги:**
1. Открыть Filter Results Screen созданного фильтра через Navigation Sidebar.
2. В textarea RSQL-выражения ввести `assignee_id==assignee-uuid-1`.
3. Дождаться обновления результатов таблицы.
4. Очистить textarea и ввести `assignee_id=in=(assignee-uuid-1,assignee-uuid-2)`.
5. Дождаться обновления результатов таблицы.

**Ожидаемый результат:**
- После шага 3 в таблице отображаются только задачи, где ответственный — пользователь с UUID `assignee-uuid-1`.
- После шага 5 в таблице отображаются задачи с ответственными `assignee-uuid-1` и `assignee-uuid-2`.
