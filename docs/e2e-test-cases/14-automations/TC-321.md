# TC-321: Редактирование критериев отбора автоматизации

**Предусловие:** Пользователь авторизован. Существует личная автоматизация «Фильтр по статусу» с триггером task.status_changed, без критериев отбора. Открыта Automation Modal.

**Шаги:**
1. Нажать на строку автоматизации «Фильтр по статусу» для раскрытия.
2. В секции критериев отбора добавить условие: new_status == "closed".
3. Сохранить изменения.

**Ожидаемый результат:**
- Критерии отбора автоматизации обновлены.
- При раскрытии автоматизации отображается добавленный критерий: new_status == "closed".
- Автоматизация будет срабатывать только при изменении статуса на «closed».
