# TC-324: Включение автоматизации (toggle) -- без подтверждения

**Предусловие:** Пользователь авторизован. Существует выключенная личная автоматизация «Тестовая». Открыта Automation Modal.

**Шаги:**
1. Найти автоматизацию «Тестовая» в списке.
2. Нажать переключатель включения/выключения (toggle) для перевода в активное положение.

**Ожидаемый результат:**
- Автоматизация включается немедленно, без Confirmation Popup.
- Переключатель переходит в активное положение.
- Автоматизация начинает реагировать на соответствующие триггеры.
