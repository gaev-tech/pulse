# TC-329: Личные автоматизации -- доступны владельцу

**Предусловие:** Пользователь A авторизован и создал личную автоматизацию «Моя автоматизация». Пользователь B авторизован.

**Шаги:**
1. Пользователь A открывает Automation Modal через Navigation Sidebar (кнопка личных автоматизаций).
2. Пользователь A проверяет, что автоматизация «Моя автоматизация» отображается в списке.
3. Пользователь B открывает Automation Modal через Navigation Sidebar (кнопка личных автоматизаций).
4. Пользователь B проверяет список своих личных автоматизаций.

**Ожидаемый результат:**
- Пользователь A видит автоматизацию «Моя автоматизация» в своём списке личных автоматизаций.
- Пользователь B НЕ видит автоматизацию «Моя автоматизация» — она принадлежит Пользователю A.
- Каждый пользователь видит только свои личные автоматизации.
