# TC-325: Выключение автоматизации (toggle) -- без подтверждения

**Предусловие:** Пользователь авторизован. Существует включённая личная автоматизация «Тестовая». Открыта Automation Modal.

**Шаги:**
1. Найти автоматизацию «Тестовая» в списке.
2. Нажать переключатель включения/выключения (toggle) для перевода в неактивное положение.

**Ожидаемый результат:**
- Автоматизация выключается немедленно, без Confirmation Popup.
- Переключатель переходит в неактивное положение.
- Автоматизация перестаёт реагировать на триггеры.
