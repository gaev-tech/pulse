# TC-323: Доступ к полям payload триггера в действии

**Предусловие:** Пользователь авторизован. Открыта Automation Modal.

**Шаги:**
1. Начать создание новой автоматизации.
2. Ввести название «Вебхук со статусом».
3. Выбрать триггер task.status_changed.
4. Добавить действие — вызов API: метод POST, URL https://example.com/webhook.
5. В теле запроса действия использовать поля payload триггера: task.key, old_status, new_status.
6. Сохранить автоматизацию.

**Ожидаемый результат:**
- Автоматизация создана успешно.
- В настройках действия доступны поля payload триггера task.status_changed: task (id, key, title, status, owner_type, owner_id), actor (id, email, username), old_status, new_status.
- Поля payload корректно подставляются в тело запроса действия.
