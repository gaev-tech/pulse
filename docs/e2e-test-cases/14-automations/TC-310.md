# TC-310: Открытие Automation Modal

**Предусловие:** Пользователь авторизован.

**Шаги:**
1. В Navigation Sidebar нажать кнопку открытия модалки личных автоматизаций.

**Ожидаемый результат:**
- Открывается Automation Modal.
- Отображается список личных автоматизаций (или пустой список, если автоматизаций нет).
- Доступна возможность создания новой автоматизации.
