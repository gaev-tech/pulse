# TC-314: Создание автоматизации -- валидация обязательного поля "Название"

**Предусловие:** Пользователь авторизован. Открыта Automation Modal.

**Шаги:**
1. Начать создание новой автоматизации.
2. Оставить поле «Название» пустым.
3. Выбрать триггер task.created.
4. Добавить действие — вызов API (например, POST https://example.com/webhook).
5. Попытаться сохранить автоматизацию.

**Ожидаемый результат:**
- Автоматизация не создаётся.
- Отображается ошибка валидации, указывающая что поле «Название» обязательно для заполнения.
