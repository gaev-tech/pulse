# TC-322: Редактирование действий автоматизации

**Предусловие:** Пользователь авторизован. Существует личная автоматизация «Вебхук» с одним действием: POST https://example.com/old-webhook. Открыта Automation Modal.

**Шаги:**
1. Нажать на строку автоматизации «Вебхук» для раскрытия.
2. Изменить URL действия с https://example.com/old-webhook на https://example.com/new-webhook.
3. Сохранить изменения.

**Ожидаемый результат:**
- Действие автоматизации обновлено.
- При раскрытии автоматизации отображается новый URL: https://example.com/new-webhook.
