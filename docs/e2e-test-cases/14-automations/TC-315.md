# TC-315: Создание автоматизации -- валидация обязательного поля "Триггер"

**Предусловие:** Пользователь авторизован. Открыта Automation Modal.

**Шаги:**
1. Начать создание новой автоматизации.
2. Ввести название «Тестовая автоматизация».
3. Не выбирать триггер.
4. Добавить действие — вызов API (например, POST https://example.com/webhook).
5. Попытаться сохранить автоматизацию.

**Ожидаемый результат:**
- Автоматизация не создаётся.
- Отображается ошибка валидации, указывающая что поле «Триггер» обязательно для заполнения.
