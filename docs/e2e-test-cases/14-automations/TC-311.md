# TC-311: Отображение списка автоматизаций (название, триггер, переключатель, удаление)

**Предусловие:** Пользователь авторизован. У пользователя есть две личные автоматизации: «Уведомление о статусе» (триггер task.status_changed, включена) и «Логирование создания» (триггер task.created, выключена).

**Шаги:**
1. В Navigation Sidebar нажать кнопку открытия модалки личных автоматизаций.

**Ожидаемый результат:**
- В списке отображаются обе автоматизации.
- Для каждой автоматизации видны: название, триггер, переключатель включения/выключения, кнопка удаления.
- Автоматизация «Уведомление о статусе» имеет переключатель в активном положении.
- Автоматизация «Логирование создания» имеет переключатель в неактивном положении.
