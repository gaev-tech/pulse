# TC-327: Отмена удаления автоматизации

**Предусловие:** Пользователь авторизован. Существует личная автоматизация «Важная автоматизация». Открыта Automation Modal.

**Шаги:**
1. Найти автоматизацию «Важная автоматизация» в списке.
2. Нажать кнопку удаления автоматизации.
3. В появившемся Confirmation Popup нажать кнопку «Отмена».

**Ожидаемый результат:**
- Confirmation Popup закрывается.
- Автоматизация «Важная автоматизация» остаётся в списке.
- Состояние автоматизации не изменилось (включена/выключена, все настройки сохранены).
