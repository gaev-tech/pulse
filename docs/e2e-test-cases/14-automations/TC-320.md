# TC-320: Редактирование триггера автоматизации

**Предусловие:** Пользователь авторизован. Существует личная автоматизация «Тестовая» с триггером task.created. Открыта Automation Modal.

**Шаги:**
1. Нажать на строку автоматизации «Тестовая» для раскрытия.
2. Изменить триггер с task.created на task.status_changed.
3. Сохранить изменения.

**Ожидаемый результат:**
- Триггер автоматизации обновлён на task.status_changed.
- В свёрнутой строке автоматизации отображается новый триггер task.status_changed.
