# TC-319: Редактирование названия автоматизации

**Предусловие:** Пользователь авторизован. Существует личная автоматизация «Старое название». Открыта Automation Modal.

**Шаги:**
1. Нажать на строку автоматизации «Старое название» для раскрытия.
2. Изменить название с «Старое название» на «Новое название».
3. Сохранить изменения.

**Ожидаемый результат:**
- Название автоматизации обновлено на «Новое название».
- В списке автоматизаций отображается обновлённое название.
