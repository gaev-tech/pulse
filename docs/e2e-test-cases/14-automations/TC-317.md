# TC-317: Создание автоматизации с триггером и одним действием

**Предусловие:** Пользователь авторизован. Открыта Automation Modal.

**Шаги:**
1. Начать создание новой автоматизации.
2. Ввести название «Уведомление о закрытии».
3. Выбрать триггер task.status_changed.
4. Добавить действие — вызов API: метод POST, URL https://example.com/webhook.
5. Сохранить автоматизацию.

**Ожидаемый результат:**
- Автоматизация «Уведомление о закрытии» успешно создана.
- В списке автоматизаций появляется новая строка с названием «Уведомление о закрытии», триггером task.status_changed.
- Переключатель включения/выключения находится в активном положении по умолчанию.
