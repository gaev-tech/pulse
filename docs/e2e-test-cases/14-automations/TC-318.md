# TC-318: Создание автоматизации с несколькими действиями

**Предусловие:** Пользователь авторизован. Открыта Automation Modal.

**Шаги:**
1. Начать создание новой автоматизации.
2. Ввести название «Мульти-действие».
3. Выбрать триггер task.created.
4. Добавить первое действие — вызов API: метод POST, URL https://example.com/webhook1.
5. Добавить второе действие — вызов API: метод POST, URL https://example.com/webhook2.
6. Сохранить автоматизацию.

**Ожидаемый результат:**
- Автоматизация «Мульти-действие» успешно создана.
- При раскрытии автоматизации отображаются оба действия с корректными URL.
