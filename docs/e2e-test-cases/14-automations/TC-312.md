# TC-312: Раскрытие автоматизации по клику на строку

**Предусловие:** Пользователь авторизован. Открыта Automation Modal с автоматизацией «Уведомление о статусе».

**Шаги:**
1. Нажать на строку автоматизации «Уведомление о статусе» (не на переключатель и не на кнопку удаления).

**Ожидаемый результат:**
- Строка автоматизации раскрывается инлайн.
- Отображаются поля для редактирования: название, критерии отбора триггеров, список действий (вызовы API).
- Все текущие значения автоматизации заполнены корректно.
