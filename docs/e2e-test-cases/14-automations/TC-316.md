# TC-316: Создание автоматизации -- валидация наличия хотя бы одного действия

**Предусловие:** Пользователь авторизован. Открыта Automation Modal.

**Шаги:**
1. Начать создание новой автоматизации.
2. Ввести название «Тестовая автоматизация».
3. Выбрать триггер task.created.
4. Не добавлять ни одного действия.
5. Попытаться сохранить автоматизацию.

**Ожидаемый результат:**
- Автоматизация не создаётся.
- Отображается ошибка валидации, указывающая что необходимо добавить хотя бы одно действие.
