# TC-313: Одновременно может быть раскрыта только одна автоматизация

**Предусловие:** Пользователь авторизован. Открыта Automation Modal с двумя автоматизациями: «Уведомление о статусе» и «Логирование создания». Автоматизация «Уведомление о статусе» раскрыта.

**Шаги:**
1. Нажать на строку автоматизации «Логирование создания».

**Ожидаемый результат:**
- Автоматизация «Уведомление о статусе» сворачивается.
- Автоматизация «Логирование создания» раскрывается инлайн.
- Одновременно раскрыта только одна автоматизация.
