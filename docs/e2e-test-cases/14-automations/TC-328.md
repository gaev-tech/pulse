# TC-328: Командные автоматизации -- требуется право team.manage_automations

**Предусловие:** Пользователь A является участником команды «ACME» с правом team.manage_automations. Пользователь B является участником команды «ACME» без права team.manage_automations.

**Шаги:**
1. Пользователь A открывает Automation Modal для команды «ACME» (через Navigation Sidebar, раздел команды).
2. Пользователь A создаёт командную автоматизацию «Командный вебхук» с триггером task.created и действием POST https://example.com/team-webhook.
3. Пользователь A сохраняет автоматизацию.
4. Пользователь B пытается открыть Automation Modal для команды «ACME».

**Ожидаемый результат:**
- Пользователь A успешно создаёт командную автоматизацию.
- Пользователь B не имеет доступа к управлению командными автоматизациями из-за отсутствия права team.manage_automations.
