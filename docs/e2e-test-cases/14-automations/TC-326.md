# TC-326: Удаление автоматизации -- Confirmation Popup

**Предусловие:** Пользователь авторизован. Существует личная автоматизация «Для удаления». Открыта Automation Modal.

**Шаги:**
1. Найти автоматизацию «Для удаления» в списке.
2. Нажать кнопку удаления автоматизации.
3. В появившемся Confirmation Popup нажать кнопку «Да, подтверждаю».

**Ожидаемый результат:**
- Перед удалением отображается Confirmation Popup с текстом подтверждения.
- После подтверждения автоматизация «Для удаления» удаляется из списка.
- Автоматизация больше не отображается в Automation Modal.
