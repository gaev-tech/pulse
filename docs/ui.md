# UI

## Navigation Sidebar
Меню навигации в левой части экрана на всю высоту. Сверху вниз: иконка и название системы; кнопка перехода на Personal Activity Screen; раздел "Личные фильтры" (схлопывается, состояние сохраняется на бэкенде) — кнопка создания фильтра, список личных фильтров (клик — Filter Results Screen, в строчке — кнопка открытия Filter Activity Screen); раздел "Команды" (схлопывается, состояние сохраняется на бэкенде) — кнопка создания команды, список команд (каждая схлопывается, состояние сохраняется на бэкенде; клик — Team Activity Screen; в строчке — кнопка добавления командного фильтра; под строчкой — список фильтров команды: клик — Filter Results Screen, в строчке — кнопка Filter Activity Screen); кнопка открытия модалки личных автоматизаций; кнопка открытия модалки личных меток; кнопка перехода на About Screen; кнопка перехода на Documentation Screen; аватарка и имя пользователя — открывают Profile Popup.

## Profile Popup
Всплывающее меню, открывающееся при клике на аватарку и имя в нижней части Navigation Sidebar. Содержит сверху вниз: переключатель языка (русский/English), переключатель темы (светлая/тёмная/системная), список всех авторизованных на данном клиенте пользователей (клик — переход на Personal Activity Screen выбранного пользователя) с кнопкой "Добавить пользователя" в конце списка (ведёт на Auth Screen), кнопка выхода из системы.

## Auth Screen
Экран авторизации. Содержит форму входа и регистрации.

## About Screen
Экран знакомства с системой: описание продукта, его ценность, маркетинговая информация со скриншотами.

## Documentation Screen
Экран с документацией по системе.

## Personal Activity Screen
Экран личной ленты событий пользователя. Отображает все события по задачам, доступным пользователю, с учётом окна доступа.

## Team Activity Screen
Экран ленты событий команды. Отображает все события по задачам команды. Содержит фильтр по авторам изменений.

## Team Activity Filter By Member
Фильтр на экране ленты команды, позволяющий отображать события только по выбранным авторам изменений.

## Filter Activity Screen
Экран ленты событий по фильтру. Отображает: изменения фильтра, попадание и исчезновение задач из результатов, все события по задачам в период их вхождения в результаты фильтра.

## Filter Results Screen
Экран результатов фильтра. Верхняя панель: название фильтра (редактируется по клику), теги критериев, кнопка настроек таблицы (Table Settings Popup), кнопка "..." (Filter Popup с переходом на Filter Activity Screen и удалением фильтра). При изменении критерия результаты обновляются немедленно, появляется кнопка "Сохранить фильтр". Таблица задач: обязательные ячейки — ключ и название; необязательные — метки, команда (только в личных фильтрах), ответственный, статус, родительская задача, наличие вложений/блокеров/связей/ссылок, даты создания и изменения, авторы создания и изменения. Набор и порядок необязательных ячеек настраивается перетаскиванием в Table Settings Popup. Сортировка включается по клику на заголовок колонки (только для отображаемых колонок), направление обозначается стрелкой; shift+клик по другому заголовку добавляет вторичную сортировку — в заголовке отображается стрелка с цифрой «2». Настройки таблицы (набор, порядок, сортировка) сохраняются для фильтра и применяются для всех пользователей с доступом к нему.

## Task Share Modal
Модальное окно шаринга задачи. Поле поиска пользователя и список прав для выдачи. Право `view` отображается задизейблированным — оно выдаётся автоматически и может быть отозвано только через полный отзыв доступа к задаче.

## Task Screen
Полноэкранная страница задачи. Две части: левая (широкая) — название, описание, лента событий (обновляется динамически); правая — элементы управления сверху, затем остальные поля задачи (статус, ответственный, метки, ссылки, связи, блокирование, родитель, вложения).

## Task Modal
Модальное окно задачи. Отображает те же поля и структуру, что и Task Screen, но поверх текущего экрана. Содержит кнопку "Открыть задачу в отдельной странице".

## Task Create Modal
Модальное окно создания задачи. Позволяет указать название, описание и основные поля. Владелец задачи выбирается из двух вариантов: личная задача или одна из команд, где у пользователя есть право создания задач. После создания закрывается и открывается Task Modal созданной задачи — пользователь остаётся на текущей странице.

## Team Create Modal
Модальное окно создания команды. Поля: название (уникальное), prefix (уникальный). Позволяет сразу добавить участников. В списке прав право `view` отображается задизейблированным — оно обязательно и может быть отозвано только через удаление участника из команды.

## Team Edit Modal
Модальное окно редактирования команды. Позволяет изменить название и prefix, управлять правами участников, а также выдавать права пользователям, которых ещё нет в команде. Содержит кнопку удаления команды. В списке прав право `view` отображается задизейблированным — оно обязательно и может быть отозвано только через удаление участника из команды.

## Filter Create Modal
Модальное окно создания фильтра. Поля: название, критерии, тип — личный или командный. При выборе командного — список команд, в которых у пользователя есть право редактирования фильтров.

## Automation Modal
Модальное окно управления автоматизациями. Содержит список автоматизаций — каждая строка: название, триггер, переключатель включения/выключения, кнопка удаления. Клик по строке (не по переключателю или кнопке) — инлайн-раскрытие с редактированием названия, критериев отбора триггеров и вызовов API. Одновременно может быть раскрыта только одна строка.

## Global Search
Кнопка поиска под заголовком системы в Navigation Sidebar. При нажатии открывает Search Modal.

## Search Modal
Модальное окно поиска. Содержит поле ввода и список результатов под ним.

## Confirmation Popup
Всплывающее окно подтверждения опасной операции. Появляется перед удалением любой сущности и перед удалением участника команды. Содержит текст «Вы подтверждаете <операция>?», красную кнопку «Да, подтверждаю» и кнопку «Отмена».

## Task Link Toast
Всплывающее уведомление о том, что ссылка на задачу скопирована. Появляется после нажатия кнопки "Скопировать ссылку на задачу".

## Blocking Toast
Всплывающее уведомление о том, что задача не может быть заблокирована, так как сама находится в цепочке блокирующих задач.

## Blocking Tasks Popup
Всплывающее окно для просмотра и редактирования списка блокирующих задач. При добавлении — поиск по ключу, названию, описанию (contains). Задачи, образующие цикл блокирования, подсвечиваются восклицательным знаком и недоступны для выбора.

## Related Tasks Popup
Всплывающее окно для просмотра и редактирования списка связанных задач. При добавлении — поиск по ключу, названию, описанию (contains).

## Related Links Popup
Всплывающее окно для просмотра и редактирования списка внешних ссылок задачи.
